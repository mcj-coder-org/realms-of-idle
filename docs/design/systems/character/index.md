## Class Categories

| Category  | Examples               | Unlock Actions   |
| --------- | ---------------------- | ---------------- |
| Gathering | Forager, Miner, Hunter | Gather resources |
| Crafting  | Blacksmith, Alchemist  | Craft items      |
| Trade     | Trader, Merchant       | Complete trades  |
| Combat    | Warrior, Adventurer    | Defeat enemies   |
| Magic     | Mage, Enchanter        | Cast spells      |
| Social    | Scholar, Trainer       | Research, teach  |
| Dark      | Thief, Assassin        | Criminal acts    |

## Skill Tiers

| Tier     | Effect                    | Scaling          |
| -------- | ------------------------- | ---------------- |
| Lesser   | Base effect               | +1% per level    |
| Greater  | Improved + secondary      | +0.75% per level |
| Enhanced | Best + multiple secondary | +0.5% per level  |
