## Settlement Types

| Type    | Population | Features                  |
| ------- | ---------- | ------------------------- |
| Hamlet  | 10-50      | Basic services            |
| Village | 50-200     | Market, crafters          |
| Town    | 200-1000   | Guilds, specialized shops |
| City    | 1000+      | All services, politics    |

## Climate Zones

| Zone      | Effect                           | Regions          |
| --------- | -------------------------------- | ---------------- |
| Temperate | Baseline                         | Central areas    |
| Desert    | Heat damage, stamina drain       | Southern wastes  |
| Tundra    | Cold damage, limited resources   | Northern reaches |
| Tropical  | Disease risk, dense resources    | Jungle regions   |
| Swamp     | Poison risk, hidden dangers      | Wetlands         |
| Mountain  | Altitude effects, rare resources | High peaks       |
