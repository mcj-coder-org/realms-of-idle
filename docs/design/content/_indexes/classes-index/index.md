### Carpenter Tier Progression

**Carpenter** uses tier-based progression to unlock different types of woodworking:

| Tier       | Level Range | Tags Grant                        | Products                         |
| ---------- | ----------- | --------------------------------- | -------------------------------- |
| Base       | 1-9         | `Crafting/Woodworking`            | Basic wooden items               |
| Apprentice | 10-24       | `Crafting/Woodworking/Weapon`     | Weapons (bows, staves), shields  |
| Journeyman | 25-49       | `Crafting/Woodworking/Furniture`  | Furniture (chairs, tables, beds) |
| Master     | 50+         | `Crafting/Woodworking/Structural` | Building components, structures  |

## Class Progression Paths

### Crafter Path

```
Crafter (Tier 1)
    ├─ Blacksmith (Tier 2)
    │   ├─ Weaponsmith (Tier 3)
    │   └─ Armorsmith (Tier 3)
    ├─ Carpenter (Tier 2)
    │   ├─ Apprentice: Weapons, shields
    │   ├─ Journeyman: Furniture
    │   └─ Master: Structural components
    ├─ Leatherworker (Tier 2)
    ├─ Tailor (Tier 2)
    ├─ Cook (Tier 2)
    ├─ Jeweler (Tier 2)
    ├─ Alchemist (Tier 2)
    ├─ Glassblower (Tier 2)
    └─ Weaver (Tier 2)
```

### Fighter Path

```
Fighter (Tier 1)
    ├─ Warrior (Tier 2)
    │   └─ Knight (Tier 3)
    ├─ Archer (Tier 2)
    ├─ Guard (Tier 2)
    └─ Scout (Tier 2)
```

### Rogue Path

```
Rogue (Tier 1)
    └─ Assassin (Tier 2)
```

### Channeler Path

```
Channeler (Tier 1)
    └─ Enchanter (Tier 2)
```

### Gatherer Path

```

    └─ Herbalist (Tier 2)
```
