# Army of Dead

## Overview

Summon an army of undead minions to aid you in battle, with their power and loyalty increasing with caster power.

| Property  | Value       |
| --------- | ----------- |
| School    | Necromancy  |
| Element   | Undead      |
| Targeting | Point       |
| Classes   | Necromancer |

## Effects

### Lesser

- Summons 3 undead minions
- Minion HP: 25-35
- Minion damage: +10%
- Duration: 30 seconds
- 2s cast time
- Mana cost: 60
- 40 second cooldown
- **Scaling**: +2 HP per minion per class level

### Greater

- Summons 4 undead minions
- Minion HP: 50-70
- Minion damage: +20%
- Minions can self-heal 5 HP per 5s
- Duration: 45 seconds
- 1.5s cast time
- Mana cost: 90
- 40 second cooldown
- **Scaling**: +3 HP per minion per class level

### Enhanced

- Summons 5 undead minions
- Minion HP: 80-110
- Minion damage: +35%
- Minions can self-heal 10 HP per 3s
- Can coordinate minion attacks for bonus damage
- Duration: 60 seconds
- 1s cast time
- Mana cost: 120
- 40 second cooldown
- **Scaling**: +4 HP per minion per class level
