# Blizzard

## Overview

Launch a devastating winter storm that engulfs an area with freezing ice and wind, dealing ice damage to all targets and freezing nearby enemies.

| Property  | Value       |
| --------- | ----------- |
| School    | Destruction |
| Element   | Ice         |
| Targeting | Point (AoE) |

## Effects

### Lesser

- 50-70 ice damage
- 3m radius
- 2s cast time
- Freeze: 20% chance for 2s
- **Scaling**: +2 damage per class level

### Greater

- 85-110 ice damage
- 5m radius
- 1.5s cast time
- Freeze: 40% chance for 3s
- Slow: 40% speed reduction for 5s
- **Scaling**: +3 damage per class level

### Enhanced

- 120-160 ice damage
- 7m radius
- 1s cast time
- Freeze: 60% chance for 4s
- Slow: 60% speed reduction for 8s
- Icy Ground: 50% movement penalty for 10s in affected area
- **Scaling**: +4 damage per class level
