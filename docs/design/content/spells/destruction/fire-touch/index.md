# Fire Touch

## Overview

Ignite targets through physical contact, dealing fire damage.

| Property  | Value         |
| --------- | ------------- |
| School    | Destruction   |
| Element   | Fire          |
| Targeting | Touch (melee) |

## Effects

### Innate (Creature)

- 15-25 fire damage on hit
- Ignites flammable materials
- **Scaling**: Creature level

## Innate Users

| Creature                                                                     | Notes          |
| ---------------------------------------------------------------------------- | -------------- |
| [Fire Elemental](../../creatures/magical/elementals/fire-elemental/index.md) | Primary attack |
| Flame Sprite                                                                 | Lesser damage  |

## Notes

- This spell is innate to fire-based creatures
- Cannot be learned by standard magic users
- Countered by fire resistance
- Creatures immune to their own element
