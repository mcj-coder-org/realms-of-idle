# Inferno

## Overview

Unleash a devastating explosion of flame in a wide area, consuming everything in a massive inferno.

| Property  | Value       |
| --------- | ----------- |
| School    | Destruction |
| Element   | Fire        |
| Targeting | Point (AoE) |
| Classes   | Pyromancer  |

## Effects

### Lesser

- 80-100 fire damage
- 12m radius
- 2s cast time
- Burning: 8 damage/s for 4s
- 25 second cooldown
- **Scaling**: +2 damage per class level

### Greater

- 130-160 fire damage
- 16m radius
- 1.5s cast time
- Burning: 12 damage/s for 5s
- 50% chance to ignite nearby enemies
- 25 second cooldown
- **Scaling**: +3 damage per class level

### Enhanced

- 180-220 fire damage
- 20m radius
- 1s cast time
- Burning: 15 damage/s for 6s
- 75% chance to ignite, spreads to nearby foes
- Ignites flammable objects
- 25 second cooldown
- **Scaling**: +4 damage per class level
