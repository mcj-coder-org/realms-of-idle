# Destruction School

Destruction magic focuses on dealing damage through elemental and arcane forces.

## Elements

| Element   | Characteristics                                             |
| --------- | ----------------------------------------------------------- |
| Fire      | High damage, burning DoT, blocked by fire resistance        |
| Ice       | Moderate damage, slowing effect, blocked by cold resistance |
| Lightning | Fast casting, chain potential, blocked by shock resistance  |
| Arcane    | Pure damage, no resistances, no secondary effects           |

## Spells

| Spell                                     | Element   | Tier | Effect               |
| ----------------------------------------- | --------- | ---- | -------------------- |
| [Fireball](fireball/index.md)             | Fire      | All  | Area fire damage     |
| [Lightning Bolt](lightning-bolt/index.md) | Lightning | All  | Chain lightning      |
| [Ice Shard](ice-shard/index.md)           | Ice       | All  | Piercing cold damage |

## Specialization

Destruction mages can specialize in a single element for +25% damage but -50% effectiveness with others.
