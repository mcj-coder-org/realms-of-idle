# Lightning Bolt

## Overview

Hurl a bolt of lightning that can chain to nearby enemies.

| Property  | Value          |
| --------- | -------------- |
| School    | Destruction    |
| Element   | Lightning      |
| Targeting | Target (Chain) |

## Effects

### Lesser

- 40-55 lightning damage
- No chaining
- 1.5s cast time
- **Scaling**: +2 damage per class level

### Greater

- 65-85 lightning damage
- Chains to 1 nearby target (50% damage)
- 1.2s cast time
- **Scaling**: +3 damage per class level

### Enhanced

- 90-120 lightning damage
- Chains to 2 nearby targets (50% damage each)
- 0.8s cast time
- Stuns targets briefly
- **Scaling**: +4 damage per class level
