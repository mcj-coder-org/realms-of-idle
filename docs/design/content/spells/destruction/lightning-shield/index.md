# Lightning Shield

## Overview

Create a protective barrier of electrical energy around the target, shielding them from harm while shocking melee attackers. Lightning Shield combines protection with electrical counterattack.

| Property  | Value                 |
| --------- | --------------------- |
| School    | Protection            |
| Element   | Lightning             |
| Targeting | Self or single target |

## Effects

### Lesser

- 60-80 shield absorption
- 20-30 return lightning damage to attackers
- 15% stun chance on attacker (1s stun)
- 45s duration or until depleted
- 2s cast time
- **Scaling**: +1 absorption per class level

### Greater

- 90-120 shield absorption
- 40-60 return lightning damage to attackers
- 30% stun chance on attacker (2s stun)
- 60s duration or until depleted
- 1.5s cast time
- Chain reaction: Each stunned attacker chains damage to nearby allies
- **Scaling**: +2 absorption per class level

### Enhanced

- 130-170 shield absorption
- 70-100 return lightning damage to attackers
- 50% stun chance on attacker (2-3s stun)
- 90s duration or until depleted
- 1s cast time
- Chain reaction: Each stunned attacker chains damage to 2-3 nearby allies
- Overload: Shield reflects +50% damage as lightning chains to nearby enemies
- **Scaling**: +3 absorption per class level
