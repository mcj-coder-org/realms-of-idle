# Spirit Wind

## Overview

Call upon air spirits to manifest powerful winds that blast foes away and disrupt formations.

| Property  | Value                |
| --------- | -------------------- |
| School    | Nature (Shamanic)    |
| Element   | Air                  |
| Targeting | Area / Single Target |

## Effects

### Lesser

- Damage: 15-25 HP
- Knockback: 5m away
- Range: 20m
- Area: 10m radius
- Cost: 110 Spirit Pool
- Duration: Instant
- Scaling: +0.8 damage per Wind Walker level

### Greater

- Damage: 30-50 HP
- Knockback: 8m away
- Range: 25m
- Area: 14m radius
- Cost: 190 Spirit Pool
- Duration: Instant
- Disorientation: -30% accuracy for 5 seconds
- Scaling: +1.8 damage per Wind Walker level

### Enhanced

- Damage: 60-100 HP
- Knockback: 12m away
- Range: 30m
- Area: 18m radius
- Cost: 300 Spirit Pool
- Duration: Instant
- Disorientation: -60% accuracy for 8 seconds
- Scatter: Disrupts enemy formations, spreads groups
- Scaling: +3 damage per Wind Walker level

## Acquisition

| Method | Requirement             |
| ------ | ----------------------- |
| Spell  | Wind Walker (automatic) |
