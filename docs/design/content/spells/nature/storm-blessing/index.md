# Storm Blessing

## Overview

Invoke storm spirits to bless allies with protective electricity and enhanced speed.

| Property  | Value             |
| --------- | ----------------- |
| School    | Nature (Shamanic) |
| Element   | Lightning         |
| Targeting | Area / Self       |

## Effects

### Lesser

- Protection: +15% electrical resistance
- Speed: +15% action speed
- Duration: 10 minutes
- Cost: 90 Spirit Pool
- Range: 20m
- Scaling: +0.5% per Storm Caller level

### Greater

- Protection: +30% electrical resistance
- Speed: +25% action speed
- Duration: 20 minutes
- Cost: 160 Spirit Pool
- Range: 25m
- Additional: +5 reflexes (dodge)
- Scaling: +1% per Storm Caller level

### Enhanced

- Protection: +50% electrical resistance
- Speed: +40% action speed
- Duration: 30 minutes
- Cost: 270 Spirit Pool
- Range: 30m
- Additional: +10 reflexes (dodge)
- Counterstrike: Attacks against blessed gain 20% chance to stun attacker
- Scaling: +1.5% per Storm Caller level

## Acquisition

| Method | Requirement              |
| ------ | ------------------------ |
| Spell  | Storm Caller (automatic) |
