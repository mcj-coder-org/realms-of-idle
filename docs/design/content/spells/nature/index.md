# Nature School

Nature magic is exclusive to shamans and draws power from the tribal spirit pool rather than personal mana.

## Access Restriction

| Race Type  | Access                           |
| ---------- | -------------------------------- |
| Tribal     | Full access through Shaman class |
| Settlement | No access (nature affinity only) |

## Spell Categories

| Category      | Focus                              |
| ------------- | ---------------------------------- |
| Environmental | Weather, terrain manipulation      |
| Communal      | Area healing, tribal buffs         |
| Beast         | Animal influence and summoning     |
| Spiritual     | Ancestor guidance, curse cleansing |
