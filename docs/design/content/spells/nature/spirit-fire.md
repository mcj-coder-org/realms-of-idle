# Spirit Fire

## Overview

Call upon fire spirits to manifest sacred flames that burn away impurity and protect the tribe.

| Property  | Value                |
| --------- | -------------------- |
| School    | Nature (Shamanic)    |
| Element   | Fire                 |
| Targeting | Area / Single Target |

## Effects

### Lesser

- Damage: 20-30 HP
- Range: 20m
- Area: 8m radius
- Cost: 120 Spirit Pool
- Duration: Instant
- Scaling: +1 damage per Flame Keeper level

### Greater

- Damage: 40-60 HP
- Range: 25m
- Area: 12m radius
- Cost: 200 Spirit Pool
- Duration: Instant
- Purification: Removes 1 minor curse
- Scaling: +2 damage per Flame Keeper level

### Enhanced

- Damage: 80-120 HP
- Range: 30m
- Area: 15m radius
- Cost: 300 Spirit Pool
- Duration: Instant
- Purification: Removes 1 moderate curse
- Scaling: +3 damage per Flame Keeper level

## Acquisition

| Method | Requirement              |
| ------ | ------------------------ |
| Spell  | Flame Keeper (automatic) |
