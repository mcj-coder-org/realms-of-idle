# Dream Walk

## Overview

Step into the dream realm to travel through spirit space and gain understanding beyond normal sight.

| Property  | Value             |
| --------- | ----------------- |
| School    | Nature (Shamanic) |
| Element   | Spiritual         |
| Targeting | Self / Group      |

## Effects

### Lesser

- Effect: Enter dream realm form
- Duration: 20 seconds
- Cost: 140 Spirit Pool
- Benefit: Can pass through physical obstacles as ethereal form
- Range: Can move 30m per second in dream
- Scaling: +5% per Spirit Walker level

### Greater

- Effect: Enter dream realm form
- Duration: 30 seconds
- Cost: 240 Spirit Pool
- Benefit: Can pass through physical obstacles, see hidden things
- Range: Can move 50m per second in dream
- Allies: Can bring 1 willing ally
- Scaling: +10% per Spirit Walker level

### Enhanced

- Effect: Enter dream realm form
- Duration: 40 seconds
- Cost: 360 Spirit Pool
- Benefit: Can pass through anything, see all hidden/astral things
- Range: Can move 80m per second in dream
- Allies: Can bring up to 3 willing allies
- Transcendence: Invulnerable to physical attacks while dreaming
- Scaling: +15% per Spirit Walker level
