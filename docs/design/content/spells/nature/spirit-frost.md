# Spirit Frost

## Overview

Call upon ice spirits to manifest crystalline frost that slows and damages foes.

| Property  | Value                |
| --------- | -------------------- |
| School    | Nature (Shamanic)    |
| Element   | Ice                  |
| Targeting | Area / Single Target |

## Effects

### Lesser

- Damage: 20-30 HP
- Slow: -40% movement speed
- Range: 20m
- Area: 8m radius
- Cost: 120 Spirit Pool
- Duration: 5 seconds slow
- Scaling: +1 damage per Frost Shaman level

### Greater

- Damage: 40-60 HP
- Slow: -60% movement speed
- Range: 25m
- Area: 12m radius
- Cost: 200 Spirit Pool
- Duration: 8 seconds slow
- Freeze chance: 25%
- Scaling: +2 damage per Frost Shaman level

### Enhanced

- Damage: 80-120 HP
- Slow: -80% movement speed
- Range: 30m
- Area: 15m radius
- Cost: 300 Spirit Pool
- Duration: 10 seconds slow
- Freeze chance: 50%
- Brittle: Frozen enemies take +30% damage
- Scaling: +3 damage per Frost Shaman level

## Acquisition

| Method | Requirement              |
| ------ | ------------------------ |
| Spell  | Frost Shaman (automatic) |
