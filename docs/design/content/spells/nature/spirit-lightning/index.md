# Spirit Lightning

## Overview

Call upon storm spirits to manifest lightning bolts that strike foes and chain between targets.

| Property  | Value                |
| --------- | -------------------- |
| School    | Nature (Shamanic)    |
| Element   | Lightning            |
| Targeting | Area / Single Target |

## Effects

### Lesser

- Damage: 25-35 HP
- Chain: Jumps to 2 nearby enemies
- Chain range: 10m
- Range: 20m
- Cost: 130 Spirit Pool
- Duration: Instant
- Scaling: +1.2 damage per Storm Caller level

### Greater

- Damage: 50-70 HP
- Chain: Jumps to 4 nearby enemies
- Chain range: 15m
- Range: 25m
- Cost: 220 Spirit Pool
- Duration: Instant
- Stun: 20% chance for 2 seconds
- Scaling: +2.2 damage per Storm Caller level

### Enhanced

- Damage: 100-140 HP
- Chain: Jumps to 6 nearby enemies
- Chain range: 20m
- Range: 30m
- Cost: 320 Spirit Pool
- Duration: Instant
- Stun: 40% chance for 3 seconds
- Paralysis: Stunned enemies paralyzed (-50% action speed)
- Scaling: +3.5 damage per Storm Caller level

## Acquisition

| Method | Requirement              |
| ------ | ------------------------ |
| Spell  | Storm Caller (automatic) |
