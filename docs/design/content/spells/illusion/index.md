# Illusion School

Illusion magic manipulates perception, creates false images, and affects minds.

## Spell Categories

| Category     | Focus                  |
| ------------ | ---------------------- |
| Concealment  | Invisibility, stealth  |
| Deception    | False images, sounds   |
| Mind Control | Charm, fear, confusion |
| Sensory      | Enhanced/dulled senses |

## Spells

| Spell         | Category     | Tier | Effect                   |
| ------------- | ------------ | ---- | ------------------------ |
| Invisibility  | Concealment  | All  | Target becomes invisible |
| Charm         | Mind Control | All  | Target becomes friendly  |
| Fear          | Mind Control | All  | Target flees             |
| Muffle        | Concealment  | All  | Silence movement         |
| Phantom Image | Deception    | All  | Create false copy        |

## Specialization

Illusion mages can specialize in concealment for extended durations or mind control for stronger effects.
