# Mass Invisibility

## Overview

Cloak multiple creatures in illusions of invisibility, allowing them to move unseen and position without detection.

| Property  | Value           |
| --------- | --------------- |
| School    | Illusion        |
| Element   | Mind            |
| Targeting | Area            |
| Duration  | Until cancelled |
| Classes   | Illusionist     |

## Effects

### Lesser

- Affects up to 3 creatures
- Duration: 20 seconds
- Breaks on hostile action (attack/spell)
- Movement speed while invisible: +10%
- Detection difficulty: +20%
- Mana cost: 70
- 2s cast time
- 35 second cooldown
- **Scaling**: +1s duration per class level

### Greater

- Affects up to 4 creatures
- Duration: 35 seconds
- Breaks on hostile action (attack/spell)
- Movement speed while invisible: +25%
- Detection difficulty: +40%
- Can cast one spell while remaining invisible
- Mana cost: 110
- 1.5s cast time
- 35 second cooldown
- **Scaling**: +1.5s duration per class level

### Enhanced

- Affects up to 6 creatures
- Duration: 60 seconds
- Breaks on hostile action (attack/spell)
- Movement speed while invisible: +40%
- Detection difficulty: +70%
- Can cast multiple spells while remaining invisible
- Invisible allies can coordinate silently
- Mana cost: 160
- 1s cast time
- 35 second cooldown
- **Scaling**: +2s duration per class level
