# Heal

## Overview

Channel restorative energy to heal a single target's wounds instantly.

| Property  | Value         |
| --------- | ------------- |
| School    | Restoration   |
| Targeting | Single target |
| Range     | Touch to 10m  |

## Effects

### Lesser

- Restore 30-50 health
- 2s cast time
- Touch range
- **Scaling**: +3 healing per class level

### Greater

- Restore 60-90 health
- 1.5s cast time
- 5m range
- Removes minor wounds
- **Scaling**: +5 healing per class level

### Enhanced

- Restore 100-140 health
- 1s cast time
- 10m range
- Removes moderate wounds
- Cleanses 1 debuff
- **Scaling**: +7 healing per class level
