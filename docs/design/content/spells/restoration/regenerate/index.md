# Regenerate

## Overview

Grant a target enhanced healing over time, slowly restoring health.

| Property  | Value            |
| --------- | ---------------- |
| School    | Restoration      |
| Targeting | Single target    |
| Duration  | Variable by tier |

## Effects

### Lesser

- +5 health per second
- 10 second duration (50 total)
- 1.5s cast time
- **Scaling**: +0.3 health/s per class level

### Greater

- +8 health per second
- 15 second duration (120 total)
- 1.2s cast time
- Stacks with other healing
- **Scaling**: +0.5 health/s per class level

### Enhanced

- +12 health per second
- 20 second duration (240 total)
- 1s cast time
- Can regenerate limbs over extended periods
- **Scaling**: +0.7 health/s per class level
