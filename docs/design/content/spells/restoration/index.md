# Restoration School

Restoration magic focuses on healing wounds, curing ailments, and enhancing allies.

## Spell Categories

| Category      | Focus                           |
| ------------- | ------------------------------- |
| Healing       | Direct health restoration       |
| Regeneration  | Health over time effects        |
| Curing        | Remove poison, disease, debuffs |
| Fortification | Temporary stat boosts           |

## Spells

| Spell                             | Category      | Tier | Effect                |
| --------------------------------- | ------------- | ---- | --------------------- |
| [Heal](heal/index.md)             | Healing       | All  | Direct health restore |
| [Regenerate](regenerate/index.md) | Regeneration  | All  | Health over time      |
| Cure Poison                       | Curing        | All  | Remove poison effects |
| Fortify                           | Fortification | All  | Temporary stat boost  |

## Specialization

Restoration mages can specialize in healing for +30% healing effectiveness or fortification for +20% buff duration.
