# Shield

## Overview

Create a magical barrier around yourself that absorbs incoming damage.

| Property  | Value          |
| --------- | -------------- |
| School    | Abjuration     |
| Targeting | Self           |
| Duration  | Until depleted |

## Effects

### Lesser

- Absorbs 50-80 damage
- Instant cast
- 30 second maximum duration
- **Scaling**: +5 absorption per class level

### Greater

- Absorbs 100-140 damage
- Instant cast
- 45 second maximum duration
- Reflects 10% damage to melee attackers
- **Scaling**: +8 absorption per class level

### Enhanced

- Absorbs 180-250 damage
- Instant cast
- 60 second maximum duration
- Reflects 20% damage to melee attackers
- Blocks 1 spell completely
- **Scaling**: +12 absorption per class level
