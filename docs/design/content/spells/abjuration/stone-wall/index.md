# Stone Wall

## Overview

Raise a wall of solid stone that protects allies from harm, blocks enemy movement, and can reflect projectiles.

| Property  | Value           |
| --------- | --------------- |
| School    | Abjuration      |
| Element   | Earth           |
| Targeting | Area            |
| Duration  | Until destroyed |
| Classes   | Geomancer       |

## Effects

### Lesser

- Absorbs 100-140 damage
- Wall length: 8m
- Wall height: 3m
- Blocks enemy movement completely
- Instant cast
- 45 second cooldown
- **Scaling**: +6 absorption per class level

### Greater

- Absorbs 170-220 damage
- Wall length: 12m
- Wall height: 5m
- Blocks enemy movement completely
- Reflects 20% of projectile damage back
- Instant cast
- 45 second cooldown
- **Scaling**: +8 absorption per class level

### Enhanced

- Absorbs 260-320 damage
- Wall length: 16m
- Wall height: 7m
- Blocks enemy movement completely
- Reflects 40% of projectile damage back
- Reflects melee attacks with 15% reduction
- Can create corner formations
- Instant cast
- 45 second cooldown
- **Scaling**: +10 absorption per class level
