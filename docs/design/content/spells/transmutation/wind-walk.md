# Wind Walk

## Overview

Enhance movement speed using air magic, enabling swift travel, rapid repositioning, and temporary flight.

| Property  | Value           |
| --------- | --------------- |
| School    | Transmutation   |
| Element   | Air             |
| Targeting | Self or ally    |
| Duration  | Until cancelled |
| Classes   | Aeromancer      |

## Effects

### Lesser

- Movement speed: +30%
- Can travel short distances airborne (3m hover)
- Duration: Until mana depleted
- Mana drain: 2 per second
- Instant cast
- 10 second cooldown (when started fresh)
- **Scaling**: +0.5% speed per class level

### Greater

- Movement speed: +50%
- Can travel airborne (6m hover)
- Climb walls and steep terrain at normal movement speed
- Duration: Until mana depleted
- Mana drain: 1.5 per second
- Can extend to nearby ally
- Instant cast
- 10 second cooldown
- **Scaling**: +0.75% speed per class level

### Enhanced

- Movement speed: +80%
- Can travel airborne (10m hover and flight)
- Climb and descend freely
- Duration: Until mana depleted
- Mana drain: 1 per second
- Can extend to multiple nearby allies
- Allies gain half the movement bonuses
- Instant cast
- 10 second cooldown
- **Scaling**: +1% speed per class level
