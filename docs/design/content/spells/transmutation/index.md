# Transmutation School

Transmutation magic alters the fundamental properties of creatures and objects.

## Spell Categories

| Category       | Focus                       |
| -------------- | --------------------------- |
| Enhancement    | Temporary stat improvements |
| Transformation | Change form or properties   |
| Movement       | Alter speed and mobility    |
| Material       | Change material properties  |

## Spells

| Spell        | Category       | Tier | Effect                  |
| ------------ | -------------- | ---- | ----------------------- |
| Haste        | Movement       | All  | Increase speed          |
| Slow         | Movement       | All  | Reduce target speed     |
| Stoneskin    | Transformation | All  | Skin becomes stone-like |
| Polymorph    | Transformation | All  | Change creature form    |
| Featherfall  | Movement       | All  | Slow falling            |
| Waterwalking | Enhancement    | All  | Walk on water surface   |

## Specialization

Transmutation mages can specialize in enhancement for stronger/longer buffs or transformation for more powerful alterations.
