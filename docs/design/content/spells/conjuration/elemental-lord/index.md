# Elemental Lord

## Overview

Summon a mighty elemental lord from the planes to aid you in battle with devastating power and supreme capability.

| Property  | Value       |
| --------- | ----------- |
| School    | Conjuration |
| Element   | Planar      |
| Targeting | Point       |
| Classes   | Summoner    |

## Effects

### Lesser

- Summons 1 elemental lord
- Lord HP: 120-150
- Lord damage: +50%
- Lord abilities: Elemental mastery (+40% elemental damage), Area control (control 12m radius)
- Duration: 50 seconds
- 2.5s cast time
- Mana cost: 150
- 60 second cooldown
- **Scaling**: +4 HP per class level

### Greater

- Summons 1 elemental lord
- Lord HP: 200-250
- Lord damage: +75%
- Lord abilities: Elemental mastery (+70% elemental damage), Area control (control 15m radius), Planar presence (nearby allies gain +20% damage)
- Duration: 70 seconds
- 2s cast time
- Mana cost: 220
- 60 second cooldown
- **Scaling**: +5 HP per class level

### Enhanced

- Summons 1 elemental lord
- Lord HP: 300-360
- Lord damage: +100%
- Lord abilities: Elemental mastery (+100% elemental damage), Area control (control 20m radius), Planar presence (nearby allies gain +40% damage), Elemental aura (damage aura to all nearby enemies)
- Can channel lord's power for bonus effects
- Duration: 90 seconds
- 1.5s cast time
- Mana cost: 300
- 60 second cooldown
- **Scaling**: +6 HP per class level
