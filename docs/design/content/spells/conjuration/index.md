# Conjuration School

Conjuration magic summons creatures, creates objects, and manipulates space.

## Spell Categories

| Category      | Focus                        |
| ------------- | ---------------------------- |
| Summoning     | Call creatures to serve      |
| Teleportation | Move instantly through space |
| Creation      | Conjure temporary objects    |
| Binding       | Trap or contain creatures    |

## Spells

| Spell           | Category      | Tier | Effect                    |
| --------------- | ------------- | ---- | ------------------------- |
| Summon Creature | Summoning     | All  | Call creature to fight    |
| Teleport        | Teleportation | All  | Instant travel            |
| Conjure Weapon  | Creation      | All  | Create temporary weapon   |
| Summon Mount    | Summoning     | All  | Call riding creature      |
| Banish          | Binding       | All  | Return summoned creatures |

## Specialization

Conjuration mages can specialize in summoning for stronger/longer-lasting creatures or teleportation for greater range.
