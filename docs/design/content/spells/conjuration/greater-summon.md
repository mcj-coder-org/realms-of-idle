# Greater Summon

## Overview

Summon powerful planar entities from other dimensions to aid you in battle with significant combat capability.

| Property  | Value       |
| --------- | ----------- |
| School    | Conjuration |
| Element   | Planar      |
| Targeting | Point       |
| Classes   | Summoner    |

## Effects

### Lesser

- Summons 1 powerful planar entity
- Entity HP: 60-80
- Entity damage: +25%
- Entity special ability: Planar strike (+30% damage)
- Duration: 40 seconds
- 2s cast time
- Mana cost: 80
- 50 second cooldown
- **Scaling**: +3 HP per class level

### Greater

- Summons 1-2 powerful planar entities
- Entity HP: 110-140
- Entity damage: +40%
- Entity special abilities: Planar strike (+50% damage), Planar barrier (+15% all resistance)
- Duration: 60 seconds
- 1.5s cast time
- Mana cost: 120
- 50 second cooldown
- **Scaling**: +4 HP per class level

### Enhanced

- Summons 2 powerful planar entities
- Entity HP: 180-220
- Entity damage: +60%
- Entity special abilities: Planar strike (+75% damage), Planar barrier (+25% all resistance), Planar echo (duplicate hits at 50% damage)
- Can coordinate entity attacks for bonus damage
- Duration: 90 seconds
- 1s cast time
- Mana cost: 160
- 50 second cooldown
- **Scaling**: +5 HP per class level
