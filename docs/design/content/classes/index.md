# Character Classes

Classes are organized in a specialization tree. Each Foundation class branches into specialized paths, which may further specialize into master-level classes.

See [Class Tiers](../../systems/character/class-tiers.md) for tier mechanics and XP requirements.

## Class Trees

### [Fighter](./fighter/)

Combat-focused foundation class.

- [Warrior](./fighter/warrior/) → [Knight](./fighter/warrior/knight/)
- [Archer](./fighter/archer/)
- [Scout](./fighter/scout/) → [Ranger](./gatherer/hunter/ranger/) _(consolidation with Hunter)_
- [Guard](./fighter/guard/)
- [Adventurer](./fighter/adventurer/)

### [Crafter](./crafter/)

Creation and manufacturing foundation class.

- [Blacksmith](./crafter/blacksmith/) → [Weaponsmith](./crafter/blacksmith/weaponsmith/), [Armorsmith](./crafter/blacksmith/armorsmith/)
- [Alchemist](./crafter/alchemist/)
- [Tailor](./crafter/tailor/)
- [Carpenter](./crafter/carpenter/)
- [Cook](./crafter/cook/)
- [Jeweler](./crafter/jeweler/)
- [Glassblower](./crafter/glassblower/)
- [Leatherworker](./crafter/leatherworker/)
- [Weaver](./crafter/weaver/)

### [Gatherer](./gatherer/)

Resource acquisition foundation class.

- [Miner](./gatherer/miner/)
- [Forager](./gatherer/forager/)
- [Hunter](./gatherer/hunter/) → [Ranger](./gatherer/hunter/ranger/) _(consolidation with Scout)_
- [Lumberjack](./gatherer/lumberjack/)
- [Fisher](./gatherer/fisher/)
- [Farmer](./gatherer/farmer/)
- [Herbalist](./gatherer/herbalist/)

### [Channeler](./channeler/)

Magic foundation class specializing in arcane disciplines.

- [Mage](./channeler/mage/) → [Pyromancer](./channeler/mage/pyromancer/), [Cryomancer](./channeler/mage/cryomancer/), [Electromancer](./channeler/mage/electromancer/), [Geomancer](./channeler/mage/geomancer/), [Aeromancer](./channeler/mage/aeromancer/), [Hydromancer](./channeler/mage/hydromancer/), [Necromancer](./channeler/mage/necromancer/), [Summoner](./channeler/mage/summoner/), [Illusionist](./channeler/mage/illusionist/)

### [Spiritcaller](./spiritcaller/)

Magic foundation class specializing in spirit magic and shamanic traditions.

- [Shaman](./spiritcaller/shaman/) → [Flame Keeper](./spiritcaller/shaman/flame-keeper/), [Frost Shaman](./spiritcaller/shaman/frost-shaman/), [Storm Caller](./spiritcaller/shaman/storm-caller/), [Earth Shaman](./spiritcaller/shaman/earth-shaman/), [Wind Walker](./spiritcaller/shaman/wind-walker/), [Tide Caller](./spiritcaller/shaman/tide-caller/), [Ancestor Shaman](./spiritcaller/shaman/ancestor-shaman/), [Beast Speaker](./spiritcaller/shaman/beast-speaker/), [Spirit Walker](./spiritcaller/shaman/spirit-walker/)

### [Trader](./trader/)

Commerce foundation class.

- [Merchant](./trader/merchant/)
- [Caravaner](./trader/caravaner/)
- [Appraiser](./trader/appraiser/)

### [Rogue](./rogue/)

Stealth and criminal foundation class.

- [Thief](./rogue/thief/)
- [Assassin](./rogue/assassin/)
- [Fence](./rogue/fence/)
- [Smuggler](./rogue/smuggler/)

## Consolidation Classes

Multi-discipline classes combining two foundation classes:

### Mage Consolidations

- [Battlemage](./channeler/mage/battlemage/) - Mage + Warrior
- [Enchanter](./consolidation/enchanter/) - Mage + Crafter
- [Healer](./consolidation/healer/) - Mage + Herbalist

### Shaman Consolidations

- [War Shaman](./spiritcaller/shaman/war-shaman/) - Shaman + Warrior
- [Totem Carver](./spiritcaller/shaman/totem-carver/) - Shaman + Carpenter

### Other Consolidations

- [Ranger](./gatherer/hunter/ranger/) - Scout + Hunter
- [Artificer](./consolidation/index.md) - Crafter + Enchanter
