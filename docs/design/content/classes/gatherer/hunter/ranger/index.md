# Ranger

## Lore

Rangers are masters of the wild places, combining the Hunter's knowledge of beasts and terrain with the Scout's mobility and awareness. They serve as guides through dangerous wilderness, trackers of dangerous prey, and guardians of frontier settlements.

## Mechanics

### Prerequisites

| Requirement      | Value                                                                   |
| ---------------- | ----------------------------------------------------------------------- |
| XP Threshold     | 50,000 XP from wilderness hunting and tracking actions                  |
| Related Classes  | [Hunter](../) and [Scout](../../../fighter/scout/) (provide XP bonuses) |
| Tag Depth Access | 3+ levels                                                               |

### Primary Tags

- `Gathering/Hunting`
- `Combat/Ranged`
- `Stealth`

## Progression

This is a Tier 3 consolidation class. No further specializations available.
