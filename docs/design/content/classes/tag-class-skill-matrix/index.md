# Tag-Class-Skill-Spell Matrix

This document maps each skill tag to the classes, skills, and spells that reference it **exactly** (no inheritance).

**Legend:**

- **Depth**: Tag depth level (1=base, 2=specialization, 3=mastery)
- **Class Tiers**: T1=Foundation, T2=Specialist, T3=Consolidation

See [Class Tiers](../../systems/character/class-tiers.md) for tier definitions and [Skill Tags](../../systems/character/skill-tags.md) for tag depth mechanics.
