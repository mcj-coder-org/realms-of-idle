# Enhancement Enchantments

Enchantments focused on improving how items perform their primary functions - making tools work faster, weapons cut better, and items last longer.

## Available Enchantments

| Enchantment                 | Tier       | Summary                                               | Cost Range |
| --------------------------- | ---------- | ----------------------------------------------------- | ---------- |
| [Sharpness](sharpness.md)   | Apprentice | Maintains supernatural edge, increased cutting damage | 30-12000c  |
| [Efficiency](efficiency.md) | Apprentice | Faster work speed, reduced stamina cost               | 25-10000c  |
| [Fortune](fortune.md)       | Journeyman | Increased resource yields and rare item discovery     | 200-30000c |
| [Unbreaking](unbreaking.md) | Apprentice | Extended durability for tools and weapons             | 20-8000c   |
| [Durability](durability.md) | Apprentice | Fabric longevity enhancement (clothing-specific)      | 10-2500c   |

## Category Overview

**Primary Users:** Craftspeople, laborers, gatherers, everyone (everyday use)

**ROI Leader:** Efficiency pays for itself fastest (days to weeks) through increased productivity.

**Best Combinations:**

- Efficiency + Fortune + Unbreaking = Professional tool set (fast work, better yields, never breaks)
- Sharpness + Flaming/Frost/Lightning = Elemental cutting weapon
- Durability + Cleanliness = Eternal, clean clothing
- Durability + Comfort = Lifetime perfect wardrobe

**Budget Recommendations:**

- Essential (workers): Efficiency II (100c) + Unbreaking I (20c)
- Gatherers add: Fortune II (800c)
- Professional standard: All at Level III (~4000c)
- Heirloom investment: Auto-Repair + high-level enhancement

**Special Notes:**

- Sharpness only works on bladed items (no blunt weapons)
- Efficiency at 60% effectiveness on weapons (combat use)
- Fortune at 60% effectiveness for combat drops
- Durability specifically for fabrics; use Unbreaking for other materials
