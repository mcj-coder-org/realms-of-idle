# Races Library

All playable races are sentient and intelligent. NPCs and players use identical systems.

## Race Categories

| Category                    | Races                                       | Structure        |
| --------------------------- | ------------------------------------------- | ---------------- |
| [Enlightened](enlightened/) | Humans, Elves, Dwarves, Lizardfolk, Kobolds | Settlement-based |
| [Monstrous](monstrous/)     | Gnolls, Goblins, Orcs, Ogres, Trolls        | Tribal           |

## Spectrum Position

```
Monstrous ◄────────────────────────────────────────────► Enlightened

Trolls ─ Ogres ─ Orcs ─ Goblins ─ Gnolls ─ Kobolds ─ Lizardfolk ─ Dwarves ─ Humans ─ Elves
```
