# Consumables

Consumables are items that are used up and provide temporary or one-time effects.

## Consumable Items

- **[Food](food/)** - Basic sustenance for stamina recovery

## Uses

| Consumable | Effect           | Duration |
| ---------- | ---------------- | -------- |
| Food       | Restores stamina | Instant  |
