# Processed Materials

Processed materials have been refined from their raw state and are ready to use in crafting recipes.

## Processed Items

- **[Ingot](ingot/)** - Refined metal for smithing

## Production

| Recipe                                           | Input      | Output      |
| ------------------------------------------------ | ---------- | ----------- |
| [Smelt Ore](../../../recipe/smelting/smelt-ore/) | Ore + Coal | Metal Ingot |

## Used In

- **[Weapons](../weapon/)** - Forged from ingots
- **[Armor](../armor/)** - Crafted from ingots and components
- **[Tools](../tool/)** - Made from ingots and handles
