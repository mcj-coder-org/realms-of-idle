# Raw Materials

Raw materials are the direct output of gathering activities. They must be processed before being used in most crafting recipes.

## Raw Items

- **[Ore](ore/)** - Metal-bearing rock from mining

## Sources

| Class                                               | Source       | Raw Items                  |
| --------------------------------------------------- | ------------ | -------------------------- |
| [Miner](../../../classes/gatherer/miner/)           | Mining nodes | Copper Ore, Iron Ore, Coal |
| [Lumberjack](../../../classes/gatherer/lumberjack/) | Forests      | Timber, logs               |
| [Hunter](../../../classes/gatherer/hunter/)         | Creatures    | Hides, pelts               |

## Processing

Raw materials are processed into usable components:

- **Ore** → [Smelt Ore](../../../recipe/smelting/smelt-ore/) → [Ingot](../processed/ingot/)
- **Timber** → [Wood Handle](../../../recipe/processing/wood-handle/) → Wood Handle
- **Hides** → [Leather Strip](../../../recipe/processing/leather-strip/) → Leather Strip
