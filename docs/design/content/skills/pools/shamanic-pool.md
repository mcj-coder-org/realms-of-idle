# Shamanic Pool Skills

## Synergy Classes

Classes with strong synergies: Shaman, War Painter (tribal races)

**Synergy Benefits**: 2x XP acquisition, enhanced effectiveness, reduced costs (scales with class level)

## Skills

### Spirit Communion (Mechanic Unlock)

Communicate with tribal spirits.

| Effect   | Description                                               |
| -------- | --------------------------------------------------------- |
| Unlock   | Can communicate with spirits in tribal pool               |
| Benefits | Request guidance, learn tribal history, sense spirit mood |
| Ritual   | Requires brief ritual (10-30 minutes)                     |

**Note:** Core Shaman mechanic—offered at class acceptance

**Synergy Classes**: Shaman

**XP Sources**:

- Perform rituals: 10-20 XP per ritual
- Tend to spirits: 5-15 XP per session
- Guided training: 10-100 XP/session
- Unguided training: 5-50 XP/session

**Level-Up Availability**:

- Strong synergy: ~500 XP (~50 actions)
- Moderate synergy: ~1000 XP
- No synergy: ~1500 XP

**Synergy Examples**:

- **Strong (2x XP)**: Shaman Level 15 - Spirit specialist - Enhanced communication, extended rituals, 2x XP from communion
- **Moderate (1.5x XP)**: War Painter Level 10 - Spirit aware - Basic communication
- **None (1x XP)**: Warrior - No connection - No bonuses
