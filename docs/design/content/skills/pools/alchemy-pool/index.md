# Alchemy Pool Skills

## Synergy Classes

Classes with strong synergies: Alchemist, Brewer, Cook

**Synergy Benefits**: 2x XP acquisition, enhanced effectiveness, reduced costs (scales with class level)

## Skills

### Potent Brews (Tiered)

Stronger alchemical creations.

| Tier     | Effect                                             | Scaling                |
| -------- | -------------------------------------------------- | ---------------------- |
| Lesser   | +15% potion/poison effectiveness                   | +1% per class level    |
| Greater  | +25% effectiveness, +20% duration                  | +0.75% per class level |
| Enhanced | +40% effectiveness, +35% duration, no side effects | +0.5% per class level  |

**Synergy Classes**: Alchemist

**XP Sources**:

- Create potions/poisons: 5-15 XP per creation
- Experiment with recipes: 10-20 XP
- Guided training: 10-100 XP/session
- Unguided training: 5-50 XP/session

**Level-Up Availability**:

- Strong synergy: ~500 XP (~50 actions)
- Moderate synergy: ~1000 XP
- No synergy: ~1500 XP

**Synergy Examples**:

- **Strong (2x XP)**: Alchemist Level 15 - Brewing specialist - +50% effectiveness, +45% duration, 2x XP from brewing
- **Moderate (1.5x XP)**: Cook Level 10 - Recipe expert - +30% effectiveness
- **None (1x XP)**: Warrior - No connection - No bonuses

### Transmutation Basics (Mechanic Unlock)

Convert materials.

| Effect     | Description                                      |
| ---------- | ------------------------------------------------ |
| Unlock     | Can convert common materials (iron→copper, etc.) |
| Ratio      | 3:1 or worse conversion ratio                    |
| Limitation | Cannot create rare from common                   |

**Tier 2:** Advanced Transmutation—2:1 ratio, can create uncommon
**Tier 3:** Master Transmutation—1:1 ratio, can create rare (with rare catalyst)

**Prerequisite:** Master rank

**Synergy Classes**: Alchemist

**XP Sources**:

- Experiment with transmutation: 15-30 XP per experiment
- Study material properties: 10-20 XP
- Guided training: 10-100 XP/session
- Unguided training: 5-50 XP/session

**Level-Up Availability**:

- Strong synergy: ~500 XP (~50 actions)
- Moderate synergy: ~1000 XP
- No synergy: ~1500 XP

**Synergy Examples**:

- **Strong (2x XP)**: Alchemist Level 30 - Transmutation specialist - 2:1 ratio, rare creation, 2x XP from transmutation
- **Moderate (1.5x XP)**: Mage Level 20 - Material transformation - Better ratios
- **None (1x XP)**: Farmer - No connection - No bonuses

### Explosive Compounds (Mechanic Unlock)

Create alchemical explosives.

| Effect | Description                                     |
| ------ | ----------------------------------------------- |
| Unlock | Can craft bombs, grenades, explosive traps      |
| Types  | Fire, smoke, poison, flashbang                  |
| Safety | Handle explosives without accidental detonation |

**No scaling—binary unlock (damage scales with recipe)**

**Note:** May be restricted or monitored

**Synergy Classes**: Alchemist

**XP Sources**:

- Experiment with volatile compounds: 15-30 XP
- Study explosives: 10-20 XP
- Craft explosives: 10-25 XP per explosive
- Guided training: 10-100 XP/session
- Unguided training: 5-50 XP/session

**Level-Up Availability**:

- Strong synergy: ~500 XP (~50 actions)
- Moderate synergy: ~1000 XP
- No synergy: ~1500 XP

**Synergy Examples**:

- **Strong (2x XP)**: Alchemist Level 15 - Explosives specialist - +25% potency, safer handling, 2x XP from explosive creation
- **Moderate (1.5x XP)**: Engineer Level 10 - Demolition expert - Better placement
- **None (1x XP)**: Farmer - No connection - No bonuses
