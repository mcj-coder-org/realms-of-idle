# Magic Pool Skills

## Synergy Classes

Classes with strong synergies: Mage, Enchanter, Necromancer (mana-based)

**Synergy Benefits**: 2x XP acquisition, enhanced effectiveness, reduced costs (scales with class level)

## Skills

### Spell Focus (Tiered)

Enhanced spell effectiveness.

| Tier     | Effect                                      | Scaling               |
| -------- | ------------------------------------------- | --------------------- |
| Lesser   | +10% spell damage/healing, -10% mana cost   | +0.5% per class level |
| Greater  | +18% effect, -18% mana cost                 | +0.4% per class level |
| Enhanced | +25% effect, -25% mana cost, faster casting | +0.3% per class level |

**Synergy Classes**: All Magic pool

**XP Sources**:

- Cast spells: 3-8 XP per cast
- Practice magic: 5-15 XP per session
- Guided training: 10-100 XP/session
- Unguided training: 5-50 XP/session

**Level-Up Availability**:

- Strong synergy: ~500 XP (~50 actions)
- Moderate synergy: ~1000 XP
- No synergy: ~1500 XP

**Synergy Examples**:

- **Strong (2x XP)**: Mage Level 15 - Spellcasting specialist - +30% effect, -30% cost, 2x XP from casting
- **Moderate (1.5x XP)**: Enchanter Level 10 - Magic user - +20% effect
- **None (1x XP)**: Warrior - No connection - No bonuses
