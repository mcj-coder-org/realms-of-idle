# Dark Pool Skills

## Synergy Classes

Classes with strong synergies: Thief, Assassin, Smuggler, Fence, Extortionist

**Synergy Benefits**: 2x XP acquisition, enhanced effectiveness, reduced costs (scales with class level)

## Skills

### Underworld Connections (Passive Generator)

Criminal network contacts.

| Tier     | Effect                                                |
| -------- | ----------------------------------------------------- |
| Lesser   | Weekly: Learn of 1-2 criminal opportunities           |
| Greater  | Weekly: 2-4 opportunities, fence access, safe houses  |
| Enhanced | Weekly: 3-6 opportunities, exclusive jobs, protection |

**Synergy Classes**: All Dark pool

**XP Sources**:

- Operate in underground: 5-10 XP per week
- Build criminal reputation: 10-20 XP per milestone
- Guided training: 10-100 XP/session
- Unguided training: 5-50 XP/session

**Level-Up Availability**:

- Strong synergy: ~500 XP (~50 actions)
- Moderate synergy: ~1000 XP
- No synergy: ~1500 XP

**Synergy Examples**:

- **Strong (2x XP)**: Thief Level 15 - Underground specialist - 4-8 opportunities, exclusive access, 2x XP from underworld work
- **Moderate (1.5x XP)**: Smuggler Level 10 - Criminal network - 3-5 opportunities
- **None (1x XP)**: Knight - No connection - No bonuses

### Evidence Elimination (Tiered)

Remove traces of crimes.

| Tier     | Effect                                                | Scaling                |
| -------- | ----------------------------------------------------- | ---------------------- |
| Lesser   | -25% chance of evidence at crime scene                | +1% per class level    |
| Greater  | -50% evidence chance, can remove existing evidence    | +0.75% per class level |
| Enhanced | -75% evidence, remove evidence remotely, frame others | +0.5% per class level  |

**Synergy Classes**: All Dark pool

**XP Sources**:

- Commit crimes cleanly: 10-20 XP per clean crime
- Remove evidence: 15-30 XP per removal
- Guided training: 10-100 XP/session
- Unguided training: 5-50 XP/session

**Level-Up Availability**:

- Strong synergy: ~500 XP (~50 actions)
- Moderate synergy: ~1000 XP
- No synergy: ~1500 XP

**Synergy Examples**:

- **Strong (2x XP)**: Assassin Level 15 - Evidence specialist - -85% evidence, frame others, 2x XP from evidence work
- **Moderate (1.5x XP)**: Thief Level 10 - Clean operator - -60% evidence
- **None (1x XP)**: Knight - No connection - No bonuses
