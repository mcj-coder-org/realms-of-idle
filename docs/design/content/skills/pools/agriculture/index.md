# Agriculture Skills

## Overview

Skills focused on farming, crop cultivation, and animal husbandry. These skills excel in sustainable food production, managing livestock, and optimizing agricultural yields.

**Synergy Classes**: Farmer, Rancher, Beekeeper

**Synergy Benefits**:

- 2x XP acquisition from relevant actions
- Enhanced effectiveness (scales with class level)
- Reduced costs (scales with class level)

## Skills in This Pool

### Tiered Skills

- [Green Thumb](green-thumb/index.md) - Improved crop growth
- [Animal Husbandry](animal-husbandry/index.md) - Better livestock management

### Mechanic Unlocks

- [Season Sense](season-sense/index.md) - Perfect agricultural timing
- [Pest Resistance](pest-resistance/index.md) - Protect crops from pests and disease
- [Beekeeper's Bond](beekeepers-bond/index.md) - Work with bees safely and productively

### Passive Generators

- [Surplus Generator](surplus-generator/index.md) - Regular agricultural surplus production
