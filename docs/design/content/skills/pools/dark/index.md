# Dark Skills

## Overview

Skills focused on criminal activities, underground operations, and morally questionable practices. These skills excel in illegal profit, avoiding law enforcement, and operating in the criminal underworld.

**Synergy Classes**: Thief, Assassin, Smuggler, Fence, Extortionist

**Synergy Benefits**:

- 2x XP acquisition from relevant actions
- Enhanced effectiveness (scales with class level)
- Reduced costs (scales with class level)

## Skills in This Pool

### Tiered Skills

None - Dark skills are primarily unlocks and generators

### Mechanic Unlocks

- [Extortion](extortion/index.md) - Extract payment through threats
- [Contract Killing](contract-killing/index.md) - Accept assassination jobs
- [Blackmail](blackmail/index.md) - Leverage secrets for profit
- [False Identity](false-identity/index.md) - Create and maintain fake identities
- [Evidence Elimination](evidence-elimination/index.md) - Remove traces of crimes
- [Money Laundering](money-laundering/index.md) - Clean illegal profits

### Passive Generators

- [Underworld Connections](underworld-connections/index.md) - Criminal network contacts
