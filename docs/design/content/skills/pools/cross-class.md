# Cross-Class Skills

## Overview

Some skills are available to multiple unrelated classes based on thematic fit rather than a single pool. These skills synergize with specific classes from different domains.

## Skills

### Water Breathing (Mechanic Unlock)

Breathe underwater indefinitely.

| Effect        | Description                           |
| ------------- | ------------------------------------- |
| Unlock        | Can breathe underwater without limit  |
| Depth         | No pressure damage to moderate depths |
| Communication | Can speak underwater (muffled)        |

**Synergy Classes**: Fisher, Sailor, certain Mage schools

**XP Sources**:

- Extended underwater activity: 10-20 XP per session
- Near-drowning survival: 30-50 XP
- Guided training: 10-100 XP/session
- Unguided training: 5-50 XP/session

**Level-Up Availability**:

- Strong synergy: ~500 XP (~50 actions)
- Moderate synergy: ~1000 XP
- No synergy: ~1500 XP

**Synergy Examples**:

- **Strong (2x XP)**: Fisher Level 15 - Aquatic specialist - Extended depth tolerance, 2x XP from underwater work
- **Moderate (1.5x XP)**: Sailor Level 10 - Water worker - Standard depth
- **None (1x XP)**: Blacksmith - No connection - No bonuses

### Night Vision (Mechanic Unlock)

See in complete darkness.

| Effect     | Description                         |
| ---------- | ----------------------------------- |
| Unlock     | Full vision in any darkness         |
| Quality    | Color vision, full detail           |
| Limitation | Bright light causes brief blindness |

**Synergy Classes**: Thief, Scout, Miner, certain races (improved version)

**XP Sources**:

- Extended time in darkness: 5-10 XP per session
- Adapt to underground: 10-20 XP
- Guided training: 10-100 XP/session
- Unguided training: 5-50 XP/session

**Level-Up Availability**:

- Strong synergy: ~500 XP (~50 actions)
- Moderate synergy: ~1000 XP
- No synergy: ~1500 XP

**Synergy Examples**:

- **Strong (2x XP)**: Miner Level 15 - Underground specialist - Enhanced range, no light blindness, 2x XP from dark work
- **Moderate (1.5x XP)**: Thief Level 10 - Night operator - Standard vision
- **None (1x XP)**: Farmer - No connection - No bonuses

### Ambidexterity (Mechanic Unlock)

Use either hand equally.

| Effect   | Description                        |
| -------- | ---------------------------------- |
| Unlock   | No off-hand penalty for any action |
| Combat   | Full damage with either weapon     |
| Crafting | Can work with either hand          |

**Synergy Classes**: Warrior, Thief, Crafter specialties

**XP Sources**:

- Practice off-hand: 5-10 XP per session
- Train both hands: 10-20 XP per training
- Guided training: 10-100 XP/session
- Unguided training: 5-50 XP/session

**Level-Up Availability**:

- Strong synergy: ~500 XP (~50 actions)
- Moderate synergy: ~1000 XP
- No synergy: ~1500 XP

**Synergy Examples**:

- **Strong (2x XP)**: Warrior Level 15 - Dual fighter - +10% attack speed both hands, 2x XP from ambidextrous work
- **Moderate (1.5x XP)**: Blacksmith Level 10 - Two-handed crafter - Better efficiency
- **None (1x XP)**: Scholar - No connection - No bonuses

### Danger Sense (Enhanced) (Mechanic Unlock)

Supernatural threat awareness.

| Effect  | Description                                  |
| ------- | -------------------------------------------- |
| Unlock  | Sense danger before it manifests             |
| Types   | Traps, ambushes, betrayal, natural disasters |
| Warning | Brief (seconds) but reliable                 |

**Synergy Classes**: Scout, Assassin, Adventurer, certain Mage paths

**XP Sources**:

- Survive ambushes: 15-30 XP per survival
- Avoid disasters: 20-40 XP
- Guided training: 10-100 XP/session
- Unguided training: 5-50 XP/session

**Level-Up Availability**:

- Strong synergy: ~500 XP (~50 actions)
- Moderate synergy: ~1000 XP
- No synergy: ~1500 XP

**Synergy Examples**:

- **Strong (2x XP)**: Scout Level 15 - Danger specialist - Extended warning (5-10 sec), 2x XP from danger avoidance
- **Moderate (1.5x XP)**: Assassin Level 10 - Threat aware - Standard warning
- **None (1x XP)**: Merchant - No connection - No bonuses

### Iron Will (Mechanic Unlock)

Immunity to mental control.

| Effect     | Description                               |
| ---------- | ----------------------------------------- |
| Unlock     | Immune to charm, domination, mind control |
| Resistance | +50% vs other mental effects              |
| Detection  | Know when mental attack attempted         |

**Synergy Classes**: Knight, certain Mage schools, Shaman

**XP Sources**:

- Resist mental attacks: 20-40 XP per resistance
- Maintain mental discipline: 10-20 XP per session
- Guided training: 10-100 XP/session
- Unguided training: 5-50 XP/session

**Level-Up Availability**:

- Strong synergy: ~500 XP (~50 actions)
- Moderate synergy: ~1000 XP
- No synergy: ~1500 XP

**Synergy Examples**:

- **Strong (2x XP)**: Knight Level 15 - Mental fortitude specialist - Immunity to all mental effects, 2x XP from mental resistance
- **Moderate (1.5x XP)**: Mage Level 10 - Mind defender - +65% resistance
- **None (1x XP)**: Farmer - No connection - No bonuses
