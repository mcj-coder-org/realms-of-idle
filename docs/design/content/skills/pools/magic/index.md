# Magic Skills

## Overview

Skills focused on mana manipulation, spellcasting, enchantment, and arcane mastery. These skills excel in harnessing magical energy, combining spell effects, and understanding mystical forces.

**Synergy Classes**: Mage, Enchanter, Necromancer (mana-based)

**Synergy Benefits**:

- 2x XP acquisition from relevant actions
- Enhanced effectiveness (scales with class level)
- Reduced costs (scales with class level)

## Skills in This Pool

### Tiered Skills

- [Spell Focus](spell-focus.md) - Enhanced spell effectiveness
- [Mana Well](mana-well.md) - Expanded magical reserves
- [School Mastery](school-mastery.md) - Deep specialization in magic school

### Mechanic Unlocks

- [Spell Weaving](spell-weaving.md) - Combine spell effects

- [Mana Transfer](mana-transfer.md) - Share mana with others
- [Counter Magic](counter-magic.md) - Dispel and counter enemy spells
- [Undead Binding](undead-binding.md) - Command and control undead
