# Leadership Pool Skills

## Synergy Classes

Classes with strong synergies: Leader, Chieftain, Guildmaster, Settlement Leader

**Synergy Benefits**: 2x XP acquisition, enhanced effectiveness, reduced costs (scales with class level)

## Skills

### Inspiring Leader (Tiered)

Enhance follower capabilities.

| Tier     | Effect                                            | Range    | Scaling               |
| -------- | ------------------------------------------------- | -------- | --------------------- |
| Lesser   | Followers +10% all stats                          | Medium   | +0.5% per class level |
| Greater  | Followers +18% stats, +15% morale                 | Far      | +0.4% per class level |
| Enhanced | Followers +25% stats, +25% morale, immune to fear | Very Far | +0.3% per class level |

**Synergy Classes**: Leader, Chieftain, Knight

**XP Sources**:

- Lead followers: 5-10 XP per session
- Inspire through action: 10-20 XP per inspiring deed
- Guided training: 10-100 XP/session
- Unguided training: 5-50 XP/session

**Level-Up Availability**:

- Strong synergy: ~500 XP (~50 actions)
- Moderate synergy: ~1000 XP
- No synergy: ~1500 XP

**Synergy Examples**:

- **Strong (2x XP)**: Leader Level 15 - Inspiration specialist - +30% stats, immune to fear, 2x XP from leading
- **Moderate (1.5x XP)**: Knight Level 10 - Commander - +20% stats
- **None (1x XP)**: Thief - No connection - No bonuses
