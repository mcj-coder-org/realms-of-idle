# Smithing Skills

## Overview

Skills focused on metalworking, forging, tempering, and creating weapons and armor. These skills excel in working with metal, understanding forge techniques, and crafting durable equipment.

**Synergy Classes**: Blacksmith, Weaponsmith, Armorer

**Synergy Benefits**:

- 2x XP acquisition from relevant actions
- Enhanced effectiveness (scales with class level)
- Reduced costs (scales with class level)

## Skills in This Pool

### Tiered Skills

- [Perfect Temper](perfect-temper/index.md) - Superior metal tempering technique
- [Weapon Enhancement](weapon-enhancement/index.md) - Create enhanced weapon properties

### Mechanic Unlocks

- [Metal Sense](metal-sense/index.md) - Identify metal quality and properties
- [Forge Fire](forge-fire/index.md) - Mastery of forge temperature control
- [Armor Fitting](armor-fitting/index.md) - Custom fit armor for maximum effectiveness
