# Shamanic Skills

## Overview

Skills focused on spirit communion, tribal magic, totem crafting, and ancestral connection. These skills excel in working with spirits, creating sacred objects, and channeling ancestral power.

**Synergy Classes**: Shaman, War Painter (tribal races)

**Synergy Benefits**:

- 2x XP acquisition from relevant actions
- Enhanced effectiveness (scales with class level)
- Reduced costs (scales with class level)

## Skills in This Pool

### Tiered Skills

- [Spirit Communion](spirit-communion/index.md) - Enhanced communication with spirits
- [Expanded Spirit Reach](expanded-spirit-reach/index.md) - Increased range for spirit interaction

### Mechanic Unlocks

- [Spirit Walk](spirit-walk/index.md) - Enter spirit realm
- [Spirit Binding](spirit-binding/index.md) - Bind spirits to service
- [Totem Crafting](totem-crafting/index.md) - Create sacred totems
- [Fetish Creation](fetish-creation/index.md) - Craft magical fetishes
- [Ancestral Memory](ancestral-memory/index.md) - Access ancestral knowledge
- [War Painter](war-painter/index.md) - Apply sacred war paint for bonuses
