# Shamanic Skills

## Overview

Skills focused on spirit communion, tribal magic, totem crafting, and ancestral connection. These skills excel in working with spirits, creating sacred objects, and channeling ancestral power.

**Synergy Classes**: Shaman, War Painter (tribal races)

**Synergy Benefits**:

- 2x XP acquisition from relevant actions
- Enhanced effectiveness (scales with class level)
- Reduced costs (scales with class level)

## Skills in This Pool

### Tiered Skills

- [Spirit Communion](spirit-communion.md) - Enhanced communication with spirits
- [Expanded Spirit Reach](expanded-spirit-reach.md) - Increased range for spirit interaction

### Mechanic Unlocks

- [Spirit Walk](spirit-walk.md) - Enter spirit realm
- [Spirit Binding](spirit-binding.md) - Bind spirits to service
- [Totem Crafting](totem-crafting.md) - Create sacred totems
- [Fetish Creation](fetish-creation.md) - Craft magical fetishes
- [Ancestral Memory](ancestral-memory.md) - Access ancestral knowledge
- [War Painter](war-painter.md) - Apply sacred war paint for bonuses
