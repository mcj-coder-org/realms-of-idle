# Smithing Pool Skills

## Synergy Classes

Classes with strong synergies: Blacksmith, Weaponsmith, Armorer

**Synergy Benefits**: 2x XP acquisition, enhanced effectiveness, reduced costs (scales with class level)

## Skills

### Metal Sense (Mechanic Unlock)

Intuitive understanding of metals.

| Effect      | Description                                               |
| ----------- | --------------------------------------------------------- |
| Unlock      | Know optimal temperature, timing, technique for any metal |
| Assessment  | Identify metal type, purity, origin by inspection         |
| Combination | Know which alloys work (and which are impossible)         |

**No scaling—binary unlock**

**Synergy Classes**: Blacksmith, Weaponsmith, Armorer

**XP Sources**:

- Work with varied metals: 5-10 XP per metal type
- Experiment with alloys: 15-30 XP per experiment
- Guided training: 10-100 XP/session
- Unguided training: 5-50 XP/session

**Level-Up Availability**:

- Strong synergy: ~500 XP (~50 actions)
- Moderate synergy: ~1000 XP
- No synergy: ~1500 XP

**Synergy Examples**:

- **Strong (2x XP)**: Blacksmith Level 15 - Metal specialist - Instant assessment, perfect techniques, 2x XP from metalworking
- **Moderate (1.5x XP)**: Miner Level 10 - Metal handler - Better metal identification
- **None (1x XP)**: Farmer - No connection - No bonuses

### Perfect Temper (Tiered)

Superior heat treatment.

| Tier     | Effect                                                   | Scaling               |
| -------- | -------------------------------------------------------- | --------------------- |
| Lesser   | +10% weapon damage OR +10% armor protection              | +0.5% per class level |
| Greater  | +18% damage/protection, +15% durability                  | +0.4% per class level |
| Enhanced | +25% damage/protection, +25% durability, resist breaking | +0.3% per class level |

**Synergy Classes**: Blacksmith, Weaponsmith, Armorer

**XP Sources**:

- Forge quality weapons/armor: 10-20 XP per item
- Practice heat treatment: 5-15 XP per session
- Guided training: 10-100 XP/session
- Unguided training: 5-50 XP/session

**Level-Up Availability**:

- Strong synergy: ~500 XP (~50 actions)
- Moderate synergy: ~1000 XP
- No synergy: ~1500 XP

**Synergy Examples**:

- **Strong (2x XP)**: Weaponsmith Level 15 - Tempering specialist - +30% damage, +30% durability, 2x XP from tempering
- **Moderate (1.5x XP)**: Blacksmith Level 10 - Heat treatment expert - +20% effect
- **None (1x XP)**: Thief - No connection - No bonuses
