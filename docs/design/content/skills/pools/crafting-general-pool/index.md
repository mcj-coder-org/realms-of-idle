# Crafting (General) Pool Skills

## Synergy Classes

Classes with strong synergies: All crafting specialties (Blacksmith, Weaponsmith, Armorer, Alchemist, Carpenter, Bowyer, Tailor, Leatherworker, Brewer, Cook, and others)

**Synergy Benefits**: 2x XP acquisition, enhanced effectiveness, reduced costs (scales with class level)

## Skills

### Artisan's Focus (Tiered)

Enhanced crafting concentration.

| Tier     | Effect                                                       | Scaling               |
| -------- | ------------------------------------------------------------ | --------------------- |
| Lesser   | +10% quality, -15% failure chance                            | +0.5% per class level |
| Greater  | +18% quality, -30% failure chance, sense flaws               | +0.4% per class level |
| Enhanced | +25% quality, -50% failure chance, auto-correct minor errors | +0.3% per class level |

**Synergy Classes**: All Crafting specialties

**XP Sources**:

- Complete quality crafts: 5-15 XP per craft
- Achieve exceptional results: 15-30 XP
- Guided training: 10-100 XP/session
- Unguided training: 5-50 XP/session

**Level-Up Availability**:

- Strong synergy: ~500 XP (~50 actions)
- Moderate synergy: ~1000 XP
- No synergy: ~1500 XP

**Synergy Examples**:

- **Strong (2x XP)**: Blacksmith Level 15 - Crafting specialist - +30% quality, -60% failure, 2x XP from focused crafting
- **Moderate (1.5x XP)**: Carpenter Level 10 - Craftsperson - +20% quality
- **None (1x XP)**: Warrior - No connection - No bonuses
