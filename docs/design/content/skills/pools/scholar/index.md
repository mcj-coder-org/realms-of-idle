# Scholar Skills

## Overview

Skills focused on knowledge, research, teaching, and learning. These skills excel in acquiring information, preserving knowledge, training others, and accessing ancient wisdom.

**Synergy Classes**: Scholar, Scribe, Trainer

**Synergy Benefits**:

- 2x XP acquisition from relevant actions
- Enhanced effectiveness (scales with class level)
- Reduced costs (scales with class level)

## Skills in This Pool

### Tiered Skills

- [Rapid Learning](rapid-learning.md) - Learn new skills faster
- [Perfect Memory](perfect-memory.md) - Enhanced recall of information

### Mechanic Unlocks

- [Research](research.md) - Conduct systematic investigation
- [Teaching](teaching.md) - Train others effectively
- [Linguistic Mastery](linguistic-mastery.md) - Learn languages quickly
- [Ancient Knowledge](ancient-knowledge.md) - Access forgotten lore
- [Skill Transference](skill-transference.md) - Pass knowledge to students
