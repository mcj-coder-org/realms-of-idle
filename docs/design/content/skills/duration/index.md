# Duration Skills

Skills that provide effects lasting for a set duration.

## Mechanics

| Aspect     | Description                        |
| ---------- | ---------------------------------- |
| Activation | Active use to trigger              |
| Duration   | Effect lasts for specified time    |
| Expiration | Effect ends when duration complete |
| Extension  | Some tiers extend duration         |

## Duration Scaling

| Tier     | Typical Duration |
| -------- | ---------------- |
| Lesser   | 30s - 2min       |
| Greater  | 2min - 5min      |
| Enhanced | 5min - 15min     |

## Planned Skills

_This category is reserved for future skill implementations._

Skills in this category will include temporary buffs, protective effects, and enhanced states with fixed durations.
