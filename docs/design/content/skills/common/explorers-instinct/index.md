# Explorer's Instinct

**Pool**: Common
**Type**: Passive
**Universal Access**: Available to all classes

## Description

Natural instinct for exploration increases discovery rate and exploration XP.

## Tiers

| Tier     | Base Effect                                 | Scaling          |
| -------- | ------------------------------------------- | ---------------- |
| Lesser   | +10% discover locations, +5% exploration XP | +1% per level    |
| Greater  | +20% discover, +10% XP, sense POIs          | +0.75% per level |
| Enhanced | +30% discover, +15% XP, identify POIs       | +0.5% per level  |

## Acquisition

**Trigger Actions**: Discover new locations, explore unknown areas, find points of interest
**Influence Rate**: 1.0 per new location, 0.5 per POI found

## Scaling

Scales with total combined class levels (rewards generalists).
