# Swift

**Pool**: Common
**Type**: Passive
**Universal Access**: Available to all classes

## Description

Move faster in all situations, with enhanced sprint speed at higher tiers.

## Tiers

| Tier     | Base Effect               | Scaling         |
| -------- | ------------------------- | --------------- |
| Lesser   | +5% movement speed        | +0.5% per level |
| Greater  | +10% movement speed       | +0.4% per level |
| Enhanced | +15% movement, +5% sprint | +0.3% per level |

## Acquisition

**Trigger Actions**: Travel long distances, run/sprint frequently, chase or flee
**Influence Rate**: 0.1 per hour of travel, 0.5 per chase/flee situation

## Scaling

Scales with total combined class levels (rewards generalists).
