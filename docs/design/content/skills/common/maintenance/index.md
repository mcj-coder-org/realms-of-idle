# Maintenance

**Pool**: Common
**Type**: Passive
**Universal Access**: Available to all classes

## Description

Reduce equipment degradation through proper care and perform field repairs.

## Tiers

| Tier     | Base Effect                              | Scaling         |
| -------- | ---------------------------------------- | --------------- |
| Lesser   | -10% equipment degradation               | +0.5% per level |
| Greater  | -20% degradation, basic field repairs    | +0.4% per level |
| Enhanced | -30% degradation, moderate field repairs | +0.3% per level |

## Acquisition

**Trigger Actions**: Repair equipment, maintain gear, use equipment extensively
**Influence Rate**: 0.5 per repair, 0.1 per day of equipment use

## Scaling

Scales with total combined class levels (rewards generalists).
