# Careful Hands

**Pool**: Common
**Type**: Passive
**Universal Access**: Available to all classes

## Description

Precise and careful craftsmanship improves item quality and reduces the chance of critical failures.

## Tiers

| Tier     | Base Effect                            | Scaling         |
| -------- | -------------------------------------- | --------------- |
| Lesser   | +5% crafting quality                   | +0.5% per level |
| Greater  | +10% crafting quality                  | +0.4% per level |
| Enhanced | +15% quality, reduced critical failure | +0.3% per level |

## Acquisition

**Trigger Actions**: Craft items, achieve quality results, complete fine work
**Influence Rate**: 0.2 per craft, 1.0 per quality+ result

## Scaling

Scales with total combined class levels (rewards generalists).
