# Quick Recovery

**Pool**: Common
**Type**: Passive
**Universal Access**: Available to all classes

## Description

Recover stamina more efficiently from rest, making downtime more effective.

## Tiers

| Tier     | Base Effect                              | Scaling          |
| -------- | ---------------------------------------- | ---------------- |
| Lesser   | Rest recovers stamina 10% faster         | +1% per level    |
| Greater  | Rest 20% faster, naps 15% more effective | +0.75% per level |
| Enhanced | Rest 30% faster, naps 25% more effective | +0.5% per level  |

## Acquisition

**Trigger Actions**:

- Complete rest cycles
- Take naps
- Wake naturally (not interrupted)

**Influence Rate**: 0.25 per full rest, 0.1 per nap

## Scaling

Scales with total combined class levels (rewards generalists).
