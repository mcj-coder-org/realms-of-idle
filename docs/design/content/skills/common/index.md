# Common Skills

## Overview

Universal skills available to all classes without specialized synergies. These skills provide foundational abilities, general improvements, and broadly useful capabilities that any character can develop.

**Universal Access**: Available to all classes

**Acquisition**: Common skills use a base XP rate without class-specific multipliers, making them equally accessible to everyone.
