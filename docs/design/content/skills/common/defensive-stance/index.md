# Defensive Stance

**Pool**: Common
**Type**: Toggle
**Universal Access**: Available to all classes

## Description

Adopt a defensive posture, taking less damage while reducing your offensive capability.

## Tiers

| Tier     | Base Effect            | Scaling          | Cost           |
| -------- | ---------------------- | ---------------- | -------------- |
| Lesser   | -15% taken, -20% dealt | +1% per level    | Low drain      |
| Greater  | -25% taken, -15% dealt | +0.75% per level | Low drain      |
| Enhanced | -35% taken, -10% dealt | +0.5% per level  | Very low drain |

## Acquisition

**Trigger Actions**:

- Block attacks
- Survive while outnumbered
- Use shields

**Influence Rate**: 0.1 per combat survived while wounded, 0.5 per outnumbered survival

## Scaling

Scales with total combined class levels (rewards generalists).
