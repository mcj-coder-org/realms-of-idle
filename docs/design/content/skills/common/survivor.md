# Survivor

**Pool**: Common
**Type**: Passive
**Universal Access**: Available to all classes

## Description

General resilience to harsh conditions, reducing stamina drain and increasing exposure resistance.

## Tiers

| Tier     | Base Effect                          | Scaling          |
| -------- | ------------------------------------ | ---------------- |
| Lesser   | -10% stamina drain from hazards      | +1% per level    |
| Greater  | -20% drain, +10% exposure resistance | +0.75% per level |
| Enhanced | -30% drain, +20% resistance          | +0.5% per level  |

## Acquisition

**Trigger Actions**:

- Survive environmental hazards
- Travel through dangerous terrain
- Endure harsh conditions

**Influence Rate**: 0.5 per hazard exposure, 1.0 per condition survived

## Scaling

Scales with total combined class levels (rewards generalists).
