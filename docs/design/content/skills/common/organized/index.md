# Organized

**Pool**: Common
**Type**: Passive
**Universal Access**: Available to all classes

## Description

Manage inventory more effectively through better organization and stacking.

## Tiers

| Tier     | Base Effect                            | Scaling         |
| -------- | -------------------------------------- | --------------- |
| Lesser   | +10% effective inventory (stacking)    | +0.5% per level |
| Greater  | +20% effective inventory, auto-sort    | +0.4% per level |
| Enhanced | +30% effective inventory, quick-access | +0.3% per level |

## Acquisition

**Trigger Actions**: Manage inventory, organize storage, access items efficiently
**Influence Rate**: 0.1 per inventory organization, 0.3 per efficient access

## Scaling

Scales with total combined class levels (rewards generalists).
