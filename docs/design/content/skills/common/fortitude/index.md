# Fortitude

**Pool**: Common
**Type**: Passive
**Universal Access**: Available to all classes

## Description

Natural resistance to poisons and diseases, protecting you from environmental and combat hazards.

## Tiers

| Tier     | Base Effect                       | Scaling          |
| -------- | --------------------------------- | ---------------- |
| Lesser   | +10% poison/disease resistance    | +1% per level    |
| Greater  | +20% poison/disease resistance    | +0.75% per level |
| Enhanced | +30% resistance, reduced duration | +0.5% per level  |

## Acquisition

**Trigger Actions**:

- Survive poison
- Recover from disease
- Consume antidotes

**Influence Rate**: 1.0 per poison survived, 2.0 per disease recovered

## Scaling

Scales with total combined class levels (rewards generalists).
