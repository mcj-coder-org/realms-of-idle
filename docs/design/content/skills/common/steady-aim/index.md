# Steady Aim

**Pool**: Common
**Type**: Toggle
**Universal Access**: Available to all classes

## Description

Focus carefully on ranged attacks to improve accuracy, trading attack speed for precision.

## Tiers

| Tier     | Base Effect               | Scaling          | Cost           |
| -------- | ------------------------- | ---------------- | -------------- |
| Lesser   | +10% accuracy, -10% speed | +1% per level    | Low drain      |
| Greater  | +18% accuracy, -5% speed  | +0.75% per level | Low drain      |
| Enhanced | +25% accuracy, no penalty | +0.5% per level  | Very low drain |

## Acquisition

**Trigger Actions**:

- Land ranged attacks
- Kill enemies with ranged
- Use bows/crossbows

**Influence Rate**: 0.05 per ranged kill, 0.2 per long-range kill

## Scaling

Scales with total combined class levels (rewards generalists).
