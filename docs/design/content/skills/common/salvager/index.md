# Salvager

**Pool**: Common
**Type**: Passive
**Universal Access**: Available to all classes

## Description

Recover more materials when breaking down items, eventually salvaging heavily damaged equipment.

## Tiers

| Tier     | Base Effect                             | Scaling         |
| -------- | --------------------------------------- | --------------- |
| Lesser   | +10% materials from breakdown           | +0.5% per level |
| Greater  | +20% materials, salvage damaged items   | +0.4% per level |
| Enhanced | +30% materials, salvage heavily damaged | +0.3% per level |

## Acquisition

**Trigger Actions**: Break down items, salvage equipment, recover materials
**Influence Rate**: 0.2 per salvage, 1.0 per valuable salvage

## Scaling

Scales with total combined class levels (rewards generalists).
