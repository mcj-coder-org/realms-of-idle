# Adrenaline Surge

**Pool**: Common
**Type**: Passive
**Universal Access**: Available to all classes

## Description

When critically wounded, adrenaline kicks in to restore stamina and temporarily boost damage.

## Tiers

| Tier     | Base Effect                               | Scaling         |
| -------- | ----------------------------------------- | --------------- |
| Lesser   | 5% stamina when below 25% HP              | +0.5% per level |
| Greater  | 10% stamina when below 30% HP             | +0.4% per level |
| Enhanced | 15% stamina below 35% HP, +10% damage 10s | +0.3% per level |

## Acquisition

**Trigger Actions**: Drop below 25% health in combat, survive near-death, fight while wounded
**Influence Rate**: 2.0 per near-death combat survival, 0.5 per wounded combat

## Scaling

Scales with total combined class levels (rewards generalists).
