# Group Fortitude

**Pool**: Common
**Type**: Passive
**Universal Access**: Available to all classes

## Description

Your presence strengthens party defenses against conditions, with shared recovery at higher tiers.

## Tiers

| Tier     | Base Effect                            | Scaling         | Range  |
| -------- | -------------------------------------- | --------------- | ------ |
| Lesser   | Party +5% condition resistance         | +0.5% per level | Close  |
| Greater  | Party +10% resistance, shared recovery | +0.4% per level | Medium |
| Enhanced | Party +15% resistance, faster recovery | +0.3% per level | Far    |

## Acquisition

**Trigger Actions**: Protect party members, help party recover, endure hardship together
**Influence Rate**: 0.3 per party protection action, 1.0 per party hardship survived

## Scaling

Scales with total combined class levels (rewards generalists).
