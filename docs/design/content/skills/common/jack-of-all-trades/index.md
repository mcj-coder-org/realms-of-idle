# Jack of All Trades

**Pool**: Common
**Type**: Passive
**Universal Access**: Available to all classes

## Description

Attempt unfamiliar tasks with less penalty, eventually having untrained successes count toward unlocks.

## Tiers

| Tier     | Base Effect                                        | Scaling          |
| -------- | -------------------------------------------------- | ---------------- |
| Lesser   | -15% untrained penalty                             | +1% per level    |
| Greater  | -30% untrained penalty                             | +0.75% per level |
| Enhanced | -50% penalty, untrained success counts for unlocks | +0.5% per level  |

## Acquisition

**Trigger Actions**: Attempt untrained actions, succeed at unfamiliar tasks, try new activities
**Influence Rate**: 0.5 per untrained attempt, 2.0 per untrained success

## Scaling

Scales with total combined class levels (rewards generalists).
