# Lucky Find

**Pool**: Common
**Type**: Passive
**Universal Access**: Available to all classes

## Description

Luck favors you when gathering, increasing chances of rare materials and bonus yields.

## Tiers

| Tier     | Base Effect                              | Scaling         |
| -------- | ---------------------------------------- | --------------- |
| Lesser   | +5% rare materials                       | +0.5% per level |
| Greater  | +10% rare, +5% bonus yield               | +0.4% per level |
| Enhanced | +15% rare, +10% bonus, exceptional finds | +0.3% per level |

## Acquisition

**Trigger Actions**: Find rare materials, get bonus yields, discover exceptional resources
**Influence Rate**: 2.0 per rare find, 0.5 per bonus yield

## Scaling

Scales with total combined class levels (rewards generalists).
