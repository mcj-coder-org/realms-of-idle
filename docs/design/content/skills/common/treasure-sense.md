# Treasure Sense

**Pool**: Common
**Type**: Passive
**Universal Access**: Available to all classes

## Description

Intuition for finding hidden loot, sensing valuables, and detecting secret compartments.

## Tiers

| Tier     | Base Effect                                | Scaling          |
| -------- | ------------------------------------------ | ---------------- |
| Lesser   | +10% find hidden loot                      | +1% per level    |
| Greater  | +20% find loot, sense valuables            | +0.75% per level |
| Enhanced | +30% find loot, detect secret compartments | +0.5% per level  |

## Acquisition

**Trigger Actions**: Find hidden loot, discover secret containers, loot thoroughly
**Influence Rate**: 1.0 per hidden loot found, 0.2 per thorough search

## Scaling

Scales with total combined class levels (rewards generalists).
