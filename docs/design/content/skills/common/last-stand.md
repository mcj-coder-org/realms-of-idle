# Last Stand

**Pool**: Common
**Type**: Passive
**Universal Access**: Available to all classes

## Description

Fight with desperate ferocity when near death, gaining damage, dodge, and damage reduction.

## Tiers

| Tier     | Base Effect                                      | Scaling          |
| -------- | ------------------------------------------------ | ---------------- |
| Lesser   | +10% damage below 20% HP                         | +1% per level    |
| Greater  | +20% damage, +10% dodge below 25% HP             | +0.75% per level |
| Enhanced | +30% damage, +15% dodge, -10% taken below 30% HP | +0.5% per level  |

## Acquisition

**Trigger Actions**: Fight while below 25% HP, kill enemies while wounded, survive near-death
**Influence Rate**: 0.5 per kill while wounded, 3.0 per near-death victory

## Scaling

Scales with total combined class levels (rewards generalists).
