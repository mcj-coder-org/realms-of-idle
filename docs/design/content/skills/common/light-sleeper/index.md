# Light Sleeper

**Pool**: Common
**Type**: Passive
**Universal Access**: Available to all classes

## Description

Remain partially aware while sleeping, increasing your chances of waking when threats approach.

## Tiers

| Tier     | Base Effect                                      | Scaling          |
| -------- | ------------------------------------------------ | ---------------- |
| Lesser   | +15% wake chance from threats                    | +1% per level    |
| Greater  | +30% wake chance, +5% stamina on wake            | +0.75% per level |
| Enhanced | +45% wake chance, +10% stamina, adrenaline boost | +0.5% per level  |

## Acquisition

**Trigger Actions**:

- Wake from rest due to threats
- Stand watch duty
- Interrupted sleep

**Influence Rate**: 2.0 per threat wake, 0.5 per watch shift

## Scaling

Scales with total combined class levels (rewards generalists).
