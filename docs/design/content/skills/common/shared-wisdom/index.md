# Shared Wisdom

**Pool**: Common
**Type**: Passive
**Universal Access**: Available to all classes

## Description

Share your knowledge with party members, providing them with an XP bonus.

## Tiers

| Tier     | Base Effect   | Scaling          | Range  |
| -------- | ------------- | ---------------- | ------ |
| Lesser   | Party +3% XP  | +0.3% per level  | Close  |
| Greater  | Party +6% XP  | +0.25% per level | Medium |
| Enhanced | Party +10% XP | +0.2% per level  | Far    |

## Acquisition

**Trigger Actions**: Train party members, share knowledge, teach skills
**Influence Rate**: 0.5 per training session given, 1.0 per skill taught

## Scaling

Scales with total combined class levels (rewards generalists).
