# Focused Mind

**Pool**: Common
**Type**: Passive
**Universal Access**: Available to all classes

## Description

Resist fear, confusion, and charm effects through mental fortitude, eventually breaking free actively.

## Tiers

| Tier     | Base Effect                                 | Scaling          |
| -------- | ------------------------------------------- | ---------------- |
| Lesser   | +10% resist fear/confusion/charm            | +1% per level    |
| Greater  | +20% resist mental, faster recovery         | +0.75% per level |
| Enhanced | +30% resist mental, break free from effects | +0.5% per level  |

## Acquisition

**Trigger Actions**: Resist mental effects, overcome fear, break charm/confusion
**Influence Rate**: 2.0 per mental effect resisted, 1.0 per effect overcome

## Scaling

Scales with total combined class levels (rewards generalists).
