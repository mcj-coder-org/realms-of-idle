# Second Wind

**Pool**: Common
**Type**: Cooldown
**Universal Access**: Available to all classes

## Description

Tap into emergency reserves to recover stamina when exhausted, providing a crucial recovery option in dire situations.

## Tiers

| Tier     | Base Effect                                   | Scaling          | Cooldown |
| -------- | --------------------------------------------- | ---------------- | -------- |
| Lesser   | Recover 15% max stamina                       | +1% per level    | 10 min   |
| Greater  | Recover 25% max stamina                       | +0.75% per level | 8 min    |
| Enhanced | Recover 35% max stamina, +10% regen for 2 min | +0.5% per level  | 6 min    |

## Acquisition

**Trigger Actions**:

- Collapse from zero stamina
- Rest from exhaustion
- Use stamina potions

**Influence Rate**: 1 point per collapse, 0.5 per exhausted rest

## Scaling

Scales with total combined class levels (rewards generalists).
