# Diplomat

**Pool**: Common
**Type**: Passive
**Universal Access**: Available to all classes

## Description

Mediate disputes and calm hostile situations, eventually negotiating with outright hostile factions.

## Tiers

| Tier     | Base Effect                                     | Scaling          |
| -------- | ----------------------------------------------- | ---------------- |
| Lesser   | +10% mediation, calm minor hostilities          | +1% per level    |
| Greater  | +20% mediation, calm moderate hostilities       | +0.75% per level |
| Enhanced | +30% mediation, negotiate with hostile factions | +0.5% per level  |

## Acquisition

**Trigger Actions**: Mediate disputes, calm hostile situations, negotiate peace
**Influence Rate**: 2.0 per successful mediation, 1.0 per calmed hostility

## Scaling

Scales with total combined class levels (rewards generalists).
