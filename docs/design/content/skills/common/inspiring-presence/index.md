# Inspiring Presence

**Pool**: Common
**Type**: Passive
**Universal Access**: Available to all classes

## Description

Your presence inspires nearby allies, improving their stamina regeneration and morale.

## Tiers

| Tier     | Base Effect                      | Scaling         | Range  |
| -------- | -------------------------------- | --------------- | ------ |
| Lesser   | Allies +5% stamina regen         | +0.5% per level | Close  |
| Greater  | Allies +10% stamina, +5% morale  | +0.4% per level | Medium |
| Enhanced | Allies +15% stamina, +10% morale | +0.3% per level | Far    |

## Acquisition

**Trigger Actions**: Lead party, support allies, boost party morale
**Influence Rate**: 0.2 per party action led, 1.0 per morale-boosting event

## Scaling

Scales with total combined class levels (rewards generalists).
