# Reputation Builder

**Pool**: Common
**Type**: Passive
**Universal Access**: Available to all classes

## Description

Build faction reputation more quickly while losing it more slowly, accelerating your standing with organizations.

## Tiers

| Tier     | Base Effect             | Scaling          |
| -------- | ----------------------- | ---------------- |
| Lesser   | +10% faction gains      | +1% per level    |
| Greater  | +20% gains, -10% losses | +0.75% per level |
| Enhanced | +30% gains, -20% losses | +0.5% per level  |

## Acquisition

**Trigger Actions**: Complete faction quests, improve faction standing, interact with faction members
**Influence Rate**: 0.5 per faction quest, 0.2 per positive faction interaction

## Scaling

Scales with total combined class levels (rewards generalists).
