# Environmental Reading

**Pool**: Common
**Type**: Passive
**Universal Access**: Available to all classes

## Description

Read environmental signs to predict weather and detect unnatural changes in your surroundings.

## Tiers

| Tier     | Base Effect                              | Scaling            |
| -------- | ---------------------------------------- | ------------------ |
| Lesser   | Predict weather 2-4 hrs ahead            | +0.5 hrs per level |
| Greater  | Predict weather 6-12 hrs, sense threats  | +0.5 hrs per level |
| Enhanced | Predict weather 24 hrs, detect unnatural | +0.5 hrs per level |

## Acquisition

**Trigger Actions**: Travel through varied terrain, experience weather changes, avoid weather hazards
**Influence Rate**: 0.2 per weather prediction used, 1.0 per hazard avoided via prediction

## Scaling

Scales with total combined class levels (rewards generalists).
