# Efficient Worker

**Pool**: Common
**Type**: Passive
**Universal Access**: Available to all classes

## Description

Work more efficiently, reducing stamina costs for crafting and gathering activities.

## Tiers

| Tier     | Base Effect                        | Scaling         |
| -------- | ---------------------------------- | --------------- |
| Lesser   | -10% stamina cost (craft/gather)   | +0.5% per level |
| Greater  | -20% stamina cost                  | +0.4% per level |
| Enhanced | -30% stamina cost, +10% work speed | +0.3% per level |

## Acquisition

**Trigger Actions**: Perform extended labor, craft multiple items, gather for long periods
**Influence Rate**: 0.1 per hour of labor, 0.3 per extended session

## Scaling

Scales with total combined class levels (rewards generalists).
