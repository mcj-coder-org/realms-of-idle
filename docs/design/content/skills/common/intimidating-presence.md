# Intimidating Presence

**Pool**: Common
**Type**: Passive
**Universal Access**: Available to all classes

## Description

Intimidate others through your presence, affecting enemy morale and encouraging surrender.

## Tiers

| Tier     | Base Effect                            | Scaling          |
| -------- | -------------------------------------- | ---------------- |
| Lesser   | +10% intimidation, minor morale effect | +1% per level    |
| Greater  | +20% intimidation, moderate morale     | +0.75% per level |
| Enhanced | +30% intimidation, +10% flee/surrender | +0.5% per level  |

## Acquisition

**Trigger Actions**: Successfully intimidate, cause enemies to flee, demand surrender
**Influence Rate**: 1.0 per successful intimidation, 2.0 per enemy surrender

## Scaling

Scales with total combined class levels (rewards generalists).
