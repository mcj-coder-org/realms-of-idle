# Deep Reserves

**Pool**: Common
**Type**: Passive
**Universal Access**: Available to all classes

## Description

Extended stamina reserves allow you to work longer and harder before exhaustion sets in.

## Tiers

| Tier     | Base Effect                         | Scaling          |
| -------- | ----------------------------------- | ---------------- |
| Lesser   | +5% max stamina                     | +1% per level    |
| Greater  | +10% max stamina                    | +0.75% per level |
| Enhanced | +15% max stamina, +5% stamina regen | +0.5% per level  |

## Acquisition

**Trigger Actions**:

- Perform extended labor
- Sprint long distances
- Craft for extended periods

**Influence Rate**: 0.1 per hour of heavy stamina use

## Scaling

Scales with total combined class levels (rewards generalists).
