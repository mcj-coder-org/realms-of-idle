# Keen Senses

**Pool**: Common
**Type**: Passive
**Universal Access**: Available to all classes

## Description

Heightened senses improve your ability to detect hidden enemies, traps, and objects.

## Tiers

| Tier     | Base Effect                            | Scaling          |
| -------- | -------------------------------------- | ---------------- |
| Lesser   | +10% detection                         | +1% per level    |
| Greater  | +20% detection, light concealment      | +0.75% per level |
| Enhanced | +30% detection, sense invisible nearby | +0.5% per level  |

## Acquisition

**Trigger Actions**: Detect hidden enemies, find traps, discover hidden objects
**Influence Rate**: 1.0 per hidden enemy detected, 0.5 per trap found

## Scaling

Scales with total combined class levels (rewards generalists).
