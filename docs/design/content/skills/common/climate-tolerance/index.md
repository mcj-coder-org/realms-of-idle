# Climate Tolerance

**Pool**: Common
**Type**: Passive
**Universal Access**: Available to all classes

## Description

Adapt to extreme temperatures, reducing the stamina penalties from heat and cold.

## Tiers

| Tier     | Base Effect                          | Scaling          |
| -------- | ------------------------------------ | ---------------- |
| Lesser   | -15% stamina drain from heat OR cold | +1% per level    |
| Greater  | -25% chosen, -10% opposite           | +0.75% per level |
| Enhanced | -35% all temperature extremes        | +0.5% per level  |

## Acquisition

**Trigger Actions**:

- Spend time in extreme climates
- Suffer exposure conditions

**Influence Rate**: 0.25 per hour in extreme climate, 1.0 per exposure condition survived

## Scaling

Scales with total combined class levels (rewards generalists).
