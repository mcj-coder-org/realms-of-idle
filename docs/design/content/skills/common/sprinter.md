# Sprinter

**Pool**: Common
**Type**: Passive
**Universal Access**: Available to all classes

## Description

Sprint faster with reduced stamina cost and quick acceleration.

## Tiers

| Tier     | Base Effect                               | Scaling         |
| -------- | ----------------------------------------- | --------------- |
| Lesser   | +10% sprint speed, -10% sprint cost       | +0.5% per level |
| Greater  | +20% sprint speed, -20% sprint cost       | +0.4% per level |
| Enhanced | +30% sprint speed, -30% cost, quick accel | +0.3% per level |

## Acquisition

**Trigger Actions**: Sprint, chase enemies, flee from threats
**Influence Rate**: 0.1 per sprint use, 1.0 per successful chase/flee

## Scaling

Scales with total combined class levels (rewards generalists).
