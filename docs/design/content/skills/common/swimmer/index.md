# Swimmer

**Pool**: Common
**Type**: Passive
**Universal Access**: Available to all classes

## Description

Swim faster and hold breath longer, eventually swimming in light armor.

## Tiers

| Tier     | Base Effect                                        | Scaling          |
| -------- | -------------------------------------------------- | ---------------- |
| Lesser   | +20% swim speed, +25% breath                       | +1% per level    |
| Greater  | +40% swim speed, +50% breath                       | +0.75% per level |
| Enhanced | +60% swim speed, +100% breath, swim in light armor | +0.5% per level  |

## Acquisition

**Trigger Actions**: Swim, hold breath underwater, cross water obstacles
**Influence Rate**: 0.3 per water crossing, 1.0 per underwater action

## Scaling

Scales with total combined class levels (rewards generalists).
