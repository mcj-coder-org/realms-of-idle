# Quick Learner

**Pool**: Common
**Type**: Passive
**Universal Access**: Available to all classes

## Description

Learn skills and recipes faster through accelerated XP gain.

## Tiers

| Tier     | Base Effect                         | Scaling         |
| -------- | ----------------------------------- | --------------- |
| Lesser   | +10% skill XP gain                  | +0.5% per level |
| Greater  | +20% skill XP, +10% recipe learning | +0.4% per level |
| Enhanced | +30% skill XP, +20% recipe learning | +0.3% per level |

## Acquisition

**Trigger Actions**: Learn new skills, acquire recipes, train with others
**Influence Rate**: 1.0 per new skill/recipe, 0.3 per training session

## Scaling

Scales with total combined class levels (rewards generalists).
