# Likeable

**Pool**: Common
**Type**: Passive
**Universal Access**: Available to all classes

## Description

Natural charisma makes people like you more quickly, improving starting disposition and relationship building speed.

## Tiers

| Tier     | Base Effect                                      | Scaling        |
| -------- | ------------------------------------------------ | -------------- |
| Lesser   | +5 starting disposition                          | +0.5 per level |
| Greater  | +10 disposition, faster relationships            | +0.4 per level |
| Enhanced | +15 disposition, reduced first offense hostility | +0.3 per level |

## Acquisition

**Trigger Actions**: Positive NPC interactions, make friends, improve relationships
**Influence Rate**: 0.2 per positive interaction, 1.0 per new friendship

## Scaling

Scales with total combined class levels (rewards generalists).
