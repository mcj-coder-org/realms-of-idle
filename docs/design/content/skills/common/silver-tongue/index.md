# Silver Tongue

**Pool**: Common
**Type**: Passive
**Universal Access**: Available to all classes

## Description

Deceive others through convincing lies and bluffs, eventually resisting magical truth detection.

## Tiers

| Tier     | Base Effect                               | Scaling          |
| -------- | ----------------------------------------- | ---------------- |
| Lesser   | +10% lying/bluffing success               | +1% per level    |
| Greater  | +20% success, harder to detect magically  | +0.75% per level |
| Enhanced | +30% success, +15% resist truth detection | +0.5% per level  |

## Acquisition

**Trigger Actions**: Successfully deceive NPCs, bluff in negotiations, avoid detection of lies
**Influence Rate**: 1.0 per successful deception, 0.5 per bluff

## Scaling

Scales with total combined class levels (rewards generalists).
