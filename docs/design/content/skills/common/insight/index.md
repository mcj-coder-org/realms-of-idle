# Insight

**Pool**: Common
**Type**: Passive
**Universal Access**: Available to all classes

## Description

Read people's intentions and detect lies through observation and intuition.

## Tiers

| Tier     | Base Effect                        | Scaling          |
| -------- | ---------------------------------- | ---------------- |
| Lesser   | +10% detect lies                   | +1% per level    |
| Greater  | +20% detect lies, sense intentions | +0.75% per level |
| Enhanced | +35% detect lies, sense motives    | +0.5% per level  |

## Acquisition

**Trigger Actions**: Detect lies, read NPC intentions correctly, avoid deception
**Influence Rate**: 1.5 per lie detected, 0.5 per successful read

## Scaling

Scales with total combined class levels (rewards generalists).
