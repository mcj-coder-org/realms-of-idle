# Appraiser

**Pool**: Common
**Type**: Passive
**Universal Access**: Available to all classes

## Description

Accurately assess the value of items, eventually detecting fakes and identifying enchantments.

## Tiers

| Tier     | Base Effect                     | Scaling                |
| -------- | ------------------------------- | ---------------------- |
| Lesser   | Value within 25% (common items) | -1% margin per level   |
| Greater  | Value within 10%, detect fakes  | -0.5% margin per level |
| Enhanced | Exact value, ID enchantments    | Expanded item types    |

## Acquisition

**Trigger Actions**: Trade items, identify values correctly, detect forgeries
**Influence Rate**: 0.2 per trade, 2.0 per forgery detected

## Scaling

Scales with total combined class levels (rewards generalists).
