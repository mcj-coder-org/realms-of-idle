# Pack Mule

**Pool**: Common
**Type**: Passive
**Universal Access**: Available to all classes

## Description

Carry more weight with reduced encumbrance penalties, enabling greater load capacity.

## Tiers

| Tier     | Base Effect                             | Scaling          |
| -------- | --------------------------------------- | ---------------- |
| Lesser   | +15% carry capacity                     | +1% per level    |
| Greater  | +30% carry capacity                     | +0.75% per level |
| Enhanced | +50% carry, reduced encumbrance penalty | +0.5% per level  |

## Acquisition

**Trigger Actions**: Carry heavy loads, travel while encumbered, transport goods
**Influence Rate**: 0.2 per hour encumbered, 0.5 per heavy transport

**Moderate Synergy**: Miner, Blacksmith (1.25x XP, +25%)

## Scaling

Scales with total combined class levels (rewards generalists).
