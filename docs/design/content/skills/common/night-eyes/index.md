# Night Eyes

**Pool**: Common
**Type**: Passive
**Universal Access**: Available to all classes

## Description

See better in darkness, reducing penalties for low-light conditions.

## Tiers

| Tier     | Base Effect             | Scaling          |
| -------- | ----------------------- | ---------------- |
| Lesser   | -25% darkness penalties | +1% per level    |
| Greater  | -50% darkness penalties | +0.75% per level |
| Enhanced | -75% darkness penalties | +0.5% per level  |

## Acquisition

**Trigger Actions**: Operate in darkness, travel at night, fight in low light
**Influence Rate**: 0.2 per hour in darkness, 0.5 per night combat

**Note**: Different from Night Vision (cross-class mechanic unlock which provides complete vision).

## Scaling

Scales with total combined class levels (rewards generalists).
