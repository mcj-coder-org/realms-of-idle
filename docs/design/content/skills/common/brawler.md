# Brawler

**Pool**: Common
**Type**: Passive
**Universal Access**: Available to all classes

## Description

Master unarmed combat, turning fists and feet into deadly weapons.

## Tiers

| Tier     | Base Effect                             | Scaling          |
| -------- | --------------------------------------- | ---------------- |
| Lesser   | +15% unarmed damage                     | +1% per level    |
| Greater  | +30% damage, +10% speed                 | +0.75% per level |
| Enhanced | +50% damage, +15% speed, damage armored | +0.5% per level  |

## Acquisition

**Trigger Actions**: Fight unarmed, kill enemies unarmed, grapple
**Influence Rate**: 0.5 per unarmed combat, 2.0 per unarmed kill

## Scaling

Scales with total combined class levels (rewards generalists).
