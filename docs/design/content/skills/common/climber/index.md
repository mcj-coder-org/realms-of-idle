# Climber

**Pool**: Common
**Type**: Passive
**Universal Access**: Available to all classes

## Description

Climb surfaces more effectively and safely, eventually scaling any surface with reduced fall damage.

## Tiers

| Tier     | Base Effect                                        | Scaling          |
| -------- | -------------------------------------------------- | ---------------- |
| Lesser   | Climb basic surfaces, +15% speed                   | +1% per level    |
| Greater  | Climb difficult surfaces, +30% speed               | +0.75% per level |
| Enhanced | Climb any surface, +50% speed, reduced fall damage | +0.5% per level  |

## Acquisition

**Trigger Actions**: Climb surfaces, scale heights, descend safely
**Influence Rate**: 0.5 per climb, 1.0 per difficult climb

## Scaling

Scales with total combined class levels (rewards generalists).
