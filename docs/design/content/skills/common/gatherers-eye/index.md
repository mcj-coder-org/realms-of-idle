# Gatherer's Eye

**Pool**: Common
**Type**: Passive
**Universal Access**: Available to all classes

## Description

Spot resource nodes more easily and identify quality before gathering.

## Tiers

| Tier     | Base Effect                                  | Scaling          |
| -------- | -------------------------------------------- | ---------------- |
| Lesser   | +15% node detection                          | +1% per level    |
| Greater  | +30% detection, ID quality before gather     | +0.75% per level |
| Enhanced | +50% detection, sense rare nodes at distance | +0.5% per level  |

## Acquisition

**Trigger Actions**: Find resource nodes, identify resources, locate gathering spots
**Influence Rate**: 0.3 per node found, 1.0 per rare node found

## Scaling

Scales with total combined class levels (rewards generalists).
