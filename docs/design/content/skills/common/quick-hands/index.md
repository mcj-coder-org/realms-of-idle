# Quick Hands

**Pool**: Common
**Type**: Passive
**Universal Access**: Available to all classes

## Description

Manipulate items and equipment quickly, eventually using items while moving with instant swaps.

## Tiers

| Tier     | Base Effect                                     | Scaling          |
| -------- | ----------------------------------------------- | ---------------- |
| Lesser   | +15% item use speed                             | +1% per level    |
| Greater  | +30% item speed, faster equip swap              | +0.75% per level |
| Enhanced | +50% item speed, use while moving, instant swap | +0.5% per level  |

## Acquisition

**Trigger Actions**: Use items in combat, swap equipment, quick item access
**Influence Rate**: 0.3 per combat item use, 0.5 per equipment swap in combat

## Scaling

Scales with total combined class levels (rewards generalists).
