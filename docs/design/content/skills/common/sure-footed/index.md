# Sure-Footed

**Pool**: Common
**Type**: Passive
**Universal Access**: Available to all classes

## Description

Maintain footing on difficult terrain, eventually becoming immune to minor knockdowns.

## Tiers

| Tier     | Base Effect                                   | Scaling          |
| -------- | --------------------------------------------- | ---------------- |
| Lesser   | -15% terrain penalty                          | +1% per level    |
| Greater  | -30% terrain penalty, +10% balance            | +0.75% per level |
| Enhanced | -50% terrain penalty, immune minor knockdowns | +0.5% per level  |

## Acquisition

**Trigger Actions**: Travel through difficult terrain, maintain footing, avoid falls
**Influence Rate**: 0.2 per hour in difficult terrain, 1.0 per fall avoided

## Scaling

Scales with total combined class levels (rewards generalists).
