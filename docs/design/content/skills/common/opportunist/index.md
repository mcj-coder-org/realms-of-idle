# Opportunist

**Pool**: Common
**Type**: Passive
**Universal Access**: Available to all classes

## Description

Exploit advantages and favorable circumstances, eventually creating more opportunities.

## Tiers

| Tier     | Base Effect                             | Scaling          |
| -------- | --------------------------------------- | ---------------- |
| Lesser   | +10% bonus from favorable circumstances | +1% per level    |
| Greater  | +20% bonus from favorable circumstances | +0.75% per level |
| Enhanced | +30% bonus, create more opportunities   | +0.5% per level  |

## Acquisition

**Trigger Actions**: Exploit advantages, flank enemies, capitalize on openings
**Influence Rate**: 0.3 per opportunity exploited, 1.0 per created opportunity

## Scaling

Scales with total combined class levels (rewards generalists).
