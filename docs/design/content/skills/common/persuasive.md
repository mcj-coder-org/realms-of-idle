# Persuasive

**Pool**: Common
**Type**: Passive
**Universal Access**: Available to all classes

## Description

Convince others through eloquent persuasion, eventually attempting harder challenges with reduced failure penalties.

## Tiers

| Tier     | Base Effect                              | Scaling          |
| -------- | ---------------------------------------- | ---------------- |
| Lesser   | +10% persuasion success                  | +1% per level    |
| Greater  | +20% persuasion, harder attempts         | +0.75% per level |
| Enhanced | +30% persuasion, reduced failure penalty | +0.5% per level  |

## Acquisition

**Trigger Actions**: Successfully persuade NPCs, change NPC decisions, mediate disputes
**Influence Rate**: 1.0 per successful persuasion, 0.5 per difficult persuasion

## Scaling

Scales with total combined class levels (rewards generalists).
