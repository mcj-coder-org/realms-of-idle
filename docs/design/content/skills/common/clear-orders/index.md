# Clear Orders

**Pool**: Common
**Type**: Passive
**Universal Access**: Available to all classes

## Description

Issue clear, effective commands that are followed more reliably with reduced delay.

## Tiers

| Tier     | Base Effect                                    | Scaling          |
| -------- | ---------------------------------------------- | ---------------- |
| Lesser   | +10% order compliance                          | +1% per level    |
| Greater  | +20% compliance, precise orders                | +0.75% per level |
| Enhanced | +30% compliance, complex orders, reduced delay | +0.5% per level  |

## Acquisition

**Trigger Actions**: Issue orders, have orders followed, command in combat
**Influence Rate**: 0.3 per order followed, 1.0 per complex order executed

## Scaling

Scales with total combined class levels (rewards generalists).
