# Combat Awareness

**Pool**: Common
**Type**: Passive
**Universal Access**: Available to all classes

## Description

Heightened awareness in combat improves initiative, reduces flanking vulnerability, and enhances threat detection.

## Tiers

| Tier     | Base Effect                                      | Scaling          |
| -------- | ------------------------------------------------ | ---------------- |
| Lesser   | +5% initiative, -5% flanked chance               | +1% per level    |
| Greater  | +10% initiative, -10% flanked                    | +0.75% per level |
| Enhanced | +15% initiative, -15% flanked, +5% detect hidden | +0.5% per level  |

## Acquisition

**Trigger Actions**: Win initiative, avoid being flanked, detect ambushes
**Influence Rate**: 0.2 per initiative win, 1.0 per ambush detected

## Scaling

Scales with total combined class levels (rewards generalists).
