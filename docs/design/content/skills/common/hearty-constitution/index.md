# Hearty Constitution

**Pool**: Common
**Type**: Passive
**Universal Access**: Available to all classes

## Description

A naturally robust constitution that increases your maximum health and, at higher tiers, improves health regeneration.

## Tiers

| Tier     | Base Effect                       | Scaling          |
| -------- | --------------------------------- | ---------------- |
| Lesser   | +5% max health                    | +1% per level    |
| Greater  | +10% max health                   | +0.75% per level |
| Enhanced | +15% max health, +5% health regen | +0.5% per level  |

## Acquisition

**Trigger Actions**:

- Take damage
- Survive near-death
- Complete combat encounters

**Influence Rate**: 0.1 per combat, 1.0 per near-death survival

## Scaling

Scales with total combined class levels (rewards generalists).
