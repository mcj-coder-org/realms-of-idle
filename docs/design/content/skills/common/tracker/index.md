# Tracker

**Pool**: Common
**Type**: Passive
**Universal Access**: Available to all classes

## Description

Follow trails left by creatures and people, with ability improving to track older and fainter trails.

## Tiers

| Tier     | Base Effect                                   | Scaling            |
| -------- | --------------------------------------------- | ------------------ |
| Lesser   | Follow fresh trails (<6 hrs)                  | +0.5 hrs per level |
| Greater  | Follow older trails (<24 hrs), ID type        | +0.5 hrs per level |
| Enhanced | Follow faint trails (<72 hrs), ID individuals | +0.5 hrs per level |

## Acquisition

**Trigger Actions**: Successfully track creatures, follow trails, hunt animals
**Influence Rate**: 0.5 per successful track, 1.0 per difficult track

## Scaling

Scales with total combined class levels (rewards generalists).
