# Improviser

**Pool**: Common
**Type**: Passive
**Universal Access**: Available to all classes

## Description

Craft effectively without proper tools or materials, eventually working completely tool-free.

## Tiers

| Tier     | Base Effect                                         | Scaling          |
| -------- | --------------------------------------------------- | ---------------- |
| Lesser   | -10% improvised penalty                             | +1% per level    |
| Greater  | -25% penalty, substitute some materials             | +0.75% per level |
| Enhanced | -40% penalty, substitute freely, work without tools | +0.5% per level  |

## Acquisition

**Trigger Actions**: Craft without proper tools, substitute materials, work in poor conditions
**Influence Rate**: 1.0 per improvised craft, 0.5 per material substitution

## Scaling

Scales with total combined class levels (rewards generalists).
