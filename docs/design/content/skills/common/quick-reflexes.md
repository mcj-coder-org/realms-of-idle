# Quick Reflexes

**Pool**: Common
**Type**: Passive
**Universal Access**: Available to all classes

## Description

Lightning-fast reflexes allow you to dodge incoming attacks more effectively.

## Tiers

| Tier     | Base Effect                      | Scaling          |
| -------- | -------------------------------- | ---------------- |
| Lesser   | +5% dodge chance                 | +1% per level    |
| Greater  | +10% dodge chance                | +0.75% per level |
| Enhanced | +15% dodge, +5% vs aimed attacks | +0.5% per level  |

## Acquisition

**Trigger Actions**:

- Dodge attacks (successful)
- Fight without armor
- Evade traps

**Influence Rate**: 0.1 per successful dodge, 0.5 per trap evaded

**Moderate Synergy**: Thief, Warrior (1.25x XP, +25%)

## Scaling

Scales with total combined class levels (rewards generalists).
