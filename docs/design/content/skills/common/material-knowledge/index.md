# Material Knowledge

**Pool**: Common
**Type**: Passive
**Universal Access**: Available to all classes

## Description

Understand material properties intuitively, identifying quality and optimal uses.

## Tiers

| Tier     | Base Effect                         | Scaling         |
| -------- | ----------------------------------- | --------------- |
| Lesser   | ID material quality, +5% efficiency | +0.5% per level |
| Greater  | ID properties, +10% efficiency      | +0.4% per level |
| Enhanced | ID optimal uses, +15% efficiency    | +0.3% per level |

## Acquisition

**Trigger Actions**: Work with varied materials, identify material properties, optimize material use
**Influence Rate**: 0.2 per material type used, 0.5 per new material mastered

## Scaling

Scales with total combined class levels (rewards generalists).
