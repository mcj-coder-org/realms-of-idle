# Haggler

**Pool**: Common
**Type**: Passive
**Universal Access**: Available to all classes

## Description

Negotiate better prices when trading, reducing buy prices and increasing sell prices.

## Tiers

| Tier     | Base Effect                      | Scaling         |
| -------- | -------------------------------- | --------------- |
| Lesser   | -5% buy, +5% sell                | +0.5% per level |
| Greater  | -10% buy, +10% sell              | +0.4% per level |
| Enhanced | -15% buy, +15% sell, bonus deals | +0.3% per level |

## Acquisition

**Trigger Actions**: Complete trades, negotiate prices, trade in volume
**Influence Rate**: 0.1 per trade, 0.5 per large trade, 1.0 per exceptional deal

## Scaling

Scales with total combined class levels (rewards generalists).
