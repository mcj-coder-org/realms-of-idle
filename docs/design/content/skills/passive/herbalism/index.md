# Herbalism

## Overview

| Attribute | Value                                            |
| --------- | ------------------------------------------------ |
| Type      | Passive                                          |
| Pool      | [Shamanic Pool](../pools/shamanic-pool/index.md) |
| Tags      | `Shamanic/Nature`, `Gathering/Herbalism`         |

## Description

Herbalism grants knowledge of medicinal plants and the ability to create healing poultices and remedies. For shamans, this knowledge is spiritually enhanced - spirits guide them to the right herbs and bless the preparations.

## Effect

| Property   | Value                            |
| ---------- | -------------------------------- |
| Identifies | Medicinal herbs and plants       |
| Creates    | Poultices, tinctures, remedies   |
| Bonus      | Spirit blessing enhances potency |

## Acquisition

| Method      | Requirement                  |
| ----------- | ---------------------------- |
| Class Grant | Shaman (automatic)           |
| Class Grant | Herbalist (automatic)        |
| Training    | Study under healer or shaman |
