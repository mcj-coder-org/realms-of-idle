# Passive Skills

Skills that provide constant benefits without requiring activation.

## Mechanics

| Aspect     | Description                      |
| ---------- | -------------------------------- |
| Activation | None required - always active    |
| Cost       | No ongoing resource drain        |
| Stacking   | Multiple passives stack normally |
| Conditions | Some passives have prerequisites |

## Passive Categories

| Category   | Description              |
| ---------- | ------------------------ |
| Attribute  | Stat bonuses             |
| Resistance | Damage/effect mitigation |
| Perception | Enhanced senses          |
| Racial     | Species-specific traits  |
