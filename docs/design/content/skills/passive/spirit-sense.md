# Spirit Sense

## Overview

| Attribute | Value                                      |
| --------- | ------------------------------------------ |
| Type      | Passive                                    |
| Pool      | [Shamanic Pool](../pools/shamanic-pool.md) |
| Tags      | `Shamanic`                                 |

## Description

Spirit Sense grants the character awareness of spirits in their vicinity. They can detect the presence of ancestral spirits, nature spirits, and bound spirits, though not necessarily identify or communicate with them.

## Effect

| Property    | Value                                              |
| ----------- | -------------------------------------------------- |
| Range       | Short (10-15 meters)                               |
| Detection   | Spirit presence, approximate direction             |
| Limitations | Cannot identify specific spirits without communion |

## Acquisition

| Method      | Requirement               |
| ----------- | ------------------------- |
| Class Grant | Spiritcaller (automatic)  |
| Training    | Study under tribal shaman |

## Scaling

| Tier              | Effect                                |
| ----------------- | ------------------------------------- |
| Base              | Detect spirit presence                |
| Enhanced (Shaman) | Detect spirit type and mood           |
| Mastery (Tier 3)  | Detect spirit identity and intentions |
