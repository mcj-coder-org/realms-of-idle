# Charge-Based Skills

Skills that provide a limited number of uses, refreshed on rest.

## Mechanics

| Aspect    | Description                       |
| --------- | --------------------------------- |
| Uses      | Fixed number per rest period      |
| Depletion | Cannot use when charges exhausted |
| Refresh   | Full refresh on rest              |
| Partial   | Some skills allow partial refresh |

## Charge Scaling

| Tier     | Typical Charges |
| -------- | --------------- |
| Lesser   | 1-2 charges     |
| Greater  | 2-3 charges     |
| Enhanced | 3-5 charges     |

## Planned Skills

_This category is reserved for future skill implementations._

Skills in this category will include powerful abilities with limited daily uses, such as emergency abilities, powerful buffs, or resource-intensive actions.
