# Toggle Skills

Skills that maintain an ongoing effect while active.

## Mechanics

| Aspect       | Description                          |
| ------------ | ------------------------------------ |
| Activation   | Toggle on to start effect            |
| Maintenance  | May drain stamina/mana while active  |
| Deactivation | Toggle off or automatic on depletion |
| Stacking     | Multiple toggles may conflict        |

## Toggle Categories

| Category | Drain Type | Example Use         |
| -------- | ---------- | ------------------- |
| Stance   | None       | Combat positioning  |
| Aura     | Stamina    | Party buffs         |
| Focus    | Mana       | Enhanced perception |
| Mode     | Variable   | Behavioral changes  |

## Planned Skills

_This category is reserved for future skill implementations._

Skills in this category will include combat stances, sustained auras, and mode-switching abilities.
