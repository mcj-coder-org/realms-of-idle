# Stone Skin

## Overview

Harden your skin with stone armor that passively reduces incoming damage and protects against physical attacks.

| Property | Value     |
| -------- | --------- |
| Pool     | Mana      |
| Category | Tiered    |
| Classes  | Geomancer |

## Effects

### Lesser

- Damage reduction: 8-12%
- Physical damage resistance: 15%
- Stone armor durability: 40 HP equivalent
- **Scaling**: +1 durability per class level

### Greater

- Damage reduction: 15-20%
- Physical damage resistance: 30%
- Stone armor durability: 80 HP equivalent
- **Scaling**: +1.5 durability per class level

### Enhanced

- Damage reduction: 25-35%
- Physical damage resistance: 50%
- Stone armor durability: 150 HP equivalent
- **Scaling**: +2 durability per class level

## Acquisition

| Method      | Requirement              |
| ----------- | ------------------------ |
| Class Grant | Geomancer (Lesser tier)  |
| Level Up    | Geomancer 10+ (Greater)  |
| Level Up    | Geomancer 25+ (Enhanced) |
