# Fire Spirit Bond

## Overview

Forge a deep bond with the spirits of sacred fire and warmth, channeling their power through your shamanic connection.

| Property | Value        |
| -------- | ------------ |
| Pool     | Shamanic     |
| Category | Tiered       |
| Classes  | Flame Keeper |

## Effects

### Lesser

- Spirit Power: +15% to fire-based shamanic effects
- Connection Strength: +20% spirit influence
- Duration: +30% for fire blessings
- Scaling: +1% per Flame Keeper level

### Greater

- Spirit Power: +30% to fire-based shamanic effects
- Connection Strength: +40% spirit influence
- Duration: +60% for fire blessings
- Scaling: +1.5% per Flame Keeper level

### Enhanced

- Spirit Power: +50% to fire-based shamanic effects
- Connection Strength: +70% spirit influence
- Duration: +120% for fire blessings
- Scaling: +2% per Flame Keeper level

## Acquisition

| Method      | Requirement                 |
| ----------- | --------------------------- |
| Class Grant | Flame Keeper (Lesser)       |
| Level Up    | Flame Keeper 10+ (Greater)  |
| Level Up    | Flame Keeper 25+ (Enhanced) |
