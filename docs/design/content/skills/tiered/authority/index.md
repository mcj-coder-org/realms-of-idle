# Authority

**Pool**: Leadership
**Type**: Tiered
**Classes**: Leader, Chieftain, Guildmaster

## Description

Your commands carry natural authority, compelling others to obey with increasing effectiveness as your mastery grows.

## Tiers

| Tier     | Effect                                                                           | Scaling                |
| -------- | -------------------------------------------------------------------------------- | ---------------------- |
| Lesser   | +20% order compliance, orders from distance                                      | +1% per class level    |
| Greater  | +35% compliance, complex orders, non-party listens                               | +0.75% per class level |
| Enhanced | +50% compliance, enemies may follow orders (surrender), instant obedience option | +0.5% per class level  |

## Acquisition

**Trigger Actions**:

- Issue commands
- Maintain authority

**Classes with Strong Synergy**: Leader, Chieftain, Guildmaster (2x XP, direct specialization)

## XP-Based Progression

Skills improve continuously with accumulated XP. More XP = better effectiveness, lower costs, higher success rates.
