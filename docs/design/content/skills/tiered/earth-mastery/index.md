# Earth Mastery

## Overview

Master the fundamentals of earth magic to increase spell power, reduce casting costs, and strengthen stone barriers and defensive magic.

| Property | Value     |
| -------- | --------- |
| Pool     | Mana      |
| Category | Tiered    |
| Classes  | Geomancer |

## Effects

### Lesser

- Earth spell power: +15%
- Mana cost: -10%
- Stone barrier strength: +25%
- **Scaling**: +0.5% spell power per class level

### Greater

- Earth spell power: +30%
- Mana cost: -20%
- Stone barrier strength: +50%
- **Scaling**: +0.75% spell power per class level

### Enhanced

- Earth spell power: +50%
- Mana cost: -30%
- Stone barrier strength: +100%
- **Scaling**: +1% spell power per class level

## Acquisition

| Method      | Requirement              |
| ----------- | ------------------------ |
| Class Grant | Geomancer (Lesser tier)  |
| Level Up    | Geomancer 10+ (Greater)  |
| Level Up    | Geomancer 25+ (Enhanced) |
