# Resource Sense

## Overview

**Type**: Tiered Skill
**Pool**: Gathering
**Description**: Detect gathering nodes at distance with increasing range and detail through tiers.

## Synergy Classes

Classes with strong synergies: All Gathering pool (Forager, Miner, Hunter, Fisher, Lumberjack, Trapper, Herbalist)

- Each class senses their specialty resources

**Synergy Benefits**: 2x XP acquisition, enhanced effectiveness, reduced costs (scales with class level)

## Tier Progression

| Tier     | Range    | Detail                                | Scaling                   |
| -------- | -------- | ------------------------------------- | ------------------------- |
| Lesser   | Medium   | Sense presence of relevant nodes      | +5% range per class level |
| Greater  | Far      | Sense presence and quality            | +4% range per class level |
| Enhanced | Very Far | Sense presence, quality, and quantity | +3% range per class level |

## XP Sources

- Detect nodes: 3-8 XP per node found
- Find high-quality nodes: 10-20 XP
- Guided training: 10-100 XP/session
- Unguided training: 5-50 XP/session

## Level-Up Availability

- Strong synergy: ~500 XP (~50 actions)
- Moderate synergy: ~1000 XP
- No synergy: ~1500 XP

## Synergy Examples

### Strong (2x XP)

**Miner Level 15** - Resource detection specialist

- Extended range
- Quality assessment
- 2x XP from sensing nodes

### Moderate (1.5x XP)

**Forager Level 10** - Gathering expert

- Enhanced range

### None (1x XP)

**Warrior** - No connection

- No bonuses
