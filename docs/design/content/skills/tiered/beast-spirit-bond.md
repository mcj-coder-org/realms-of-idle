# Beast Spirit Bond

## Overview

Forge a deep bond with the spirits of wild animals, channeling their primal strength and instinct.

| Property | Value         |
| -------- | ------------- |
| Pool     | Shamanic      |
| Category | Tiered        |
| Classes  | Beast Speaker |

## Effects

### Lesser

- Spirit Power: +15% to animal-based shamanic effects
- Primal sense: +20% detect hidden creatures
- Connection Strength: +20% spirit influence
- Scaling: +1% per Beast Speaker level

### Greater

- Spirit Power: +30% to animal-based shamanic effects
- Primal sense: +40% detect hidden creatures
- Connection Strength: +40% spirit influence
- Animal affinity: Can communicate with beasts
- Scaling: +1.5% per Beast Speaker level

### Enhanced

- Spirit Power: +50% to animal-based shamanic effects
- Primal sense: +60% detect hidden creatures
- Connection Strength: +70% spirit influence
- Animal affinity: Can communicate and command beasts
- Transformation: Can temporarily take on animal forms
- Scaling: +2% per Beast Speaker level

## Acquisition

| Method      | Requirement                  |
| ----------- | ---------------------------- |
| Class Grant | Beast Speaker (Lesser)       |
| Level Up    | Beast Speaker 10+ (Greater)  |
| Level Up    | Beast Speaker 25+ (Enhanced) |
