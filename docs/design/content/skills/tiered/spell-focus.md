# Spell Focus

## Overview

Enhanced spell effectiveness.

| Property | Value          |
| -------- | -------------- |
| Pool     | Magic          |
| Category | Tiered         |
| Classes  | All Magic pool |

## Effects

### Lesser

- +10% spell damage/healing
- -10% mana cost
- **Scaling**: +0.5% per class level

### Greater

- +18% effect
- -18% mana cost
- **Scaling**: +0.4% per class level

### Enhanced

- +25% effect
- -25% mana cost
- Faster casting
- **Scaling**: +0.3% per class level

## Acquisition

**Trigger Actions**: Cast spells, practice magic
**Classes**: All Magic pool
