# Resource Efficiency

**Type:** Tiered
**Pool:** Crafting (General)
**Strong Synergy Classes:** All Crafting specialties

## Effect

| Tier     | Effect                                      | Scaling               |
| -------- | ------------------------------------------- | --------------------- |
| Lesser   | -10% materials required                     | +0.5% per class level |
| Greater  | -18% materials, salvage from failures       | +0.4% per class level |
| Enhanced | -25% materials, no material loss on failure | +0.3% per class level |

## Trigger Actions

- Craft efficiently
- Minimize waste

## Notes

- Greater tier allows salvaging materials from failed crafts
- Enhanced tier eliminates material loss on failures entirely
- Significant economic benefit for high-volume crafters
