# Market Sense

## Overview

Know prices and opportunities.

| Property | Value                       |
| -------- | --------------------------- |
| Pool     | Trade                       |
| Category | Tiered                      |
| Classes  | Trader, Merchant, Caravaner |

## Effects

### Lesser

- Know local prices
- Sense good deals (+15%)
- **Scaling**: +1% per class level

### Greater

- Know regional prices
- Predict price changes
- **Scaling**: +0.75% per class level

### Enhanced

- Know inter-regional prices
- Manipulate local markets
- **Scaling**: +0.5% per class level

## Acquisition

**Trigger Actions**: Trade frequently, track prices, exploit opportunities
**Classes**: Trader, Merchant, Caravaner
