# Misdirection

## Overview

**Type**: Tiered Skill
**Pool**: Stealth
**Description**: Create diversions and false trails to mislead pursuers and investigators.

## Synergy Classes

Classes with strong synergies: Thief, Assassin, Smuggler

**Synergy Benefits**: 2x XP acquisition, enhanced effectiveness, reduced costs (scales with class level)

## Tier Progression

| Tier     | Effect                                                         | Scaling                |
| -------- | -------------------------------------------------------------- | ---------------------- |
| Lesser   | +20% chance to mislead pursuers, create false trails           | +1% per class level    |
| Greater  | +35% mislead, throw voice/sound, plant false evidence          | +0.75% per class level |
| Enhanced | +50% mislead, frame others convincingly, escape almost certain | +0.5% per class level  |

## XP Sources

- Escape pursuit: 15-25 XP
- Plant false evidence: 10-20 XP
- Mislead investigators: 10-20 XP
- Guided training: 10-100 XP/session
- Unguided training: 5-50 XP/session

## Level-Up Availability

- Strong synergy: ~500 XP (~50 actions)
- Moderate synergy: ~1000 XP
- No synergy: ~1500 XP

## Synergy Examples

### Strong (2x XP)

**Assassin Level 15** - Escape specialist

- +60% mislead
- Frame others easily
- 2x XP from misdirection

### Moderate (1.5x XP)

**Smuggler Level 10** - Evasion expert

- +40% mislead

### None (1x XP)

**Blacksmith** - No connection

- No bonuses
