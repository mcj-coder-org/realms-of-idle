# Evidence Elimination

**Pool**: Dark
**Type**: Tiered
**Classes**: All Dark pool

## Description

Systematically eliminate evidence of your crimes, reducing the chances of being caught and eventually enabling evidence manipulation to frame others.

## Tiers

| Tier     | Effect                                                | Scaling                |
| -------- | ----------------------------------------------------- | ---------------------- |
| Lesser   | -25% chance of evidence at crime scene                | +1% per class level    |
| Greater  | -50% evidence chance, can remove existing evidence    | +0.75% per class level |
| Enhanced | -75% evidence, remove evidence remotely, frame others | +0.5% per class level  |

## Acquisition

**Trigger Actions**:

- Commit crimes cleanly
- Remove evidence

**Classes with Strong Synergy**: All Dark pool (2x XP, direct specialization)

## XP-Based Progression

Skills improve continuously with accumulated XP. More XP = better effectiveness, lower costs, higher success rates.
