# Vision Quest

## Overview

Undertake a vision quest to gain prophetic insight and spiritual understanding of situations.

| Property | Value         |
| -------- | ------------- |
| Pool     | Shamanic      |
| Category | Tiered        |
| Classes  | Spirit Walker |

## Effects

### Lesser

- Insight: +25% understanding of situations and enemies
- Duration: 30 seconds per cast
- Revelation: Reveal one hidden fact or truth
- Scaling: +2% per Spirit Walker level

### Greater

- Insight: +50% understanding of situations and enemies
- Duration: 45 seconds per cast
- Revelation: Reveal up to 3 hidden facts or truths
- Foresight: Predict enemy actions (+20% accuracy/dodge)
- Scaling: +3% per Spirit Walker level

### Enhanced

- Insight: +75% understanding of situations and enemies
- Duration: 60 seconds per cast
- Revelation: Reveal all relevant hidden facts and truths
- Foresight: Predict enemy actions (+40% accuracy/dodge)
- Prophecy: Gain temporary ability to see near future
- Scaling: +4% per Spirit Walker level

## Acquisition

| Method      | Requirement                  |
| ----------- | ---------------------------- |
| Class Grant | Spirit Walker (Lesser)       |
| Level Up    | Spirit Walker 10+ (Greater)  |
| Level Up    | Spirit Walker 25+ (Enhanced) |
