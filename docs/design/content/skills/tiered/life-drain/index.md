# Life Drain

## Overview

Siphon life force from enemies to damage them while healing yourself, creating vampiric effects that weaken foes and strengthen the caster.

| Property | Value       |
| -------- | ----------- |
| Pool     | Mana        |
| Category | Tiered      |
| Classes  | Necromancer |

## Effects

### Lesser

- Drain damage: 5-8 HP per cast
- Healing from drain: 50% of damage dealt
- Effect range: 20m
- **Scaling**: +0.5 damage per class level

### Greater

- Drain damage: 12-18 HP per cast
- Healing from drain: 75% of damage dealt
- Effect range: 30m
- Weak enemies drained for extra healing (+25%)
- **Scaling**: +1 damage per class level

### Enhanced

- Drain damage: 20-30 HP per cast
- Healing from drain: 100% of damage dealt
- Effect range: 40m
- Can chain drain to nearby enemies
- Weak enemies drained for extra healing (+50%)
- **Scaling**: +1.5 damage per class level

## Acquisition

| Method      | Requirement                |
| ----------- | -------------------------- |
| Class Grant | Necromancer (Lesser tier)  |
| Level Up    | Necromancer 10+ (Greater)  |
| Level Up    | Necromancer 25+ (Enhanced) |
