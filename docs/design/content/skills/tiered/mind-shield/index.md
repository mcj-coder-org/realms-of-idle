# Mind Shield

## Overview

Protect yourself and allies using illusions and mental barriers that defend against perception-based attacks and mental manipulation.

| Property | Value       |
| -------- | ----------- |
| Pool     | Mana        |
| Category | Tiered      |
| Classes  | Illusionist |

## Effects

### Lesser

- Mental damage reduction: 10-15%
- Perception attack resistance: 20%
- Shield duration: Until depleted (40 HP equivalent)
- Creates illusion of fake target
- **Scaling**: +1 HP equivalent per class level

### Greater

- Mental damage reduction: 20-30%
- Perception attack resistance: 40%
- Shield duration: Until depleted (80 HP equivalent)
- Creates multiple illusion decoys
- Grants 25% chance to evade perception attacks
- **Scaling**: +1.5 HP equivalent per class level

### Enhanced

- Mental damage reduction: 35-50%
- Perception attack resistance: 70%
- Shield duration: Until depleted (150 HP equivalent)
- Creates realistic illusion decoys that mimic caster
- Grants 50% chance to evade perception attacks
- Can extend mind shield to nearby allies
- **Scaling**: +2 HP equivalent per class level

## Acquisition

| Method      | Requirement                |
| ----------- | -------------------------- |
| Class Grant | Illusionist (Lesser tier)  |
| Level Up    | Illusionist 10+ (Greater)  |
| Level Up    | Illusionist 25+ (Enhanced) |
