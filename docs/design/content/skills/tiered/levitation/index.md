# Levitation

## Overview

Rise above the battlefield using air magic to gain aerial advantage, enhanced mobility, and access to elevated areas.

| Property | Value      |
| -------- | ---------- |
| Pool     | Mana       |
| Category | Tiered     |
| Classes  | Aeromancer |

## Effects

### Lesser

- Flight height: 5m
- Movement speed while flying: +20%
- Flight duration per mana point: 15 seconds
- **Scaling**: +0.25m height per class level

### Greater

- Flight height: 10m
- Movement speed while flying: +40%
- Flight duration per mana point: 30 seconds
- Can hover indefinitely with sustained mana drain
- **Scaling**: +0.4m height per class level

### Enhanced

- Flight height: 15m
- Movement speed while flying: +60%
- Flight duration per mana point: 60 seconds
- Can use ranged abilities while airborne
- **Scaling**: +0.5m height per class level

## Acquisition

| Method      | Requirement               |
| ----------- | ------------------------- |
| Class Grant | Aeromancer (Lesser tier)  |
| Level Up    | Aeromancer 10+ (Greater)  |
| Level Up    | Aeromancer 25+ (Enhanced) |
