# Green Thumb

## Overview

**Type**: Tiered Skill
**Pool**: Agriculture
**Description**: Improved crop growth speed and yield with increasing weather resistance.

## Synergy Classes

Classes with strong synergies: Farmer

**Synergy Benefits**: 2x XP acquisition, enhanced effectiveness, reduced costs (scales with class level)

## Tier Progression

| Tier     | Effect                                               | Scaling                |
| -------- | ---------------------------------------------------- | ---------------------- |
| Lesser   | +15% crop growth speed, +10% yield                   | +1% per class level    |
| Greater  | +25% growth, +20% yield, resist weather damage       | +0.75% per class level |
| Enhanced | +40% growth, +35% yield, crops survive drought/frost | +0.5% per class level  |

## XP Sources

- Tend crops: 3-8 XP per session
- Achieve good harvests: 10-20 XP
- Guided training: 10-100 XP/session
- Unguided training: 5-50 XP/session

## Level-Up Availability

- Strong synergy: ~500 XP (~50 actions)
- Moderate synergy: ~1000 XP
- No synergy: ~1500 XP

## Synergy Examples

### Strong (2x XP)

**Farmer Level 15** - Crop specialist

- +50% growth
- +45% yield
- 2x XP from crop tending

### Moderate (1.5x XP)

**Herbalist Level 10** - Plant expert

- +30% growth

### None (1x XP)

**Warrior** - No connection

- No bonuses
