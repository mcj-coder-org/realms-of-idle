# Sacred Flame

## Overview

Channel purifying sacred fire through shamanic power, delivering both damage and protective benefits.

| Property | Value        |
| -------- | ------------ |
| Pool     | Shamanic     |
| Category | Tiered       |
| Classes  | Flame Keeper |

## Effects

### Lesser

- Damage per hit: 8-12 HP
- Purification chance: 30%
- Blessing radius: 6m
- Scaling: +0.8 damage per Flame Keeper level

### Greater

- Damage per hit: 16-24 HP
- Purification chance: 50%
- Blessing radius: 10m
- Scaling: +1.5 damage per Flame Keeper level

### Enhanced

- Damage per hit: 32-48 HP
- Purification chance: 75%
- Blessing radius: 15m
- Scaling: +2.5 damage per Flame Keeper level

## Acquisition

| Method      | Requirement                 |
| ----------- | --------------------------- |
| Class Grant | Flame Keeper (Lesser)       |
| Level Up    | Flame Keeper 10+ (Greater)  |
| Level Up    | Flame Keeper 25+ (Enhanced) |
