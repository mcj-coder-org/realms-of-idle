# Shadow Step

## Overview

**Type**: Tiered Skill
**Pool**: Stealth
**Description**: Enhanced stealth movement with increasing effectiveness through tiers.

## Synergy Classes

Classes with strong synergies: Thief, Assassin, Scout

**Synergy Benefits**: 2x XP acquisition, enhanced effectiveness, reduced costs (scales with class level)

## Tier Progression

| Tier     | Effect                                                        | Scaling                |
| -------- | ------------------------------------------------------------- | ---------------------- |
| Lesser   | +15% stealth, no penalty for slow movement                    | +1% per class level    |
| Greater  | +25% stealth, half penalty for normal movement                | +0.75% per class level |
| Enhanced | +35% stealth, no movement penalty, brief invisibility on kill | +0.5% per class level  |

## XP Sources

- Move undetected: 5-10 XP per success
- Sneak past enemies: 10-20 XP
- Stealth kills: 15-25 XP
- Guided training: 10-100 XP/session
- Unguided training: 5-50 XP/session

## Level-Up Availability

- Strong synergy: ~500 XP (~50 actions)
- Moderate synergy: ~1000 XP
- No synergy: ~1500 XP

## Synergy Examples

### Strong (2x XP)

**Thief Level 15** - Stealth specialist

- +35% stealth
- -25% noise
- 2x XP from stealth movement

### Moderate (1.5x XP)

**Scout Level 10** - Reconnaissance expert

- +25% stealth

### None (1x XP)

**Warrior** - No connection

- No bonuses
