# Wind Mastery

## Overview

Master the fundamentals of air magic to increase spell power, reduce casting costs, and enhance movement and aerial capabilities.

| Property | Value      |
| -------- | ---------- |
| Pool     | Mana       |
| Category | Tiered     |
| Classes  | Aeromancer |

## Effects

### Lesser

- Air spell power: +15%
- Mana cost: -10%
- Movement speed: +15%
- **Scaling**: +0.5% spell power per class level

### Greater

- Air spell power: +30%
- Mana cost: -20%
- Movement speed: +30%
- **Scaling**: +0.75% spell power per class level

### Enhanced

- Air spell power: +50%
- Mana cost: -30%
- Movement speed: +50%
- **Scaling**: +1% spell power per class level

## Acquisition

| Method      | Requirement               |
| ----------- | ------------------------- |
| Class Grant | Aeromancer (Lesser tier)  |
| Level Up    | Aeromancer 10+ (Greater)  |
| Level Up    | Aeromancer 25+ (Enhanced) |
