# Rapid Learning

**Pool**: Scholar
**Type**: Tiered
**Classes**: Scholar, Trainer

## Description

Learn faster through study and training, accelerating skill acquisition and recipe learning.

## Tiers

| Tier     | Effect                                               | Scaling                |
| -------- | ---------------------------------------------------- | ---------------------- |
| Lesser   | +20% XP from study/training                          | +1% per class level    |
| Greater  | +35% study XP, learn recipes/skills faster           | +0.75% per class level |
| Enhanced | +50% study XP, can learn skills above "normal" level | +0.5% per class level  |

## Acquisition

**Trigger Actions**:

- Study
- Train
- Learn new things

**Classes with Strong Synergy**: Scholar, Trainer (2x XP, direct specialization)

## XP-Based Progression

Skills improve continuously with accumulated XP. More XP = better effectiveness, lower costs, higher success rates.
