# Undead Mastery

## Overview

Master the control of undead minions to increase summoning power, improve minion durability, and enhance coordination of undead forces.

| Property | Value       |
| -------- | ----------- |
| Pool     | Mana        |
| Category | Tiered      |
| Classes  | Necromancer |

## Effects

### Lesser

- Undead minion power: +15%
- Minion durability: +25%
- Mana cost for summoning: -10%
- **Scaling**: +0.5% power per class level

### Greater

- Undead minion power: +30%
- Minion durability: +50%
- Mana cost for summoning: -20%
- Can maintain more simultaneous minions (+2 limit)
- **Scaling**: +0.75% power per class level

### Enhanced

- Undead minion power: +50%
- Minion durability: +100%
- Mana cost for summoning: -30%
- Can maintain significantly more minions (+4 limit)
- Minions gain passive damage reflection
- **Scaling**: +1% power per class level

## Acquisition

| Method      | Requirement                |
| ----------- | -------------------------- |
| Class Grant | Necromancer (Lesser tier)  |
| Level Up    | Necromancer 10+ (Greater)  |
| Level Up    | Necromancer 25+ (Enhanced) |
