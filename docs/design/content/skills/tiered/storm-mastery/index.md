# Storm Mastery

## Overview

Mastery over lightning forces, amplifying electrical spell power while improving mana efficiency and chain reaction effectiveness. Storm Mastery represents deep understanding of electrical principles - how to intensify voltage, direct chains through multiple targets, and maximize damage output.

| Property | Value                       |
| -------- | --------------------------- |
| Pool     | Magic (Lightning)           |
| Category | Tiered                      |
| Classes  | Electromancer               |
| Tags     | `Magic/Elemental/Lightning` |

## Effects

### Lesser

- +15% lightning spell damage
- -10% mana cost on lightning spells
- +1 additional chain bounce
- **Scaling**: +0.5% damage per class level

### Greater

- +30% lightning spell damage
- -20% mana cost on lightning spells
- +2 additional chain bounces
- Chain effectiveness increased 15%
- **Scaling**: +0.7% damage per class level

### Enhanced

- +50% lightning spell damage
- -30% mana cost on lightning spells
- +3 additional chain bounces
- Chain effectiveness increased 30%
- Faster casting on lightning spells (+20% cast speed)
- **Scaling**: +1.0% damage per class level

## Acquisition

| Method      | Requirement                  |
| ----------- | ---------------------------- |
| Class Grant | Electromancer (Lesser tier)  |
| Level Up    | Electromancer 10+ (Greater)  |
| Level Up    | Electromancer 25+ (Enhanced) |

## Tracked Actions

Actions that contribute to Storm Mastery:

- Cast lightning spells in combat
- Execute successful chain lightning bounces
- Defeat enemies with lightning magic
- Maintain active lightning effects
