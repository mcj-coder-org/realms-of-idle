# Planar Binding

## Overview

Strengthen the binding of planar entities to improve their capabilities, extend their summoning duration, and enhance their loyalty and power.

| Property | Value    |
| -------- | -------- |
| Pool     | Mana     |
| Category | Tiered   |
| Classes  | Summoner |

## Effects

### Lesser

- Binding strength: +15%
- Minion duration extension: +25%
- Minion loyalty: +20%
- Minion special ability power: +10%
- **Scaling**: +0.5% binding per class level

### Greater

- Binding strength: +30%
- Minion duration extension: +50%
- Minion loyalty: +40%
- Minion special ability power: +25%
- Bound minions gain passive enhancement
- **Scaling**: +0.75% binding per class level

### Enhanced

- Binding strength: +50%
- Minion duration extension: +100%
- Minion loyalty: +70%
- Minion special ability power: +50%
- Bound minions gain enhanced passive abilities
- Can bind multiple planar forms to single minion
- **Scaling**: +1% binding per class level

## Acquisition

| Method      | Requirement             |
| ----------- | ----------------------- |
| Class Grant | Summoner (Lesser tier)  |
| Level Up    | Summoner 10+ (Greater)  |
| Level Up    | Summoner 25+ (Enhanced) |
