# Weapon Mastery

## Overview

Specialized proficiency with weapon category providing enhanced damage, accuracy, and access to weapon-specific techniques.

## Synergy Classes

**Strong Synergy (2x XP)**: Warrior, Adventurer, Knight, Guard, Mercenary

Classes with strong synergies get faster learning, better effectiveness, and lower costs.

## Tiers

| Tier     | Effect                                               | Scaling                |
| -------- | ---------------------------------------------------- | ---------------------- |
| Lesser   | +10% damage, +5% accuracy with category              | +1% per class level    |
| Greater  | +18% damage, +10% accuracy, unlock weapon techniques | +0.75% per class level |
| Enhanced | +25% damage, +15% accuracy, advanced techniques      | +0.5% per class level  |

**Categories:** Swords, Axes, Maces, Spears, Bows, Crossbows, Daggers, Unarmed

## XP Progression

**XP Sources**:

- Combat kill with weapon type: 5-15 XP per kill
- Training with weapon: 10-50 XP per session
- Guided training: 10-100 XP per session
- Unguided training: 5-50 XP per session

**Level-Up Availability Thresholds**:

- Strong synergy classes: ~500 XP (~50 actions)
- Moderate synergy classes: ~1000 XP (~75 actions)
- No synergy classes: ~1500 XP (~100-150 actions)

## Synergy Bonuses

**Strong Synergy Example** (Warrior Level 15):

- Faster learning: 2x XP from weapon combat
- Better effectiveness: +25% damage bonus
- Reduced cost: -25% stamina per attack

**Scaling with Class Level**:

| Class Level | XP Multiplier | Effectiveness | Cost Reduction |
| ----------- | ------------- | ------------- | -------------- |
| Level 5     | 1.5x (+50%)   | +15%          | -15%           |
| Level 15    | 2.0x (+100%)  | +25%          | -25%           |
| Level 30+   | 2.5x (+150%)  | +35%          | -35%           |

## Mechanics

**Trigger Actions:** Combat kills with weapon type, training with weapon

**Weapon Categories:** Each weapon category must be learned separately. Mastery in Swords does not provide benefits to Axes.

**Technique Unlocks:** Greater and Enhanced tiers unlock special weapon techniques specific to the weapon category, such as sword parries, axe cleaves, or bow trick shots.
