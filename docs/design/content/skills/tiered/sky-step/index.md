# Sky Step

## Overview

Move with the grace and speed of wind spirits, granting supernatural mobility and freedom of movement.

| Property | Value       |
| -------- | ----------- |
| Pool     | Shamanic    |
| Category | Tiered      |
| Classes  | Wind Walker |

## Effects

### Lesser

- Movement speed: +25%
- Jump height: +50%
- Fall damage: -30%
- Duration: 20 seconds per cast
- Scaling: +1% per Wind Walker level

### Greater

- Movement speed: +50%
- Jump height: +100%
- Fall damage: -60%
- Air dodge: +20% evasion
- Duration: 30 seconds per cast
- Scaling: +1.5% per Wind Walker level

### Enhanced

- Movement speed: +100%
- Jump height: +150% (can short-distance float)
- Fall damage: -100% (immunity)
- Air dodge: +40% evasion
- Aerial: Can move through difficult terrain unhindered
- Duration: 40 seconds per cast
- Scaling: +2% per Wind Walker level

## Acquisition

| Method      | Requirement                |
| ----------- | -------------------------- |
| Class Grant | Wind Walker (Lesser)       |
| Level Up    | Wind Walker 10+ (Greater)  |
| Level Up    | Wind Walker 25+ (Enhanced) |
