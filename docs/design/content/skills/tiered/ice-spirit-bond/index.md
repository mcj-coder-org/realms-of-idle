# Ice Spirit Bond

## Overview

Forge a deep bond with the spirits of ice and winter, channeling their power through your shamanic connection.

| Property | Value        |
| -------- | ------------ |
| Pool     | Shamanic     |
| Category | Tiered       |
| Classes  | Frost Shaman |

## Effects

### Lesser

- Spirit Power: +15% to ice-based shamanic effects
- Connection Strength: +20% spirit influence
- Duration: +30% for cold blessings
- Scaling: +1% per Frost Shaman level

### Greater

- Spirit Power: +30% to ice-based shamanic effects
- Connection Strength: +40% spirit influence
- Duration: +60% for cold blessings
- Scaling: +1.5% per Frost Shaman level

### Enhanced

- Spirit Power: +50% to ice-based shamanic effects
- Connection Strength: +70% spirit influence
- Duration: +120% for cold blessings
- Scaling: +2% per Frost Shaman level

## Acquisition

| Method      | Requirement                 |
| ----------- | --------------------------- |
| Class Grant | Frost Shaman (Lesser)       |
| Level Up    | Frost Shaman 10+ (Greater)  |
| Level Up    | Frost Shaman 25+ (Enhanced) |
