# Dream Bond

## Overview

Forge a deep bond with the spirits of dreams and the astral realm, channeling their ethereal power.

| Property | Value         |
| -------- | ------------- |
| Pool     | Shamanic      |
| Category | Tiered        |
| Classes  | Spirit Walker |

## Effects

### Lesser

- Spirit Power: +15% to dream-based shamanic effects
- Astral sense: +20% perception in dream realm
- Connection Strength: +20% spirit influence
- Scaling: +1% per Spirit Walker level

### Greater

- Spirit Power: +30% to dream-based shamanic effects
- Astral sense: +40% perception in dream realm
- Connection Strength: +40% spirit influence
- Lucid awareness: Can see truth in deceptions (+20% insight)
- Scaling: +1.5% per Spirit Walker level

### Enhanced

- Spirit Power: +50% to dream-based shamanic effects
- Astral sense: +60% perception in dream realm
- Connection Strength: +70% spirit influence
- Lucid awareness: Can see truth in deceptions (+40% insight)
- Prophecy: Gain glimpses of near-future events
- Scaling: +2% per Spirit Walker level

## Acquisition

| Method      | Requirement                  |
| ----------- | ---------------------------- |
| Class Grant | Spirit Walker (Lesser)       |
| Level Up    | Spirit Walker 10+ (Greater)  |
| Level Up    | Spirit Walker 25+ (Enhanced) |
