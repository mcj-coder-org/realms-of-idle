# Counter Magic

## Overview

Disrupt enemy spells.

| Property | Value           |
| -------- | --------------- |
| Pool     | Magic           |
| Category | Tiered          |
| Classes  | Mage, Enchanter |

## Effects

### Lesser

- Can attempt to counter spells (30% success)
- **Scaling**: +1% per class level

### Greater

- 45% counter success
- Sense incoming spells
- **Scaling**: +0.75% per class level

### Enhanced

- 60% counter success
- Reflect chance
- **Scaling**: +0.5% per class level

## Acquisition

**Trigger Actions**: Attempt counters, fight enemy casters
**Classes**: Mage, Enchanter
