# Masterwork Chance

**Type:** Tiered
**Pool:** Crafting (General)
**Strong Synergy Classes:** All Crafting specialties

## Effect

| Tier     | Effect                                             | Scaling                |
| -------- | -------------------------------------------------- | ---------------------- |
| Lesser   | +3% masterwork chance on crafts                    | +0.2% per class level  |
| Greater  | +6% masterwork chance, +10% superior chance        | +0.15% per class level |
| Enhanced | +10% masterwork, +20% superior, sense when to push | +0.1% per class level  |

## Trigger Actions

- Achieve masterwork results
- Push for quality

## Notes

- Enhanced tier provides intuition about when extra effort will succeed
- Significantly increases chances of exceptional results
- Combines with quality-focused crafting strategies
