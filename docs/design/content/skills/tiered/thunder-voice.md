# Thunder Voice

## Overview

Shout with the power of thunder spirits, delivering sonic and electrical damage to enemies.

| Property | Value        |
| -------- | ------------ |
| Pool     | Shamanic     |
| Category | Tiered       |
| Classes  | Storm Caller |

## Effects

### Lesser

- Damage per shout: 10-15 HP
- Chain range: 8m
- Chain targets: 2 enemies
- Stun chance: 15%
- Scaling: +1 damage per Storm Caller level

### Greater

- Damage per shout: 20-30 HP
- Chain range: 12m
- Chain targets: 4 enemies
- Stun chance: 30%
- Daze effect: -20% accuracy
- Scaling: +1.8 damage per Storm Caller level

### Enhanced

- Damage per shout: 40-60 HP
- Chain range: 15m
- Chain targets: 6 enemies
- Stun chance: 50%
- Daze effect: -40% accuracy
- Vulnerability: Targets take +20% lightning damage
- Scaling: +3 damage per Storm Caller level

## Acquisition

| Method      | Requirement                 |
| ----------- | --------------------------- |
| Class Grant | Storm Caller (Lesser)       |
| Level Up    | Storm Caller 10+ (Greater)  |
| Level Up    | Storm Caller 25+ (Enhanced) |
