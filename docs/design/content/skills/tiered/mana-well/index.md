# Mana Well

## Overview

Expanded magical reserves.

| Property | Value          |
| -------- | -------------- |
| Pool     | Magic          |
| Category | Tiered         |
| Classes  | All Magic pool |

## Effects

### Lesser

- +15% mana pool
- **Scaling**: +1% per class level

### Greater

- +25% mana pool
- +10% regen
- **Scaling**: +0.75% per class level

### Enhanced

- +40% mana pool
- +20% regen
- Deep reserves mechanic
- **Scaling**: +0.5% per class level

## Acquisition

**Trigger Actions**: Deplete mana fully, practice extended casting
**Classes**: All Magic pool
