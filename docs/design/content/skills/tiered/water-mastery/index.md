# Water Mastery

## Overview

Master the fundamentals of water magic to increase spell power, reduce casting costs, and enhance healing and protective effects.

| Property | Value       |
| -------- | ----------- |
| Pool     | Mana        |
| Category | Tiered      |
| Classes  | Hydromancer |

## Effects

### Lesser

- Water spell power: +15%
- Mana cost: -10%
- Healing effectiveness: +20%
- **Scaling**: +0.5% spell power per class level

### Greater

- Water spell power: +30%
- Mana cost: -20%
- Healing effectiveness: +40%
- **Scaling**: +0.75% spell power per class level

### Enhanced

- Water spell power: +50%
- Mana cost: -30%
- Healing effectiveness: +70%
- **Scaling**: +1% spell power per class level

## Acquisition

| Method      | Requirement                |
| ----------- | -------------------------- |
| Class Grant | Hydromancer (Lesser tier)  |
| Level Up    | Hydromancer 10+ (Greater)  |
| Level Up    | Hydromancer 25+ (Enhanced) |
