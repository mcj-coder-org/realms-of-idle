# Earth Spirit Bond

## Overview

Forge a deep bond with the spirits of earth and stone, channeling their ancient power through your shamanic connection.

| Property | Value        |
| -------- | ------------ |
| Pool     | Shamanic     |
| Category | Tiered       |
| Classes  | Earth Shaman |

## Effects

### Lesser

- Spirit Power: +15% to earth-based shamanic effects
- Connection Strength: +20% spirit influence
- Duration: +30% for earth blessings
- Scaling: +1% per Earth Shaman level

### Greater

- Spirit Power: +30% to earth-based shamanic effects
- Connection Strength: +40% spirit influence
- Duration: +60% for earth blessings
- Scaling: +1.5% per Earth Shaman level

### Enhanced

- Spirit Power: +50% to earth-based shamanic effects
- Connection Strength: +70% spirit influence
- Duration: +120% for earth blessings
- Scaling: +2% per Earth Shaman level

## Acquisition

| Method      | Requirement                 |
| ----------- | --------------------------- |
| Class Grant | Earth Shaman (Lesser)       |
| Level Up    | Earth Shaman 10+ (Greater)  |
| Level Up    | Earth Shaman 25+ (Enhanced) |
