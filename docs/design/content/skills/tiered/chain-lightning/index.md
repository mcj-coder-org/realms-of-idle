# Chain Lightning

## Overview

Master the art of electrical chaining, directing lightning to bounce between enemies for devastating cascading damage. Chain Lightning represents elite control over electrical forces, allowing Electromancers to eliminate multiple targets through coordinated electrical discharge.

| Property | Value                       |
| -------- | --------------------------- |
| Pool     | Magic (Lightning)           |
| Category | Tiered                      |
| Classes  | Electromancer               |
| Tags     | `Magic/Elemental/Lightning` |

## Effects

### Lesser

- 30-40 damage per bounce
- Max 3 bounces
- 15m bounce range
- 20% stun chance per bounce (2s stun)
- **Scaling**: +1 damage per class level

### Greater

- 50-70 damage per bounce
- Max 5 bounces
- 20m bounce range
- 35% stun chance per bounce (2-3s stun)
- Chain effectiveness improved (chains to closer targets)
- **Scaling**: +2 damage per class level

### Enhanced

- 80-120 damage per bounce
- Max 7 bounces
- 25m bounce range
- 50% stun chance per bounce (3s stun)
- Chains spread electrical damage across all bounce targets simultaneously
- Bounce efficiency increased (targets further apart don't break chains)
- **Scaling**: +3 damage per class level

## Acquisition

| Method      | Requirement                  |
| ----------- | ---------------------------- |
| Class Grant | Electromancer (Lesser tier)  |
| Level Up    | Electromancer 10+ (Greater)  |
| Level Up    | Electromancer 25+ (Enhanced) |

## Tracked Actions

Actions that contribute to Chain Lightning:

- Execute successful chain lightning bounces
- Stun enemies with electrical discharge
- Defeat multiple enemies with single chain attack
- Achieve maximum bounce chains
