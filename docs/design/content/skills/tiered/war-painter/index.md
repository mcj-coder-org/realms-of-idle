# War Painter

**Pool**: Shamanic
**Type**: Tiered
**Classes**: Shaman, War Painter

## Description

Master the creation of war paints, producing more effective and longer-lasting shamanic markings for warriors.

## Tiers

| Tier     | Effect                                              | Scaling                |
| -------- | --------------------------------------------------- | ---------------------- |
| Lesser   | +15% war paint effectiveness, +25% duration         | +1% per class level    |
| Greater  | +25% effectiveness, +50% duration, complex patterns | +0.75% per class level |
| Enhanced | +40% effectiveness, 2x duration, layered paints     | +0.5% per class level  |

## Acquisition

**Trigger Actions**:

- Create war paints
- Paint warriors for battle

**Classes with Strong Synergy**: Shaman, War Painter (2x XP, direct specialization)

## XP-Based Progression

Skills improve continuously with accumulated XP. More XP = better effectiveness, lower costs, higher success rates.
