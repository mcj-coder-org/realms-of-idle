# Rapid Crafting

**Type:** Tiered
**Pool:** Crafting (General)
**Strong Synergy Classes:** All Crafting specialties

## Effect

| Tier     | Effect                                   | Scaling               |
| -------- | ---------------------------------------- | --------------------- |
| Lesser   | -15% crafting time                       | +0.5% per class level |
| Greater  | -25% crafting time, batch processing     | +0.4% per class level |
| Enhanced | -40% crafting time, instant simple items | +0.3% per class level |

## Trigger Actions

- Craft under time pressure
- Complete many items

## Notes

- Greater tier enables batch processing for identical items
- Enhanced tier allows instant creation of simple items
- Combines well with production-focused crafting strategies
