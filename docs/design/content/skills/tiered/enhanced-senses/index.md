# Enhanced Senses

**Type:** Tiered
**Pool:** Scouting
**Strong Synergy Classes:** Scout, Ranger, Hunter

## Effect

| Tier     | Effect                                          | Scaling               |
| -------- | ----------------------------------------------- | --------------------- |
| Lesser   | +20% detection range; notice hidden details     | +0.5% per class level |
| Greater  | +35% detection; sense movement and disturbances | +0.4% per class level |
| Enhanced | +50% detection; intuit danger and threats       | +0.3% per class level |

## Scaling Examples

**Scout + Enhanced Senses (Strong Synergy)**

- Base effectiveness: Full
- Level 1-10: +20% detection
- Level 25: +32.5% detection
- Level 50: +50% detection

## Trigger Actions

- Track creatures through terrain
- Spot hidden details
- Detect ambushes and traps

## Notes

- Essential for scout specialization
- Lesser tier foundational for Scouting classes
- Applies to all types of sensory detection
- Can reveal hidden enemies and traps
