# Spirit Healing

## Overview

| Attribute | Value                                      |
| --------- | ------------------------------------------ |
| Type      | Tiered (Lesser/Greater/Enhanced)           |
| Pool      | [Shamanic Pool](../pools/shamanic-pool.md) |
| Tags      | `Shamanic/Nature`, `Shamanic/Spirit`       |

## Description

Spirit Healing channels the power of nature and ancestral spirits to mend wounds and cure ailments. Unlike mage healing which uses personal mana, spirit healing draws from the tribal pool and requires spiritual communion.

## Effect by Tier

| Tier     | Healing   | Range  | Cost      | Additional             |
| -------- | --------- | ------ | --------- | ---------------------- |
| Lesser   | 20-30 HP  | Touch  | 15 spirit | Cures minor ailments   |
| Greater  | 40-60 HP  | Short  | 30 spirit | Cures diseases         |
| Enhanced | 80-120 HP | Medium | 50 spirit | Regeneration over time |

## Acquisition

| -------------- | -------------------------- |

| Level Up       | Shaman 10+ (Greater)       |
