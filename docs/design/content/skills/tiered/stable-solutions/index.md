# Stable Solutions

## Overview

Long-lasting alchemical products.

| Property | Value     |
| -------- | --------- |
| Pool     | Alchemy   |
| Category | Tiered    |
| Classes  | Alchemist |

## Effects

### Lesser

- Potions last 2x longer before expiring
- **Scaling**: +5% per class level

### Greater

- Potions last 5x longer
- Resist temperature
- **Scaling**: +4% per class level

### Enhanced

- Potions effectively permanent
- Survive any conditions
- **Scaling**: +3% per class level

## Acquisition

**Trigger Actions**: Store potions long-term, create stable formulas
**Classes**: Alchemist
