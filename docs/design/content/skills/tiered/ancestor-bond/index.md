# Ancestor Bond

## Overview

Forge a deep bond with the spirits of your ancestors, channeling their accumulated wisdom and power.

| Property | Value           |
| -------- | --------------- |
| Pool     | Shamanic        |
| Category | Tiered          |
| Classes  | Ancestor Shaman |

## Effects

### Lesser

- Spirit Power: +15% to ancestral shamanic effects
- Wisdom: +15 to decision-making and knowledge
- Connection Strength: +20% spirit influence
- Scaling: +1% per Ancestor Shaman level

### Greater

- Spirit Power: +30% to ancestral shamanic effects
- Wisdom: +30 to decision-making and knowledge
- Connection Strength: +40% spirit influence
- Ancestral recall: Can access ancestral knowledge for +20% XP gain
- Scaling: +1.5% per Ancestor Shaman level

### Enhanced

- Spirit Power: +50% to ancestral shamanic effects
- Wisdom: +60 to decision-making and knowledge
- Connection Strength: +70% spirit influence
- Ancestral recall: Can access ancestral knowledge for +40% XP gain
- Legacy power: +25% damage/healing from ancestral sources
- Scaling: +2% per Ancestor Shaman level

## Acquisition

| Method      | Requirement                    |
| ----------- | ------------------------------ |
| Class Grant | Ancestor Shaman (Lesser)       |
| Level Up    | Ancestor Shaman 10+ (Greater)  |
| Level Up    | Ancestor Shaman 25+ (Enhanced) |
