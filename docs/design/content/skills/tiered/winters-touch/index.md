# Winter's Touch

## Overview

Channel freezing cold through shamanic power, delivering damage and slowing enemy movements.

| Property | Value        |
| -------- | ------------ |
| Pool     | Shamanic     |
| Category | Tiered       |
| Classes  | Frost Shaman |

## Effects

### Lesser

- Damage per hit: 8-12 HP
- Slow effect: -30% movement speed
- Slow duration: 5 seconds
- Scaling: +0.8 damage per Frost Shaman level

### Greater

- Damage per hit: 16-24 HP
- Slow effect: -50% movement speed
- Slow duration: 8 seconds
- Freeze chance: 20% (complete stop)
- Scaling: +1.5 damage per Frost Shaman level

### Enhanced

- Damage per hit: 32-48 HP
- Slow effect: -70% movement speed
- Slow duration: 12 seconds
- Freeze chance: 40% (complete stop)
- Brittle effect: +25% damage taken for frozen duration
- Scaling: +2.5 damage per Frost Shaman level

## Acquisition

| Method      | Requirement                 |
| ----------- | --------------------------- |
| Class Grant | Frost Shaman (Lesser)       |
| Level Up    | Frost Shaman 10+ (Greater)  |
| Level Up    | Frost Shaman 25+ (Enhanced) |
