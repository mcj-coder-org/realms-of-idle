# Stone Heart

## Overview

Harden your body and spirit with earth magic, granting substantial physical protection.

| Property | Value        |
| -------- | ------------ |
| Pool     | Shamanic     |
| Category | Tiered       |
| Classes  | Earth Shaman |

## Effects

### Lesser

- Armor: +15 physical defense
- Resilience: +10% damage reduction
- Duration: 15 seconds per cast
- Scaling: +1 armor per Earth Shaman level

### Greater

- Armor: +30 physical defense
- Resilience: +20% damage reduction
- Duration: 20 seconds per cast
- Knockback resistance: +50%
- Scaling: +2 armor per Earth Shaman level

### Enhanced

- Armor: +60 physical defense
- Resilience: +35% damage reduction
- Duration: 25 seconds per cast
- Knockback resistance: +100% (immune)
- Reflection: 15% of blocked damage reflects back to attacker
- Scaling: +3 armor per Earth Shaman level

## Acquisition

| Method      | Requirement                 |
| ----------- | --------------------------- |
| Class Grant | Earth Shaman (Lesser)       |
| Level Up    | Earth Shaman 10+ (Greater)  |
| Level Up    | Earth Shaman 25+ (Enhanced) |
