# Silver Tongue (Trade)

## Overview

Superior negotiation in trade.

| Property | Value            |
| -------- | ---------------- |
| Pool     | Trade            |
| Category | Tiered           |
| Classes  | Trader, Merchant |

## Effects

### Lesser

- +15% to buy/sell price negotiations
- **Scaling**: +1% per class level

### Greater

- +25% negotiations
- Can haggle "fixed" prices
- **Scaling**: +0.75% per class level

### Enhanced

- +40% negotiations
- Create exclusive deals
- **Scaling**: +0.5% per class level

## Acquisition

**Trigger Actions**: Negotiate trades, achieve exceptional deals
**Classes**: Trader, Merchant
