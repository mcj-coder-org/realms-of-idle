# Backstab

## Overview

**Type**: Tiered Skill
**Pool**: Stealth
**Description**: Bonus damage from stealth or behind with increasing effectiveness through tiers.

## Synergy Classes

Classes with strong synergies: Thief, Assassin

**Synergy Benefits**: 2x XP acquisition, enhanced effectiveness, reduced costs (scales with class level)

## Tier Progression

| Tier     | Effect                                                    | Scaling               |
| -------- | --------------------------------------------------------- | --------------------- |
| Lesser   | +50% damage from stealth, +25% from behind                | +2% per class level   |
| Greater  | +100% damage from stealth, +50% from behind               | +1.5% per class level |
| Enhanced | +150% damage from stealth, +75% from behind, ignore armor | +1% per class level   |

## XP Sources

- Attack from stealth: 10-20 XP
- Attack from behind: 5-15 XP
- Assassinate: 20-35 XP
- Guided training: 10-100 XP/session
- Unguided training: 5-50 XP/session

## Level-Up Availability

- Strong synergy: ~500 XP (~50 actions)
- Moderate synergy: ~1000 XP
- No synergy: ~1500 XP

## Synergy Examples

### Strong (2x XP)

**Assassin Level 15** - Lethal specialist

- +175% stealth damage
- Ignore armor
- 2x XP from backstabs

### Moderate (1.5x XP)

**Thief Level 10** - Opportunistic fighter

- +110% stealth damage

### None (1x XP)

**Knight** - No connection

- No bonuses
