# Loyalty Cultivation

**Pool**: Leadership
**Type**: Tiered
**Classes**: Leader, Chieftain

## Description

Build stronger, more lasting bonds with followers through consistent leadership and care, accelerating loyalty gains while reducing losses.

## Tiers

| Tier     | Effect                                         | Scaling                |
| -------- | ---------------------------------------------- | ---------------------- |
| Lesser   | +25% loyalty gain rate                         | +1% per class level    |
| Greater  | +40% loyalty gain, -25% loyalty loss           | +0.75% per class level |
| Enhanced | +60% loyalty gain, -50% loss, inspire devotion | +0.5% per class level  |

## Acquisition

**Trigger Actions**:

- Build relationships
- Maintain follower trust

**Classes with Strong Synergy**: Leader, Chieftain (2x XP, direct specialization)

## XP-Based Progression

Skills improve continuously with accumulated XP. More XP = better effectiveness, lower costs, higher success rates.
