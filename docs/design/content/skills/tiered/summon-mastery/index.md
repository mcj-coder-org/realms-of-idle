# Summon Mastery

## Overview

Master the summoning of planar entities to increase conjuration power, improve minion strength, and enhance minion lifespan and effectiveness.

| Property | Value    |
| -------- | -------- |
| Pool     | Mana     |
| Category | Tiered   |
| Classes  | Summoner |

## Effects

### Lesser

- Summoning power: +15%
- Minion strength: +25%
- Minion lifespan: +20%
- Mana cost for summoning: -10%
- **Scaling**: +0.5% power per class level

### Greater

- Summoning power: +30%
- Minion strength: +50%
- Minion lifespan: +50%
- Mana cost for summoning: -20%
- Can maintain additional minions (+1 limit)
- **Scaling**: +0.75% power per class level

### Enhanced

- Summoning power: +50%
- Minion strength: +100%
- Minion lifespan: +100%
- Mana cost for summoning: -30%
- Can maintain additional minions (+2 limit)
- Minions gain enhanced abilities
- **Scaling**: +1% power per class level

## Acquisition

| Method      | Requirement             |
| ----------- | ----------------------- |
| Class Grant | Summoner (Lesser tier)  |
| Level Up    | Summoner 10+ (Greater)  |
| Level Up    | Summoner 25+ (Enhanced) |
