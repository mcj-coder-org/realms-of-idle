# Tiered Skills

**Type**: Tiered
**Description**: Skills with progressive power increases through tiers (Lesser, Greater, Enhanced) with scaling bonuses.

## Overview

Tiered skills provide incremental improvements as you advance through their tiers. Each tier offers enhanced benefits, and synergy classes receive additional scaling bonuses based on their class level.

## Progression

- **Lesser Tier**: Entry-level benefits
- **Greater Tier**: Enhanced benefits with additional effects
- **Enhanced Tier**: Maximum power with elite abilities

## Usage

Tiered skills are acquired through repeated use of related actions. XP requirements vary based on class synergy:

- **Strong Synergy**: ~500 XP (2x learning rate)
- **Moderate Synergy**: ~1000 XP (1.5x learning rate)
- **No Synergy**: ~1500 XP (1x learning rate)
