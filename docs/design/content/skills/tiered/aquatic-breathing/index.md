# Aquatic Breathing

## Overview

Breathe underwater and survive in aquatic environments using water magic to maintain oxygen supply and water pressure resistance.

| Property | Value       |
| -------- | ----------- |
| Pool     | Mana        |
| Category | Tiered      |
| Classes  | Hydromancer |

## Effects

### Lesser

- Underwater breathing: Unlimited duration
- Water pressure resistance: 50m depth
- Aquatic movement speed: +15%
- **Scaling**: +2m depth per class level

### Greater

- Underwater breathing: Unlimited duration
- Water pressure resistance: 150m depth
- Aquatic movement speed: +30%
- Grants water vision in darkness
- **Scaling**: +3m depth per class level

### Enhanced

- Underwater breathing: Unlimited duration
- Water pressure resistance: 300m depth
- Aquatic movement speed: +50%
- Water vision in complete darkness
- Can communicate with aquatic life
- **Scaling**: +4m depth per class level

## Acquisition

| Method      | Requirement                |
| ----------- | -------------------------- |
| Class Grant | Hydromancer (Lesser tier)  |
| Level Up    | Hydromancer 10+ (Greater)  |
| Level Up    | Hydromancer 25+ (Enhanced) |
