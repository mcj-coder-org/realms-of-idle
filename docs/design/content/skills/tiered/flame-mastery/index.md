# Flame Mastery

## Overview

Master the fundamentals of fire magic to increase spell power, reduce casting costs, and extend the duration of burning effects.

| Property | Value      |
| -------- | ---------- |
| Pool     | Mana       |
| Category | Tiered     |
| Classes  | Pyromancer |

## Effects

### Lesser

- Fire spell power: +15%
- Mana cost: -10%
- Burning duration: +25%
- **Scaling**: +0.5% spell power per class level

### Greater

- Fire spell power: +30%
- Mana cost: -20%
- Burning duration: +50%
- **Scaling**: +0.75% spell power per class level

### Enhanced

- Fire spell power: +50%
- Mana cost: -30%
- Burning duration: +100%
- **Scaling**: +1% spell power per class level

## Acquisition

| Method      | Requirement               |
| ----------- | ------------------------- |
| Class Grant | Pyromancer (Lesser tier)  |
| Level Up    | Pyromancer 10+ (Greater)  |
| Level Up    | Pyromancer 25+ (Enhanced) |
