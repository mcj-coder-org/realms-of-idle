# Potent Brews

## Overview

Stronger alchemical creations.

| Property | Value     |
| -------- | --------- |
| Pool     | Alchemy   |
| Category | Tiered    |
| Classes  | Alchemist |

## Effects

### Lesser

- +15% potion/poison effectiveness
- **Scaling**: +1% per class level

### Greater

- +25% effectiveness
- +20% duration
- **Scaling**: +0.75% per class level

### Enhanced

- +40% effectiveness
- +35% duration
- No side effects
- **Scaling**: +0.5% per class level

## Acquisition

**Trigger Actions**: Create potions/poisons, experiment with recipes
**Classes**: Alchemist
