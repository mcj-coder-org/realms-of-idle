# Storm Spirit Bond

## Overview

Forge a deep bond with the spirits of storm and lightning, channeling their power through your shamanic connection.

| Property | Value        |
| -------- | ------------ |
| Pool     | Shamanic     |
| Category | Tiered       |
| Classes  | Storm Caller |

## Effects

### Lesser

- Spirit Power: +15% to lightning-based shamanic effects
- Connection Strength: +20% spirit influence
- Duration: +30% for storm blessings
- Scaling: +1% per Storm Caller level

### Greater

- Spirit Power: +30% to lightning-based shamanic effects
- Connection Strength: +40% spirit influence
- Duration: +60% for storm blessings
- Scaling: +1.5% per Storm Caller level

### Enhanced

- Spirit Power: +50% to lightning-based shamanic effects
- Connection Strength: +70% spirit influence
- Duration: +120% for storm blessings
- Scaling: +2% per Storm Caller level

## Acquisition

| Method      | Requirement                 |
| ----------- | --------------------------- |
| Class Grant | Storm Caller (Lesser)       |
| Level Up    | Storm Caller 10+ (Greater)  |
| Level Up    | Storm Caller 25+ (Enhanced) |
