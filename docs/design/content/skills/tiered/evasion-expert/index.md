# Evasion Expert

## Overview

**Type**: Tiered Skill
**Pool**: Stealth
**Description**: Superior ability to avoid attacks, especially area-of-effect attacks.

## Synergy Classes

Classes with strong synergies: Thief, Assassin, Scout, Adventurer

**Synergy Benefits**: 2x XP acquisition, enhanced effectiveness, reduced costs (scales with class level)

## Tier Progression

| Tier     | Effect                                                        | Scaling                |
| -------- | ------------------------------------------------------------- | ---------------------- |
| Lesser   | +10% dodge, +15% vs AOE attacks                               | +1% per class level    |
| Greater  | +18% dodge, +30% vs AOE, half damage on failed dodge          | +0.75% per class level |
| Enhanced | +25% dodge, +50% vs AOE, no damage on successful dodge vs AOE | +0.5% per class level  |

## XP Sources

- Dodge attacks: 3-8 XP per dodge
- Avoid traps: 10-20 XP
- Evade AOE: 5-15 XP
- Guided training: 10-100 XP/session
- Unguided training: 5-50 XP/session

## Level-Up Availability

- Strong synergy: ~500 XP (~50 actions)
- Moderate synergy: ~1000 XP
- No synergy: ~1500 XP

## Synergy Examples

### Strong (2x XP)

**Assassin Level 15** - Evasion specialist

- +30% dodge
- +60% vs AOE
- 2x XP from evasion

### Moderate (1.5x XP)

**Adventurer Level 10** - Mobile fighter

- +20% dodge

### None (1x XP)

**Guard** - No connection

- No bonuses
