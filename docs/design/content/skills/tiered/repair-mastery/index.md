# Repair Mastery

**Type:** Tiered
**Pool:** Crafting (General)
**Strong Synergy Classes:** All Crafting specialties (for their specialty)

## Effect

| Tier     | Effect                                                  | Scaling               |
| -------- | ------------------------------------------------------- | --------------------- |
| Lesser   | Repairs restore +15% more durability                    | +0.5% per class level |
| Greater  | Repairs restore +25% more, can repair "destroyed" items | +0.4% per class level |
| Enhanced | Repairs restore +40% more, repairs may improve items    | +0.3% per class level |

## Trigger Actions

- Repair items
- Restore damaged equipment

## Notes

- Greater tier allows repair of items normally beyond saving
- Enhanced tier can accidentally improve items during repair
- Each crafter applies this to their specialty
