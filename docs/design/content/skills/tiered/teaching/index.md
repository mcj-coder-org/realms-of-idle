# Teaching

**Pool**: Scholar
**Type**: Tiered
**Classes**: Trainer, Scholar

## Description

Effectively teach others, accelerating their learning and eventually enabling advanced or simultaneous instruction.

## Tiers

| Tier     | Effect                                                  | Scaling                |
| -------- | ------------------------------------------------------- | ---------------------- |
| Lesser   | Students learn +15% faster                              | +1% per class level    |
| Greater  | Students +25% faster, can teach advanced material       | +0.75% per class level |
| Enhanced | Students +40% faster, can teach multiple simultaneously | +0.5% per class level  |

## Acquisition

**Trigger Actions**:

- Train others
- Teach skills

**Classes with Strong Synergy**: Trainer, Scholar (2x XP, direct specialization)

## XP-Based Progression

Skills improve continuously with accumulated XP. More XP = better effectiveness, lower costs, higher success rates.
