# Lockmaster

## Overview

**Type**: Tiered Skill
**Pool**: Stealth
**Description**: Superior lockpicking ability with increasing success rates through tiers.

## Synergy Classes

Classes with strong synergies: Thief, Scout

**Synergy Benefits**: 2x XP acquisition, enhanced effectiveness, reduced costs (scales with class level)

## Tier Progression

| Tier     | Effect                                             | Scaling                |
| -------- | -------------------------------------------------- | ---------------------- |
| Lesser   | +20% lockpick success, -20% lockpick breakage      | +1% per class level    |
| Greater  | +35% success, -40% breakage, pick magical locks    | +0.75% per class level |
| Enhanced | +50% success, -60% breakage, sense lock complexity | +0.5% per class level  |

## XP Sources

- Pick locks successfully: 5-15 XP per lock
- Open difficult locks: 15-30 XP
- Guided training: 10-100 XP/session
- Unguided training: 5-50 XP/session

## Level-Up Availability

- Strong synergy: ~500 XP (~50 actions)
- Moderate synergy: ~1000 XP
- No synergy: ~1500 XP

## Synergy Examples

### Strong (2x XP)

**Thief Level 15** - Lockpicking specialist

- +60% success
- -70% breakage
- 2x XP from picking locks

### Moderate (1.5x XP)

**Scout Level 10** - Infiltration expert

- +40% success

### None (1x XP)

**Blacksmith** - No connection

- No bonuses
