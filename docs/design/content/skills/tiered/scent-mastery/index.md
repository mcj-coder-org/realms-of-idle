# Scent Mastery

**Type:** Tiered
**Pool:** Animal
**Strong Synergy Classes:** Hunter, Trapper, Beastmaster

## Effect

| Tier     | Effect                                                             | Scaling                |
| -------- | ------------------------------------------------------------------ | ---------------------- |
| Lesser   | Mask your scent, +15% tracking via scent                           | +1% per class level    |
| Greater  | Create false scent trails, +25% tracking                           | +0.75% per class level |
| Enhanced | Scent invisible to animals, +40% tracking, ID individuals by scent | +0.5% per class level  |

## Trigger Actions

- Track by scent
- Avoid animal detection

## Notes

- Dual utility: defensive (hide scent) and offensive (track)
- Enhanced tier allows identification of individuals by scent signature
- False scent trails can mislead pursuers
