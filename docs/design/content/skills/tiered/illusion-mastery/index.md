# Illusion Mastery

## Overview

Master the fundamentals of illusion magic to increase spell power, strengthen deceptive effects, and enhance perception manipulation capabilities.

| Property | Value       |
| -------- | ----------- |
| Pool     | Mana        |
| Category | Tiered      |
| Classes  | Illusionist |

## Effects

### Lesser

- Illusion power: +15%
- Deception effectiveness: +20%
- Mana cost: -10%
- Illusion detection difficulty: +15%
- **Scaling**: +0.5% power per class level

### Greater

- Illusion power: +30%
- Deception effectiveness: +40%
- Mana cost: -20%
- Illusion detection difficulty: +30%
- Illusions gain enhanced realism
- **Scaling**: +0.75% power per class level

### Enhanced

- Illusion power: +50%
- Deception effectiveness: +70%
- Mana cost: -30%
- Illusion detection difficulty: +50%
- Illusions can affect multiple senses simultaneously
- Can create persistent illusion zones
- **Scaling**: +1% power per class level

## Acquisition

| Method      | Requirement                |
| ----------- | -------------------------- |
| Class Grant | Illusionist (Lesser tier)  |
| Level Up    | Illusionist 10+ (Greater)  |
| Level Up    | Illusionist 25+ (Enhanced) |
