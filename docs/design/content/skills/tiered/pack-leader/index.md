# Pack Leader

## Overview

Lead animal companions and summoned beasts with tactical command and protective instincts.

| Property | Value         |
| -------- | ------------- |
| Pool     | Shamanic      |
| Category | Tiered        |
| Classes  | Beast Speaker |

## Effects

### Lesser

- Allies: Can command up to 2 animal allies
- Bonuses: +15% damage to commanded allies
- Range: 15m from you
- Scaling: +1% per Beast Speaker level

### Greater

- Allies: Can command up to 4 animal allies
- Bonuses: +30% damage to commanded allies
- Range: 25m from you
- Pack tactics: Allies gain +20% accuracy when attacking same target
- Scaling: +1.5% per Beast Speaker level

### Enhanced

- Allies: Can command up to 6 animal allies
- Bonuses: +50% damage to commanded allies
- Range: 35m from you
- Pack tactics: Allies gain +40% accuracy and +20% damage when attacking same target
- Coordinated: Commanded allies can coordinate special attack patterns
- Scaling: +2% per Beast Speaker level

## Acquisition

| Method      | Requirement                  |
| ----------- | ---------------------------- |
| Class Grant | Beast Speaker (Lesser)       |
| Level Up    | Beast Speaker 10+ (Greater)  |
| Level Up    | Beast Speaker 25+ (Enhanced) |
