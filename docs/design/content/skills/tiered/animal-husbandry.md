# Animal Husbandry

## Overview

**Type**: Tiered Skill
**Pool**: Agriculture
**Description**: Better livestock management with improved yields, breeding success, and animal health.

## Synergy Classes

Classes with strong synergies: Rancher, Farmer

**Synergy Benefits**: 2x XP acquisition, enhanced effectiveness, reduced costs (scales with class level)

## Tier Progression

| Tier     | Effect                                               | Scaling                |
| -------- | ---------------------------------------------------- | ---------------------- |
| Lesser   | +15% animal product yield, +10% breeding success     | +1% per class level    |
| Greater  | +25% yield, +20% breeding, animals healthier         | +0.75% per class level |
| Enhanced | +40% yield, +35% breeding, animals bond (won't flee) | +0.5% per class level  |

## XP Sources

- Tend animals: 3-8 XP per session
- Breed livestock: 15-25 XP per breeding
- Manage herds: 10-20 XP per day
- Guided training: 10-100 XP/session
- Unguided training: 5-50 XP/session

## Level-Up Availability

- Strong synergy: ~500 XP (~50 actions)
- Moderate synergy: ~1000 XP
- No synergy: ~1500 XP

## Synergy Examples

### Strong (2x XP)

**Rancher Level 15** - Livestock specialist

- +50% yield
- +45% breeding
- 2x XP from animal husbandry

### Moderate (1.5x XP)

**Farmer Level 10** - Farm manager

- +30% yield

### None (1x XP)

**Mage** - No connection

- No bonuses
