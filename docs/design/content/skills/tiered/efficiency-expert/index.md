# Efficiency Expert

## Overview

**Type**: Tiered Skill
**Pool**: Gathering
**Description**: Reduced stamina and time for gathering with automatic gathering at enhanced tier.

## Synergy Classes

Classes with strong synergies: All Gathering pool (Forager, Miner, Hunter, Fisher, Lumberjack, Trapper, Herbalist)

**Synergy Benefits**: 2x XP acquisition, enhanced effectiveness, reduced costs (scales with class level)

## Tier Progression

| Tier     | Effect                                               | Scaling               |
| -------- | ---------------------------------------------------- | --------------------- |
| Lesser   | -15% stamina, -10% time                              | +0.5% per class level |
| Greater  | -25% stamina, -20% time                              | +0.4% per class level |
| Enhanced | -35% stamina, -30% time, auto-gather while traveling | +0.3% per class level |

## XP Sources

- Extended gathering sessions: 5-10 XP per session
- Optimize gathering routes: 10-20 XP
- Guided training: 10-100 XP/session
- Unguided training: 5-50 XP/session

## Level-Up Availability

- Strong synergy: ~500 XP (~50 actions)
- Moderate synergy: ~1000 XP
- No synergy: ~1500 XP
