# Burning Aura

## Overview

Surround yourself with a constant aura of flame that damages nearby enemies and can ignite them for ongoing damage.

| Property | Value      |
| -------- | ---------- |
| Pool     | Mana       |
| Category | Tiered     |
| Classes  | Pyromancer |

## Effects

### Lesser

- Damage per turn: 5-8 HP
- Aura range: 5m
- Area of effect: 8m radius
- Burn chance: 25%
- **Scaling**: +1 damage per class level

### Greater

- Damage per turn: 10-15 HP
- Aura range: 8m
- Area of effect: 12m radius
- Burn chance: 40%
- **Scaling**: +1.5 damage per class level

### Enhanced

- Damage per turn: 20-30 HP
- Aura range: 10m
- Area of effect: 15m radius
- Burn chance: 60%
- **Scaling**: +2 damage per class level

## Acquisition

| Method      | Requirement               |
| ----------- | ------------------------- |
| Class Grant | Pyromancer (Lesser tier)  |
| Level Up    | Pyromancer 10+ (Greater)  |
| Level Up    | Pyromancer 25+ (Enhanced) |
