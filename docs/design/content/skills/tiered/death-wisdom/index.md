# Death Wisdom

## Overview

Channel the wisdom of death and transition, protecting allies from death effects and guiding souls.

| Property | Value           |
| -------- | --------------- |
| Pool     | Shamanic        |
| Category | Tiered          |
| Classes  | Ancestor Shaman |

## Effects

### Lesser

- Protection: +20% resistance to death effects
- Duration: 15 seconds per cast
- Healing on death prevention: Restore 15% HP
- Scaling: +1% per Ancestor Shaman level

### Greater

- Protection: +40% resistance to death effects
- Duration: 25 seconds per cast
- Healing on death prevention: Restore 30% HP
- Return from brink: Once per combat, prevent death (5 seconds cooldown)
- Scaling: +1.5% per Ancestor Shaman level

### Enhanced

- Protection: +60% resistance to death effects
- Duration: 35 seconds per cast
- Healing on death prevention: Restore 50% HP
- Return from brink: Twice per combat, prevent death (10 seconds cooldown)
- Ancestral intervention: When prevented from death, gain +50% damage for 10 seconds
- Scaling: +2% per Ancestor Shaman level

## Acquisition

| Method      | Requirement                    |
| ----------- | ------------------------------ |
| Class Grant | Ancestor Shaman (Lesser)       |
| Level Up    | Ancestor Shaman 10+ (Greater)  |
| Level Up    | Ancestor Shaman 25+ (Enhanced) |
