# Master Forager

## Overview

**Type**: Passive Generator
**Pool**: Gathering
**Description**: Daily passive resource discovery during normal travel.

## Synergy Classes

Classes with strong synergies: Forager, Herbalist

**Synergy Benefits**: 2x XP acquisition, enhanced effectiveness, reduced costs (scales with class level)

## Tier Progression

| Tier     | Daily Effect                                                  |
| -------- | ------------------------------------------------------------- |
| Lesser   | Find 1-3 common herbs/materials during normal travel          |
| Greater  | Find 2-5 materials including uncommon during travel           |
| Enhanced | Find 3-8 materials including rare, chance of exceptional find |

## XP Sources

- Consistently gather: 3-8 XP per day
- Develop foraging habits: 10-20 XP per week
- Guided training: 10-100 XP/session
- Unguided training: 5-50 XP/session

## Level-Up Availability

- Strong synergy: ~500 XP (~50 actions)
- Moderate synergy: ~1000 XP
- No synergy: ~1500 XP

## Synergy Examples

### Strong (2x XP)

- More rare materials
- 2x XP from consistent gathering

### Moderate (1.5x XP)

**Herbalist Level 10** - Regular collector

- Better uncommon finds

### None (1x XP)

**Warrior** - No connection
