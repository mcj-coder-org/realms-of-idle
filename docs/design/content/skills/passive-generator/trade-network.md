# Trade Network

## Overview

Generate trade opportunities.

| Property | Value                       |
| -------- | --------------------------- |
| Pool     | Trade                       |
| Category | Passive Generator           |
| Classes  | Trader, Merchant, Caravaner |

## Effects

### Lesser

- Weekly: Learn of 1-2 trade opportunities

### Greater

- Weekly: 2-4 opportunities, including profitable routes

### Enhanced

- Weekly: 3-6 opportunities, including exclusive deals

## Acquisition

**Trigger Actions**: Maintain trade contacts, build reputation
**Classes**: Trader, Merchant, Caravaner
