# Trade Contacts

## Overview

Build useful contacts over time.

| Property | Value                       |
| -------- | --------------------------- |
| Pool     | Trade                       |
| Category | Passive Generator           |
| Classes  | Trader, Merchant, Caravaner |

## Effects

### Lesser

- Gain 1 trade contact in visited settlements monthly

### Greater

- Gain 2 contacts monthly
- Contacts offer better deals

### Enhanced

- Gain 3 contacts monthly
- Contacts share exclusive information

## Acquisition

**Trigger Actions**: Trade with varied merchants, maintain relationships
**Classes**: Trader, Merchant, Caravaner
