# Recipe Innovator

**Type:** Passive Generator
**Pool:** Crafting (General)
**Strong Synergy Classes:** All Crafting specialties

## Effect

| Tier     | Effect                                                       |
| -------- | ------------------------------------------------------------ |
| Lesser   | Monthly chance to conceive new recipe variant                |
| Greater  | Bi-weekly chance, variants may be improvements               |
| Enhanced | Weekly chance, can deliberately experiment with +50% success |

## Trigger Actions

- Experiment with crafting
- Create variations

## Notes

- Generates new recipe variations over time
- Enhanced tier allows deliberate experimentation with good success rates
- Can lead to unique crafting recipes unavailable elsewhere
