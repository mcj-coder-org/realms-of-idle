# Underworld Connections

**Pool**: Dark
**Type**: Passive Generator
**Classes**: All Dark pool

## Description

Your reputation in the criminal underworld generates a steady stream of illegal opportunities, contacts, and resources.

## Tiers

| Tier     | Effect                                                |
| -------- | ----------------------------------------------------- |
| Lesser   | Weekly: Learn of 1-2 criminal opportunities           |
| Greater  | Weekly: 2-4 opportunities, fence access, safe houses  |
| Enhanced | Weekly: 3-6 opportunities, exclusive jobs, protection |

## Acquisition

**Trigger Actions**:

- Operate in underground
- Build criminal reputation

**Classes with Strong Synergy**: All Dark pool (2x XP, direct specialization)

## Fixed Generation

Generation rate is fixed per tier - provides consistent criminal opportunities based on your underground standing.
