# Reagent Cultivation

## Overview

Grow alchemical ingredients.

| Property | Value                |
| -------- | -------------------- |
| Pool     | Alchemy              |
| Category | Passive Generator    |
| Classes  | Alchemist, Herbalist |

## Effects

### Lesser

- Produce 5-10 common reagents weekly

### Greater

- Produce 8-15 reagents including uncommon weekly

### Enhanced

- Produce 12-20 reagents including rare weekly

## Requirements

**Requires**: Dedicated growing space

## Acquisition

**Trigger Actions**: Cultivate reagent garden, tend alchemical plants
**Classes**: Alchemist, Herbalist
