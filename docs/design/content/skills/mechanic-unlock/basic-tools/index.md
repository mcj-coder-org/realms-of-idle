# Basic Tools

**Type:** Mechanic Unlock
**Pool:** Gathering
**Strong Synergy Classes:** Gatherer, Miner, Herbalist

## Effect

| Aspect    | Description                                   |
| --------- | --------------------------------------------- |
| Unlock    | Access and use basic gathering tools          |
| Privilege | Use pickaxes, sickles, and collection devices |
| Quality   | Base gathering yield improvement              |

## Mechanics

- Allows use of pickaxes for mining
- Enables use of sickles for harvesting
- Permits use of nets and baskets
- +10% base gathering yield

## Trigger Actions

- Gather first resource
- Use gathering tools
- Complete gathering tasks

## Notes

- Foundational for Gatherer class
- Usually auto-granted when accepting Gatherer class
- Prerequisite for advanced tool specializations
