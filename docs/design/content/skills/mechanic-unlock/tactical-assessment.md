# Tactical Assessment

## Overview

Analyze enemy capabilities to reveal health, resistances, damage types, and threat levels. Provides both active examination and passive level detection.

## Synergy Classes

**Strong Synergy (2x XP)**: Warrior, Adventurer, Knight, Guard, Mercenary, Scout

Classes with strong synergies get faster learning, better effectiveness, and lower costs.

## XP Progression

**XP Sources**:

- Fight varied enemy types: 5-10 XP per unique enemy type
- Survive difficult encounters: 10-25 XP per challenging victory
- Examine enemies: 2-5 XP per examination
- Guided training: 10-100 XP per session
- Unguided training: 5-50 XP per session

**Level-Up Availability Thresholds**:

- Strong synergy classes: ~500 XP (~50 actions)
- Moderate synergy classes: ~1000 XP (~75 actions)
- No synergy classes: ~1500 XP (~100-150 actions)

## Mechanics

**Trigger Actions:** Fight varied enemy types, survive difficult encounters

**Examination:** Active ability to target an enemy and receive detailed combat information.

**Information Revealed:**

- Health percentage (precise)
- Damage resistances and vulnerabilities
- Primary damage type(s)
- Threat level relative to character
- Special abilities (if significantly below character level)

**Passive Detection:** Automatically sense enemy threat level (color-coded) without needing to examine.

**No Cooldown:** Can examine multiple enemies per combat without limitation.
