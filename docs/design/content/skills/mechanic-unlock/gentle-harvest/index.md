# Gentle Harvest

## Overview

**Type**: Mechanic Unlock
**Pool**: Gathering
**Description**: Gather without depleting nodes, promoting sustainable resource collection.

## Synergy Classes

Classes with strong synergies: Forager, Herbalist, Farmer

**Synergy Benefits**: 2x XP acquisition, enhanced effectiveness, reduced costs (scales with class level)

## Mechanics

| Effect      | Description                                       |
| ----------- | ------------------------------------------------- |
| Unlock      | 25% chance node doesn't deplete when gathered     |
| Sustainable | Node regeneration +50% faster if gentle harvested |
| Quality     | No quality penalty                                |

## XP Sources

- Gather from same nodes repeatedly: 5-10 XP
- Practice sustainable harvesting: 10-20 XP
- Guided training: 10-100 XP/session
- Unguided training: 5-50 XP/session

## Level-Up Availability

- Strong synergy: ~500 XP (~50 actions)
- Moderate synergy: ~1000 XP
- No synergy: ~1500 XP
