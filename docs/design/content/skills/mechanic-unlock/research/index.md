# Research

**Pool**: Scholar
**Type**: Mechanic Unlock
**Classes**: Scholar, Scribe

## Description

Systematically research topics in libraries and archives to discover hidden information, recipes, locations, and weaknesses.

## Effect

| Property | Description                                              |
| -------- | -------------------------------------------------------- |
| Unlock   | Can research topics in libraries/archives                |
| Outcomes | Learn hidden information, recipes, locations, weaknesses |
| Time     | Hours to days depending on topic rarity                  |

## Acquisition

**Note**: Core Scholar mechanic

**Trigger Actions**:

- Use libraries
- Investigate topics

**Classes with Strong Synergy**: Scholar, Scribe (2x XP, direct specialization)

## Related Content

- **Strong Synergy:** [Scholar](../../classes/fighter/), [Scribe](../../classes/scribe/)
- **See Also:** [Skills Index](../index.md)

## No Scaling

Binary unlock - research time and success based on topic difficulty and skill progression.
