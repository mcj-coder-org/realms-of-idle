# Blackmail

**Pool**: Dark
**Type**: Mechanic Unlock
**Classes**: Thief, Assassin, Spy

## Description

Exploit discovered secrets to extract ongoing benefits from NPCs through the threat of exposure.

## Effect

| Property | Description                                               |
| -------- | --------------------------------------------------------- |
| Unlock   | Can blackmail NPCs with discovered secrets                |
| Returns  | Regular payments, favors, information                     |
| Risk     | Target may retaliate, expose you first, or pay to silence |

## Acquisition

**Trigger Actions**:

- Discover secrets
- Leverage information

**Classes with Strong Synergy**: Thief, Assassin, Spy (2x XP, direct specialization)

## No Scaling

Binary unlock - returns based on secret value and target's resources. Risks remain constant.
