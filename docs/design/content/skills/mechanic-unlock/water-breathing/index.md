# Water Breathing

**Pool**: Cross-Class
**Type**: Mechanic Unlock
**Classes**: Fisher, Sailor, certain Mage schools

## Description

Breathe underwater without limit, enabling extended underwater operations and exploration.

## Effect

| Property      | Description                           |
| ------------- | ------------------------------------- |
| Unlock        | Can breathe underwater without limit  |
| Depth         | No pressure damage to moderate depths |
| Communication | Can speak underwater (muffled)        |

## Acquisition

**Trigger Actions**:

- Extended underwater activity
- Near-drowning survival

**Classes with Strong Synergy**: Fisher, Sailor, certain Mage schools (2x XP, direct specialization)

## No Scaling

Binary unlock - breathing is either enabled or not. No tiers or scaling.
