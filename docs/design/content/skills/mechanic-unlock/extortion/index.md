# Extortion

**Pool**: Dark
**Type**: Mechanic Unlock (Multi-Tier)
**Classes**: Extortionist, Thief

## Description

Systematically extract payment from NPCs through credible threats, establishing ongoing protection rackets.

## Tiers

### Tier 1 (Basic)

| Property    | Description                                |
| ----------- | ------------------------------------------ |
| Unlock      | Can extort NPCs for regular payments       |
| Calculation | Based on target wealth, your threat level  |
| Risk        | May report to authorities, hire protection |

### Tier 2: Professional Extortion

| Property | Description                   |
| -------- | ----------------------------- |
| Effect   | Better rates, less resistance |

## Acquisition

**Trigger Actions**:

- Threaten for profit
- Maintain intimidation

**Classes with Strong Synergy**: Extortionist, Thief (2x XP, direct specialization)

## Related Content

- **Strong Synergy:** [Extortionist](../../classes/extortionist/), [Thief](../../classes/rogue/thief/)

## Multi-Tier Unlock

Each tier improves rates and reduces risks. Progress by building reputation and maintaining control.
