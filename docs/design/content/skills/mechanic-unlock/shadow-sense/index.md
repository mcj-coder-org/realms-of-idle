# Shadow Sense

**Type:** Mechanic Unlock
**Pool:** Rogue
**Strong Synergy Classes:** Rogue, Assassin, Scout

## Effect

| Aspect    | Description                                      |
| --------- | ------------------------------------------------ |
| Unlock    | Awareness of hiding spots and patrol patterns    |
| Technique | Identify shadowed areas suitable for concealment |
| Quality   | +15% stealth effectiveness in shadows            |

## Mechanics

- Identify ideal hiding locations
- Sense patrol patterns
- Navigate around guards
- Exploit shadows for concealment

## Trigger Actions

- Use stealth in shadowed areas
- Scout enemy patrol routes
- Successfully hide from observers

- Foundational for Rogue stealth playstyle
