# Totem Crafting

**Pool**: Shamanic
**Type**: Mechanic Unlock
**Classes**: Shaman

## Description

Create powerful totems that channel spiritual energy to provide benefits to those nearby.

## Effect

| Property | Description                                   |
| -------- | --------------------------------------------- |
| Unlock   | Can craft totems that provide area benefits   |
| Types    | Camp totem, war totem, sacred totem           |
| Power    | Totem strength based on spirit pool and skill |

## Acquisition

**Note**: Core Shaman crafting mechanic

**Trigger Actions**:

- Create totems
- Maintain sacred items

**Classes with Strong Synergy**: Shaman (2x XP, direct specialization)

Binary unlock - no tiers or scaling. Totem power scales with spirit pool and overall skill progression.
