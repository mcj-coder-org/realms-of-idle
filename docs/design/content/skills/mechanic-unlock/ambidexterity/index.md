# Ambidexterity

**Pool**: Cross-Class
**Type**: Mechanic Unlock
**Classes**: Warrior, Thief, Crafter specialties

## Description

Eliminate off-hand penalties entirely, using either hand with equal proficiency in combat, crafting, and all activities.

## Effect

| Property | Description                        |
| -------- | ---------------------------------- |
| Unlock   | No off-hand penalty for any action |
| Combat   | Full damage with either weapon     |
| Crafting | Can work with either hand          |

## Acquisition

**Trigger Actions**:

- Practice off-hand
- Train both hands

**Classes with Strong Synergy**: Warrior, Thief, Crafter specialties (2x XP, direct specialization)

## No Scaling

Binary unlock - either you have the penalty or you don't. No tiers or scaling.
