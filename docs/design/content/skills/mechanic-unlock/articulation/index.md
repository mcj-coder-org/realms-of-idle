# Articulation

**Type:** Mechanic Unlock
**Pool:** Smithing
**Strong Synergy Classes:** Armorsmith, Weaponsmith

## Effect

| Aspect    | Description                                            |
| --------- | ------------------------------------------------------ |
| Unlock    | Create flexible armor that allows full range of motion |
| Technique | Join metal plates without restricting movement         |
| Quality   | Improve wearer comfort without sacrificing protection  |

## Mechanics

- Allows crafting of armor with bonus mobility
- Enables joint articulation (shoulders, elbows, knees)
- Reduces encumbrance penalties for heavy armor
- Creates legendary armor combinations

## Trigger Actions

- Craft armor with movement requirements
- Create specialized joint pieces
- Combine protection with mobility

## Notes

- Transforms armor from restrictive to functional
- Critical for warrior and rogue hybrid armor
- Base requirement for master armor smithing
