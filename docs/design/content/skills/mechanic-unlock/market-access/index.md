# Market Access

**Type:** Mechanic Unlock
**Pool:** Trading
**Strong Synergy Classes:** Trader, Merchant, Broker

## Effect

| Aspect    | Description                                    |
| --------- | ---------------------------------------------- |
| Unlock    | Access to basic market information and trading |
| Privilege | Access to village and town markets             |
| Quality   | View prices, place buy/sell orders             |

## Mechanics

- Access market boards in settlements
- View current prices and available items
- Place buying and selling orders
- Access trade routes
- View market trends and history

## Trigger Actions

- Visit first market
- Complete first trading transaction
- Request market access from merchants

## Notes

- Foundational for Trader class
- Usually auto-granted when accepting Trader class
- Opens access to advanced trading features
- Required for merchant guild membership
