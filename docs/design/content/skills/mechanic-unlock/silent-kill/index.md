# Silent Kill

## Overview

**Type**: Mechanic Unlock
**Pool**: Stealth
**Description**: Eliminate targets without alerting nearby enemies.

## Synergy Classes

Classes with strong synergies: Assassin

**Synergy Benefits**: 2x XP acquisition, enhanced effectiveness, reduced costs (scales with class level)

## Mechanics

| Effect     | Description                                                 |
| ---------- | ----------------------------------------------------------- |
| Unlock     | Stealth kills don't alert nearby enemies (within reason)    |
| Body       | Target dies silently, body can be hidden instantly          |
| Limitation | Doesn't work if target in conversation or directly observed |

- Kill without detection: 20-35 XP
