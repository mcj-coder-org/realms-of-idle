**Type:** Mechanic Unlock
**Pool:** Knowledge/Crafting
**Strong Synergy Classes:** Tailor, Blacksmith

## Effect

| Aspect     | Description                                         |
| ---------- | --------------------------------------------------- |
| Unlock     | Can create and apply heraldic designs correctly     |
| Benefit    | Heraldic items: legitimacy, +100% value to nobility |
| Limitation | Must know proper heraldic rules and traditions      |
