# Mounted Combat

## Overview

Fight effectively from mount without penalty. Enables powerful mounted charges, seamless mount control during combat, and advanced mounted maneuvers at higher tiers.

## Synergy Classes

**Strong Synergy (2x XP)**: Knight, Adventurer, Caravaner

Classes with strong synergies get faster learning, better effectiveness, and lower costs.

## Tiers

**Base unlock is binary, with additional tiers available**

| Effect  | Description                                                       |
| ------- | ----------------------------------------------------------------- |
| Unlock  | No penalty when fighting mounted                                  |
| Charge  | Can perform mounted charge (bonus damage, breakthrough potential) |
| Control | Mount follows combat commands without check                       |

## XP Progression

**XP Sources**:

- Ride mounts: 2-5 XP per hour
- Fight while mounted: 10-20 XP per mounted combat
- Train with mounts: 5-15 XP per training session
- Perform mounted charges: 10-25 XP per successful charge
- Guided training: 10-100 XP per session
- Unguided training: 5-50 XP per session

**Level-Up Availability Thresholds**:

- Strong synergy classes: ~500 XP (~50 actions)
- Moderate synergy classes: ~1000 XP (~75 actions)
- No synergy classes: ~1500 XP (~100-150 actions)

## Mechanics

**Trigger Actions:** Ride mounts, fight while mounted, train with mounts

**Base Penalty Removal:** Without this skill, fighting mounted imposes significant accuracy and attack speed penalties. This skill removes those penalties.

**Mounted Charge:** Special attack performed while mount is moving at speed. Deals bonus damage and can break through enemy lines.

**Mount Control:** Mount automatically responds to combat commands without requiring Animal Handling checks.

**Tier 2 - Mounted Mastery:**

- Charge damage increased by +50%
- Can make attacks while mount is moving (normally requires mount to be stationary)
- Mount gains combat awareness and will avoid obvious dangers

**Mount Types:** Works with any rideable mount (horses, wolves, etc.). Effectiveness may vary by mount type.
