# Iron Will

**Pool**: Cross-Class
**Type**: Mechanic Unlock
**Classes**: Knight, certain Mage schools, Shaman

## Description

Complete immunity to mental control and domination, with enhanced resistance to all other mental effects.

## Effect

| Property   | Description                               |
| ---------- | ----------------------------------------- |
| Unlock     | Immune to charm, domination, mind control |
| Resistance | +50% vs other mental effects              |
| Detection  | Know when mental attack attempted         |

## Acquisition

**Trigger Actions**:

- Resist mental attacks
- Maintain mental discipline

**Classes with Strong Synergy**: Knight, certain Mage schools, Shaman (2x XP, direct specialization)

- **Strong Synergy:** [Knight](../../classes/knight/), [certain Mage schools](../../classes/certain mage schools/), [Shaman](../../classes/spiritcaller/shaman/)
- **See Also:** [Skills Index](../index.md)

## No Scaling

Binary unlock - immunity is absolute, resistance bonus is fixed.
