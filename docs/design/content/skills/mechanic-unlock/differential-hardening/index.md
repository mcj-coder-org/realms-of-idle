# Differential Hardening

**Type:** Mechanic Unlock
**Pool:** Smithing
**Strong Synergy Classes:** Blacksmith
**Prerequisites:** Heat Treatment

## Effect

| Aspect     | Description                                                |
| ---------- | ---------------------------------------------------------- |
| Unlock     | Can create blades with hard edge and flexible spine        |
| Benefit    | Diff-hardened blades: +30% edge retention, +20% durability |
| Limitation | Requires clay coating and precise quenching technique      |

- Traditional technique for katana and similar blades
