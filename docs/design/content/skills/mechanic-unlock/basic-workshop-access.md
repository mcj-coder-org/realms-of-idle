# Basic Workshop Access

**Type:** Mechanic Unlock
**Pool:** Crafting (General)
**Strong Synergy Classes:** All Crafting classes

## Effect

| Aspect    | Description                                   |
| --------- | --------------------------------------------- |
| Unlock    | Access and use basic crafting workspaces      |
| Privilege | Use communal or rental workshop facilities    |
| Quality   | Quality baseline: standard workshop equipment |

## Mechanics

- Allows use of village and town workshops
- Access to basic tools and equipment
- Enables crafting outside personal workspace
- Required for guild workshop access

## Trigger Actions

- Use first crafting workspace
- Complete first crafting project in workspace
- Request workshop access from NPCs

## Notes

- Foundational for any crafter
- Usually auto-granted when accepting Crafter class
- Can be upgraded to master workshop access
