# Beadwork

**Type:** Mechanic Unlock
**Pool:** General Crafting
**Strong Synergy Classes:** Jeweler, Tailor

## Effect

| Aspect     | Description                                               |
| ---------- | --------------------------------------------------------- |
| Unlock     | Can create decorative patterns using beads                |
| Benefit    | Beaded items: +50% aesthetic value, cultural significance |
| Limitation | Requires quality beads and patience                       |

## Notes

- Traditional craft in many cultures
- Can incorporate precious and semi-precious beads
- Used for jewelry, clothing trim, ceremonial items
- Some patterns have magical significance
- Can tell stories through bead patterns
