# Undead Binding

## Overview

Control undead creatures.

| Property | Value           |
| -------- | --------------- |

## Effects

| Effect | Description                                   |
| ------ | --------------------------------------------- |

| Limit  | Control limit based on class level            |
| Types  | Starts with basic (zombie, skeleton), expands |
