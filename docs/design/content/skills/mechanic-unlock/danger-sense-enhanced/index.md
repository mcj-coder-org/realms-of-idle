# Danger Sense (Enhanced)

**Pool**: Cross-Class
**Type**: Mechanic Unlock
**Classes**: Scout, Assassin, Adventurer, certain Mage paths

## Description

Supernatural awareness of danger before it manifests, providing brief but reliable warning of threats.

## Effect

| Property | Description                                  |
| -------- | -------------------------------------------- |
| Unlock   | Sense danger before it manifests             |
| Types    | Traps, ambushes, betrayal, natural disasters |
| Warning  | Brief (seconds) but reliable                 |

## Acquisition

**Trigger Actions**:

- Survive ambushes
- Avoid disasters

**Classes with Strong Synergy**: Scout, Assassin, Adventurer, certain Mage paths (2x XP, direct specialization)

**Note**: This is an enhanced version of the common Danger Sense skill, available to specialized classes.

## No Scaling

Binary unlock - warning is either provided or not. Reliability is fixed.
