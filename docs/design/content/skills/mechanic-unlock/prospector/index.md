# Prospector

## Overview

**Type**: Mechanic Unlock
**Pool**: Gathering
**Description**: Discover new resource nodes through area surveys.

## Synergy Classes

Classes with strong synergies: Miner, Forager, Explorer

**Synergy Benefits**: 2x XP acquisition, enhanced effectiveness, reduced costs (scales with class level)

## Mechanics

| Effect | Description                                       |
| ------ | ------------------------------------------------- |
| Unlock | Can survey areas to discover hidden/unknown nodes |
| Survey | Takes 1-2 hours per area, reveals all resources   |
| Claim  | First discoverer gets temporary claim             |

## Tier Progression

- **Tier 1 (Base)**: Basic surveying (1-2 hours per area)

## XP Sources

- Find new nodes: 15-30 XP per discovery
- Explore unmapped areas: 10-20 XP
- Guided training: 10-100 XP/session
- Unguided training: 5-50 XP/session

## Level-Up Availability

- Strong synergy: ~500 XP (~50 actions)
- Moderate synergy: ~1000 XP
- No synergy: ~1500 XP
