# Night Vision

**Pool**: Cross-Class
**Type**: Mechanic Unlock
**Classes**: Thief, Scout, Miner, certain races (improved version)

## Description

See perfectly in any level of darkness, including complete absence of light.

## Effect

| Property   | Description                         |
| ---------- | ----------------------------------- |
| Unlock     | Full vision in any darkness         |
| Quality    | Color vision, full detail           |
| Limitation | Bright light causes brief blindness |

## Acquisition

**Trigger Actions**:

- Extended time in darkness
- Adapt to underground

**Classes with Strong Synergy**: Thief, Scout, Miner, certain races (2x XP, direct specialization)

**Note**: Certain races may have improved versions of this ability as racial traits.

Binary unlock - vision is either enabled or not. No tiers or scaling.
