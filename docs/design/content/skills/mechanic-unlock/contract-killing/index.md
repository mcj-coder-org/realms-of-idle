# Contract Killing

**Pool**: Dark
**Type**: Mechanic Unlock
**Classes**: Assassin

## Description

Access to professional assassination contracts through the Assassins Guild, building a separate reputation track.

## Effect

| Property   | Description                                     |
| ---------- | ----------------------------------------------- |
| Unlock     | Access to assassination contracts through guild |
| Payment    | Based on target difficulty and profile          |
| Reputation | Build assassin reputation (separate track)      |

## Acquisition

**Note**: Requires Assassins Guild membership

**Trigger Actions**:

- Complete assassinations
- Build reputation

**Classes with Strong Synergy**: Assassin (2x XP, direct specialization)

## Related Content

- **Strong Synergy:** [Assassin](../../classes/rogue/assassin/)
- **See Also:** [Skills Index](../index.md)

## No Scaling

Binary unlock - contract availability and payment scale with assassin reputation, not skill level directly.
