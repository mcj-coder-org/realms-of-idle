# Linguistic Mastery

**Pool**: Scholar
**Type**: Mechanic Unlock
**Classes**: Scholar

## Description

Rapidly acquire new languages through study, gaining basic communication in days and fluency in weeks.

## Effect

| Property | Description                                         |
| -------- | --------------------------------------------------- |
| Unlock   | Can learn new languages through study               |
| Speed    | Learn basic communication in days, fluency in weeks |
| Bonus    | +25% to learning additional languages               |

## Acquisition

**Trigger Actions**:

- Study languages
- Communicate across cultures

**Classes with Strong Synergy**: Scholar (2x XP, direct specialization)

## Related Content

- **Strong Synergy:** [Scholar](../../classes/fighter/)
- **See Also:** [Skills Index](../index.md)

## No Scaling

Binary unlock - language learning speed is fixed, though additional languages become progressively easier.
