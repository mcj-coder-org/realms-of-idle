# Poison Craft

## Overview

**Type**: Mechanic Unlock
**Pool**: Stealth
**Description**: Create and apply poisons with increasing potency through progression.

## Synergy Classes

Classes with strong synergies: Assassin, Alchemist

**Synergy Benefits**: 2x XP acquisition, enhanced effectiveness, reduced costs (scales with class level)

## Mechanics

| Effect      | Description                           |
| ----------- | ------------------------------------- |
| Unlock      | Can craft poisons from ingredients    |
| Application | Apply poison to weapons (3-5 attacks) |
| Knowledge   | Identify poison types, know antidotes |

## Tier Progression

- **Tier 1 (Base)**: Basic poison crafting (3-5 attacks)
- **Tier 2**: Virulent Poisons—poisons 50% stronger, last 5-8 attacks
- **Tier 3**: Master Toxicologist—create unique poisons, poison immunity

## XP Sources

- Use poisons: 5-15 XP per application
- Harvest venoms: 10-20 XP
- Study toxins: 5-15 XP
- Guided training: 10-100 XP/session
- Unguided training: 5-50 XP/session

## Level-Up Availability

- Strong synergy: ~500 XP (~50 actions)
- Moderate synergy: ~1000 XP
- No synergy: ~1500 XP
