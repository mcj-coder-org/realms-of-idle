# Magical Weaving

**Type:** Mechanic Unlock
**Pool:** Magic/Textile
**Strong Synergy Classes:** Weaver, Enchanter
**Prerequisites:** Basic Weaving proficiency

## Effect

| Aspect     | Description                                                       |
| ---------- | ----------------------------------------------------------------- |
| Unlock     | Can weave magical thread into fabric                              |
| Benefit    | Magically woven fabric: inherent enchantment, +40% magic capacity |
| Limitation | Requires enchanted thread and magical workspace                   |
