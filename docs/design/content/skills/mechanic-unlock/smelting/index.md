# Smelting

**Type:** Mechanic Unlock
**Pool:** Smithing
**Strong Synergy Classes:** Blacksmith, Miner

## Effect

| Aspect     | Description                                         |
| ---------- | --------------------------------------------------- |
| Unlock     | Can extract metal from ore using heat               |
| Benefit    | Self-smelted ingots: +10% purity, process rare ores |
| Limitation | Requires furnace and knowledge of ore types         |

## Notes

- Transforms raw ore into usable metal
- Different ores require different temperatures
- Flux removes impurities from metal
- Foundation skill for self-sufficient smithing
- Can recover metal from scrap items
