# Sentinel's Watch

## Overview

Cannot be ambushed while awake. Provides immunity to surprise attacks, ensures first combat round action, and extends ambush resistance to nearby allies.

## Synergy Classes

**Strong Synergy (2x XP)**: Guard, Knight, Scout

Classes with strong synergies get faster learning, better effectiveness, and lower costs.

**XP Sources**:

## Mechanics

**Trigger Actions:** Detect ambushes, stand watch, maintain vigilance

**Ambush Immunity:** Cannot be surprised by ambushes, traps, or stealth attacks while conscious.

**First Action:** Always act in the first combat round regardless of initiative rolls.

**Party Protection:** Adjacent allies (within immediate proximity) gain +25% resistance to ambush attempts.

**Unconscious Limitation:** Does not function if character is unconscious, sleeping without preparation, or incapacitated.

**Watch Preparation:** If sleeping in prepared watch rotation, provides partial benefit.
