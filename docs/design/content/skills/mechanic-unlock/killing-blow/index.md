# Killing Blow

## Overview

Execute wounded enemies instantly when they fall below a health threshold. Success depends on attacker's STR versus target's END.

## Synergy Classes

**Strong Synergy (2x XP)**: Warrior, Assassin, Knight

Classes with strong synergies get faster learning, better effectiveness, and lower costs.

## XP Progression

**XP Sources**:

- Land killing blows: 10-20 XP per execution
- Finish wounded enemies: 5-10 XP per kill
- Attempt executions: 2-5 XP per attempt
- Guided training: 10-100 XP per session
- Unguided training: 5-50 XP per session

**Level-Up Availability Thresholds**:

- Strong synergy classes: ~500 XP (~50 actions)
- Moderate synergy classes: ~1000 XP (~75 actions)

## Mechanics

**Trigger Actions:** Land killing blows, finish wounded enemies

**Threshold:** Enemy must be at or below 15% HP to attempt execution.

**Success Calculation:** STR (attacker) contested against END (target). Higher STR relative to target END increases success chance.

**Failure Result:** If execution fails, performs normal attack instead (still dealing damage).

**Execution Method:** Different execution animations/methods based on weapon type.

**Boss Resistance:** Elite and boss enemies may have resistance or immunity to execution.

**Cooldown:** None - can attempt on any eligible enemy.
