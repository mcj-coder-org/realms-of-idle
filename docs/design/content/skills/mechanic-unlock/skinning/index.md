# Skinning

**Type:** Mechanic Unlock
**Pool:** Animal
**Strong Synergy Classes:** Hunter, Leatherworker, Trapper

## Effect

| Aspect     | Description                                   |
| ---------- | --------------------------------------------- |
| Unlock     | Remove pelts and hides from creatures intact  |
| Assessment | Evaluate hide quality before removal          |
| Expansion  | Learn specialized skinning for creature types |

## Notes

- Pelt condition affects final quality (damage reduces value)
- Winter pelts are more valuable than summer pelts
- Some creatures have special hide requirements (scales, feathers)
