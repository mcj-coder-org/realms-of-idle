# Spirit Binding

**Pool**: Shamanic
**Type**: Mechanic Unlock
**Classes**: Shaman

## Description

Allows the Shaman to bind defeated or willing spirits to their tribal pool, expanding the tribe's spiritual resources.

## Effect

| Property | Description                                       |
| -------- | ------------------------------------------------- |
| Unlock   | Can bind defeated/willing spirits to tribal pool  |
| Process  | Ritual combat or negotiation                      |
| Limit    | Pool can only hold spirits up to tribe's capacity |

## Acquisition

**Prerequisite**: Master Shaman rank

**Trigger Actions**:

- Encounter spirits
- Grow tribal pool

**Classes with Strong Synergy**: Shaman (2x XP, direct specialization)
