# Policy Implementation

**Pool**: Leadership
**Type**: Mechanic Unlock
**Classes**: Settlement Leader, Guildmaster

## Description

Set and modify formal policies that govern settlements, controlling taxes, enforcement, trade, and organizational permissions.

## Effect

| Property      | Description                                              |
| ------------- | -------------------------------------------------------- |
| Unlock        | Can set and modify settlement policies                   |
| Types         | Taxes, corruption, enforcement, trade, guild permissions |
| Effectiveness | Policy effects scale with leadership skill               |

## Acquisition

**Note**: Core Settlement Leader mechanic

**Trigger Actions**:

- Manage settlements
- Govern populations

**Classes with Strong Synergy**: Settlement Leader, Guildmaster (2x XP, direct specialization)

- **Strong Synergy:** [Settlement Leader](../../classes/settlement leader/), [Guildmaster](../../classes/guildmaster/)

## No Scaling

Binary unlock - policy effectiveness scales with overall leadership skill level.
