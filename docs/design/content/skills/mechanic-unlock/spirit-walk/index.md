# Spirit Walk

**Pool**: Shamanic
**Type**: Mechanic Unlock (Multi-Tier)
**Classes**: Shaman

## Description

Project your spirit from your body to observe distant locations, allowing reconnaissance and surveillance without physical travel.

## Tiers

### Tier 1 (Basic)

| Property   | Description                                         |
| ---------- | --------------------------------------------------- |
| Unlock     | Can send spirit to observe distant locations        |
| Range      | Within tribal territory (further with higher skill) |
| Limitation | Body vulnerable while projecting                    |

### Tier 2: Extended Walk

| Property | Description                     |
| -------- | ------------------------------- |
| Range    | Observe anywhere known to tribe |

### Tier 3: Spirit Journey

| Property    | Description                        |
| ----------- | ---------------------------------- |
| Interaction | Interact minimally while projected |

## Acquisition

**Trigger Actions**:

- Practice projection
- Commune with spirits

**Classes with Strong Synergy**: Shaman (2x XP, direct specialization)
