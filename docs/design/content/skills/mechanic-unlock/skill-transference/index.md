# Skill Transference

**Pool**: Scholar
**Type**: Mechanic Unlock
**Classes**: Trainer

## Description

Directly teach skills you possess to eligible students, enabling them to acquire abilities through your instruction.

## Effect

| Property | Description                                    |
| -------- | ---------------------------------------------- |
| Unlock   | Can teach skills you know to eligible students |
| Process  | Time based on skill tier and student aptitude  |
| Limit    | Can only teach skills at your tier or lower    |

## Acquisition

**Note**: Core Trainer mechanic

**Trigger Actions**:

- Teach skills
- Mentor students

**Classes with Strong Synergy**: Trainer (2x XP, direct specialization)

- **Strong Synergy:** [Trainer](../../classes/trainer/)

## No Scaling

Binary unlock - teaching time based on skill complexity and student ability.
