# Silk Weaving

**Type:** Mechanic Unlock
**Pool:** Textile
**Strong Synergy Classes:** Weaver
**Prerequisites:** Basic Weaving proficiency

## Effect

| Aspect     | Description                                          |
| ---------- | ---------------------------------------------------- |
| Unlock     | Can weave silk fiber into luxury fabric              |
| Benefit    | Silk fabric: +100% value, superior drape and comfort |
| Limitation | Silk fiber expensive and difficult to work           |

## Notes

- Silk requires gentler handling than other fibers
- Premium material for formal and magical garments
- Different weave patterns create different textures
- Can combine with gold thread for ultimate luxury
- Elven silk weaving techniques are most advanced
