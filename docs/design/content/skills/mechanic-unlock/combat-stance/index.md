# Combat Stance

**Type:** Mechanic Unlock
**Pool:** Combat
**Strong Synergy Classes:** Fighter, Warrior, Guardian

## Effect

| Aspect    | Description                                    |
| --------- | ---------------------------------------------- |
| Unlock    | Access basic combat stancing and positioning   |
| Technique | Switch between defensive and offensive stances |
| Quality   | +10% effectiveness for chosen stance           |

## Mechanics

- Toggle between stances (Defensive, Offensive, Balanced)
- Defensive: +15% DEF, -10% DMG
- Offensive: +15% DMG, -10% DEF
- Balanced: Equal offense and defense
- Stances affect combat calculations

## Trigger Actions

- Engage in combat
- Take defensive positions
- Achieve positioning advantages
