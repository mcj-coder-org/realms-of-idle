# Money Laundering

**Pool**: Dark
**Type**: Mechanic Unlock (Multi-Tier)
**Classes**: Fence, Merchant (dark path)

## Description

Convert marked or suspicious currency into clean, untraceable money at a cost.

## Tiers

### Tier 1 (Basic)

| Property | Description                                          |
| -------- | ---------------------------------------------------- |
| Unlock   | Can convert "marked" or suspicious currency to clean |
| Rate     | Lose 15-25% in process                               |
| Benefit  | Clean money can't be traced                          |

### Tier 2: Efficient Laundering

| Property | Description      |
| -------- | ---------------- |
| Rate     | Lose only 10-15% |

## Acquisition

**Trigger Actions**:

- Handle large amounts of criminal money

**Classes with Strong Synergy**: Fence, Merchant (dark path) (2x XP, direct specialization)

## Related Content

- **Strong Synergy:** [Fence](../../classes/rogue/fence/), [Merchant (dark path)](../../classes/merchant (dark path)/)
- **See Also:** [Skills Index](../index.md)

## Multi-Tier Unlock

Each tier reduces the cost of laundering. Progress by processing larger volumes successfully.
