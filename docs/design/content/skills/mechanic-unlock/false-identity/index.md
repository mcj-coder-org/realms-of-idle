# False Identity

**Pool**: Dark
**Type**: Mechanic Unlock (Multi-Tier)
**Classes**: Thief, Assassin, Smuggler, Spy

## Description

Create convincing false identities with separate reputations, allowing you to escape consequences and operate in multiple spheres.

## Tiers

### Tier 1 (Basic)

| Property    | Description                                       |
| ----------- | ------------------------------------------------- |
| Unlock      | Can create and maintain false identity            |
| Benefits    | Separate reputation, escape consequences          |
| Maintenance | Must spend time in persona, keep stories straight |

### Tier 2: Multiple Identities

| Property | Description                   |
| -------- | ----------------------------- |
| Effect   | Maintain 2-3 false identities |

### Tier 3: Master of Masks

| Property | Description                             |
| -------- | --------------------------------------- |
| Effect   | Seamless switching, nearly undetectable |

## Acquisition

**Trigger Actions**:

- Operate under false pretenses
- Maintain cover

**Classes with Strong Synergy**: Thief, Assassin, Smuggler, Spy (2x XP, direct specialization)

## Multi-Tier Unlock

Each tier expands capacity and improves convincingness. Progress by successfully maintaining identities over time.
