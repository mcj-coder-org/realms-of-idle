# Succession Planning

**Pool**: Leadership
**Type**: Mechanic Unlock
**Classes**: Leader, Chieftain, Guildmaster

## Description

Designate and train a successor, ensuring smooth transitions of power and preserving organizational stability.

## Effect

| Property   | Description                                              |
| ---------- | -------------------------------------------------------- |
| Unlock     | Can designate and train successor                        |
| Benefit    | Successor gains +25% leadership XP when working with you |
| Transition | Smooth succession (no loyalty loss) when you step down   |

## Acquisition

**Trigger Actions**:

- Mentor subordinates
- Plan for future

**Classes with Strong Synergy**: Leader, Chieftain, Guildmaster (2x XP, direct specialization)

Binary unlock - training bonus and transition effects are fixed.
