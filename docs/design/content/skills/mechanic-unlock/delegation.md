# Delegation

**Pool**: Leadership
**Type**: Mechanic Unlock (Multi-Tier)
**Classes**: Leader, Guildmaster, Settlement Leader

## Description

Efficiently assign tasks to followers who will complete them autonomously, freeing you to focus on other matters.

## Tiers

### Tier 1 (Basic)

| Property   | Description                                              |
| ---------- | -------------------------------------------------------- |
| Unlock     | Can assign tasks to followers that complete autonomously |
| Efficiency | Delegated tasks complete at 80% your efficiency          |
| Limit      | Number of simultaneous delegations = Leadership skill    |

### Tier 2: Efficient Delegation

| Property   | Description             |
| ---------- | ----------------------- |
| Efficiency | 90% efficiency          |
| Limit      | More simultaneous tasks |

## Acquisition

**Trigger Actions**:

- Manage followers
- Assign complex tasks

**Classes with Strong Synergy**: Leader, Guildmaster, Settlement Leader (2x XP, direct specialization)

Each tier improves efficiency and capacity. Progress by demonstrating effective management and delegation.
