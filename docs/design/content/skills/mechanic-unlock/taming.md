# Taming

**Type:** Mechanic Unlock
**Pool:** Animal
**Strong Synergy Classes:** Animal Trainer, Beastmaster

## Effect

| Aspect     | Description                                      |
| ---------- | ------------------------------------------------ |
| Unlock     | Can attempt to tame wild animals                 |
| Process    | Takes days/weeks depending on creature           |
| Limitation | Only tameable species, must be appropriate level |

## Progression

- **Tier 2:** Advanced Taming—tame difficult creatures, reduce time by 50%
- **Tier 3:** Master Tamer—tame exotic creatures, instant bond with common animals

## Trigger Actions

- Successfully tame animals
- Work with wild creatures

## Related Content

- **Strong Synergy:** [Animal Trainer](../../classes/animal trainer/), [Beastmaster](../../classes/gatherer/hunter/)
- **See Also:** [Skills Index](../index.md)

## Notes

- Not all creatures are tameable
- Character level affects which creatures can be tamed
- Time requirement scales with creature difficulty
