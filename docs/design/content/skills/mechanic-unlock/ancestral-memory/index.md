# Ancestral Memory

**Pool**: Shamanic
**Type**: Mechanic Unlock
**Classes**: Shaman

## Description

Consult with honored ancestors to access knowledge from the tribe's past, including history, locations, and techniques.

## Effect

| Property    | Description                                          |
| ----------- | ---------------------------------------------------- |
| Unlock      | Can consult ancestors for knowledge                  |
| Information | History, locations, techniques known to honored dead |
| Limitation  | Ancestors may not know, or may not share             |

## Acquisition

**Trigger Actions**:

- Honor ancestors
- Maintain tribal traditions

**Classes with Strong Synergy**: Shaman (2x XP, direct specialization)

## No Scaling

Binary unlock - quality of information based on relationship with ancestors.
