# Deep Vein

## Overview

**Type**: Mechanic Unlock
**Pool**: Gathering
**Description**: Access difficult resource deposits in challenging locations.

## Synergy Classes

Classes with strong synergies: Miner, Forager, Fisher

**Synergy Benefits**: 2x XP acquisition, enhanced effectiveness, reduced costs (scales with class level)

## Mechanics

| Effect      | Description                                           |
| ----------- | ----------------------------------------------------- |
| Unlock      | Can mine/gather from normally inaccessible deposits   |
| Types       | Underwater, cliffsides, deep underground, tree canopy |
| Requirement | Appropriate equipment (rope, breathing, climbing)     |

## XP Sources

- Gather from difficult locations: 15-25 XP
- Use specialized techniques: 10-20 XP
- Guided training: 10-100 XP/session
- Unguided training: 5-50 XP/session

## Level-Up Availability

- Strong synergy: ~500 XP (~50 actions)
- Moderate synergy: ~1000 XP
- No synergy: ~1500 XP
