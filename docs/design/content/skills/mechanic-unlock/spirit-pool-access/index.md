# Spirit Pool Access

## Overview

| Attribute | Value                                            |
| --------- | ------------------------------------------------ |
| Type      | Mechanic Unlock                                  |
| Pool      | [Shamanic Pool](../pools/shamanic-pool/index.md) |
| Tags      | `Shamanic`                                       |

## Description

Spirit Pool Access grants the character connection to their tribe's shared spirit pool. Unlike personal mana, the spirit pool is a communal resource drawn from the collective spiritual energy of the tribe.

## Effect

| Property | Value                                            |
| -------- | ------------------------------------------------ |
| Resource | Access to tribal spirit pool                     |
| Scaling  | Pool size scales with tribe                      |
| Distance | Effectiveness decreases with distance from tribe |

## Acquisition

| Method            | Requirement                      |
| ----------------- | -------------------------------- |
| Class Grant       | Spiritcaller (automatic)         |
| Tribal Initiation | Complete spirit communion ritual |
