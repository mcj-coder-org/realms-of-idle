# Weather Worker

## Overview

**Type**: Mechanic Unlock
**Pool**: Gathering
**Description**: Gather in adverse conditions without penalties, gaining bonuses during storms.

## Synergy Classes

Classes with strong synergies: All Gathering pool (Forager, Miner, Hunter, Fisher, Lumberjack, Trapper, Herbalist)

**Synergy Benefits**: 2x XP acquisition, enhanced effectiveness, reduced costs (scales with class level)

## Mechanics

| Effect | Description                                                  |
| ------ | ------------------------------------------------------------ |
| Unlock | No gathering penalty from weather conditions                 |
| Bonus  | +10% yield during storms (washed-up materials, flushed game) |
| Safety | Immune to weather damage while gathering                     |
