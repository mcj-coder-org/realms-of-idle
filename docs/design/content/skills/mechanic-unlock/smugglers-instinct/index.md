# Smuggler's Instinct

## Overview

**Type**: Mechanic Unlock
**Pool**: Stealth
**Description**: Innate sense for hiding contraband and detecting searches.

## Synergy Classes

Classes with strong synergies: Smuggler, Thief, Caravaner

**Synergy Benefits**: 2x XP acquisition, enhanced effectiveness, reduced costs (scales with class level)

## Mechanics

| Effect    | Description                                           |
| --------- | ----------------------------------------------------- |
| Unlock    | Know when searches are coming (brief warning)         |
| Hiding    | +30% effectiveness hiding items on person or in cargo |
| Detection | Sense hidden compartments in others' cargo            |

- Smuggle goods: 10-20 XP
- Avoid searches: 15-25 XP
- Find hidden items: 5-15 XP

## Level-Up Availability

- Strong synergy: ~500 XP (~50 actions)
- Moderate synergy: ~1000 XP
- No synergy: ~1500 XP
