# Beekeeper's Bond

## Overview

**Type**: Mechanic Unlock
**Pool**: Agriculture
**Description**: Special connection with bees enabling safe interaction and enhanced yields.

## Synergy Classes

Classes with strong synergies: Beekeeper

**Synergy Benefits**: 2x XP acquisition, enhanced effectiveness, reduced costs (scales with class level)

## Mechanics

| Effect        | Description                                     |
| ------------- | ----------------------------------------------- |
| Unlock        | Bees never sting you, +50% honey/wax yield      |
| Communication | Bees warn of dangers, scout local flowers       |
| Defense       | Can direct swarm against threats (once per day) |

## XP Sources

- Work with bees: 5-15 XP per session
- Harvest honey/wax: 10-20 XP
- Guided training: 10-100 XP/session
- Unguided training: 5-50 XP/session

## Level-Up Availability

- Strong synergy: ~500 XP (~50 actions)
- Moderate synergy: ~1000 XP
- No synergy: ~1500 XP
