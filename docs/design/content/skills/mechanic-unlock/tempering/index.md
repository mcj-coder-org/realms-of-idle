# Tempering

**Type:** Mechanic Unlock
**Pool:** Smithing
**Strong Synergy Classes:** Blacksmith

## Effect

| Aspect     | Description                                                |
| ---------- | ---------------------------------------------------------- |
| Unlock     | Can temper hardened steel to reduce brittleness            |
| Benefit    | Tempered items: balanced hardness/toughness, -30% breakage |
| Limitation | Requires precise temperature control after hardening       |

## Notes

- Critical step after hardening
- Different temperatures produce different properties
- Color of heated steel indicates temperature
- Over-tempering makes metal too soft, under-tempering leaves it brittle
