# Charm Creation

## Overview

| Attribute | Value                                      |
| --------- | ------------------------------------------ |
| Type      | Mechanic Unlock                            |
| Pool      | [Shamanic Pool](../pools/shamanic-pool.md) |
| Tags      | `Shamanic/Nature`                          |

## Description

Charm Creation allows the shaman to craft small spirit-blessed objects that provide protection, luck, or other benefits to the wearer. Charms last for days to weeks depending on quality.

## Effect

| Property | Value                                    |
| -------- | ---------------------------------------- |
| Creates  | Protection, luck, health, warning charms |
| Duration | Days to weeks (quality dependent)        |
| Cost     | Spirit points + materials                |

## Charm Types

| Type             | Effect           | Duration      |
| ---------------- | ---------------- | ------------- |
| Protection charm | Damage reduction | Days to weeks |
| Luck charm       | Fortune bonus    | Days          |
| Health charm     | Recovery bonus   | Days to weeks |
| Warning charm    | Danger sense     | Weeks         |
| Strength charm   | Attribute bonus  | Hours to days |

## Acquisition

| Method      | Requirement             |
| ----------- | ----------------------- |
| Class Grant | Shaman (automatic)      |
| Training    | Spirit communion ritual |
