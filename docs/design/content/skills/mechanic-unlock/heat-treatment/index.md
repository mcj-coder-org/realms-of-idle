# Heat Treatment

**Type:** Mechanic Unlock
**Pool:** Smithing
**Strong Synergy Classes:** Blacksmith

## Effect

| Aspect     | Description                                         |
| ---------- | --------------------------------------------------- |
| Unlock     | Can heat treat metal to improve hardness/durability |
| Benefit    | Treated items: +20% durability, +10% damage/defense |
| Limitation | Requires forge with precise temperature control     |

## Notes

- Foundation technique for advanced smithing
- Different metals require different heat treatment profiles
- Improper treatment can ruin a piece entirely
- Prerequisites for Damascus Forging and Differential Hardening
