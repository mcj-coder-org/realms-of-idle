# Dyeing

**Type:** Mechanic Unlock
**Pool:** Textile
**Strong Synergy Classes:** Tailor, Weaver

## Effect

| Aspect     | Description                                      |
| ---------- | ------------------------------------------------ |
| Unlock     | Can permanently color fabric and thread          |
| Benefit    | Dyed items: custom colors, +30% aesthetic value  |
| Limitation | Requires dye vats, mordants, and color knowledge |

## Notes

- Natural dyes from plants, insects, and minerals
- Colorfast dyeing requires proper mordanting
- Some colors (purple, certain reds) historically precious
- Can dye raw fiber, thread, or finished fabric
- Combines with Weaving for complex color patterns
