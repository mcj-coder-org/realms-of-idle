# Perfect Memory

**Pool**: Scholar
**Type**: Mechanic Unlock
**Classes**: Scholar, Scribe

## Description

Perfectly recall anything you intentionally memorize, with capacity limited by your Wit attribute.

## Capacity Based on WIT

Number of memory slots scales with WIT attribute, not skill progression. Higher WIT = more information can be stored.
