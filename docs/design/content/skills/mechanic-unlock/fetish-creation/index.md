# Fetish Creation

**Pool**: Shamanic
**Type**: Mechanic Unlock
**Classes**: Shaman

## Description

Bind spirits into physical items permanently, creating powerful shamanic artifacts equivalent to enchanted equipment.

## Effect

| Property | Description                                    |
| -------- | ---------------------------------------------- |
| Unlock   | Can create fetishes (permanent shamanic items) |
| Process  | Bind spirit into item permanently              |
| Power    | Equivalent to enchanted items                  |

## Acquisition

**Prerequisite**: Master Shaman rank

**Trigger Actions**:

- Create powerful shamanic items
- Bind willing spirits

**Classes with Strong Synergy**: Shaman (2x XP, direct specialization)

## Related Content

- **Strong Synergy:** [Shaman](../../classes/spiritcaller/shaman/)
- **See Also:** [Skills Index](../index.md)

## No Scaling

Binary unlock - fetish power scales with spirit bound and crafting materials used.
