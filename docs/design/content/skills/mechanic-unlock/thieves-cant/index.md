# Thieves' Cant

## Overview

**Type**: Mechanic Unlock
**Pool**: Stealth
**Description**: Secret criminal communication system and underworld knowledge.

## Synergy Classes

Classes with strong synergies: Thief, Assassin, Fence, Smuggler

**Synergy Benefits**: 2x XP acquisition, enhanced effectiveness, reduced costs (scales with class level)

## Mechanics

| Effect      | Description                                             |
| ----------- | ------------------------------------------------------- |
| Unlock      | Understand Thieves' Cant (criminal code language)       |
| Recognition | Identify other criminals, find black market             |
| Marking     | Read and leave thieves' marks (warnings, opportunities) |

## XP Sources

- Interact with criminals: 5-10 XP
- Operate in underground: 10-20 XP
- Use thieves' marks: 3-8 XP
- Guided training: 10-100 XP/session
- Unguided training: 5-50 XP/session

## Level-Up Availability

- Strong synergy: ~500 XP (~50 actions)
- Moderate synergy: ~1000 XP
- No synergy: ~1500 XP
