# Racial Abilities

Innate abilities that characters receive based on their race. These cannot be learned - they are part of a race's natural capabilities.

## Physical Traits

| Ability        | Description                           | Typical Races           |
| -------------- | ------------------------------------- | ----------------------- |
| Powerful Build | Counts as larger for carrying/pushing | Orcs, Trolls, Minotaurs |
| Massive Frame  | Bonus health and strength             | Giants, Ogres           |
| Small Frame    | Easier to hide, fit in small spaces   | Halflings, Gnomes       |
| Dense Build    | Natural armor bonus                   | Dwarves, Goliaths       |

## Aquatic Abilities

| Ability        | Description              | Typical Races       |
| -------------- | ------------------------ | ------------------- |
| Amphibious     | Breathe air and water    | Lizardfolk, Merfolk |
| Hold Breath    | Extended underwater time | Many aquatic races  |
| Salt Tolerance | Survive in saltwater     | Sea-dwelling races  |

## Sensory Abilities

| Ability        | Description          | Typical Races        |
| -------------- | -------------------- | -------------------- |
| Darkvision     | See in darkness      | Elves, Dwarves, Orcs |
| Keen Senses    | Enhanced perception  | Elves, Catfolk       |
| Scent Tracking | Track by smell       | Gnolls, Wolfkin      |
| Tunnel Sense   | Navigate underground | Dwarves, Kobolds     |

## Natural Weapons

| Ability       | Description           | Typical Races        |
| ------------- | --------------------- | -------------------- |
| Venomous Bite | Poisonous bite attack | Serpentfolk, Spiders |
| Savage Bite   | Powerful bite attack  | Gnolls, Lizardfolk   |
| Natural Claws | Claw attacks          | Catfolk, Lizardfolk  |

## Resistances

| Ability           | Description           | Typical Races              |
| ----------------- | --------------------- | -------------------------- |
| Poison Resistance | Resist poison effects | Dwarves, certain creatures |
| Fire Resistance   | Resist fire damage    | Tieflings, Fire Genasi     |
| Cold Resistance   | Resist cold damage    | Goliaths, Frost Giants     |
