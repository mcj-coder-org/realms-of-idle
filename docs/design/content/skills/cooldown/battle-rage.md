# Battle Rage

## Overview

Controlled fury in combat that provides significant temporary offensive bonuses with scaling defensive trade-offs that diminish at higher tiers.

## Synergy Classes

**Strong Synergy (2x XP)**: Warrior, Adventurer, Mercenary

Classes with strong synergies get faster learning, better effectiveness, and lower costs.

## Tiers

| Tier     | Type     | Effect                                          | Duration |
| -------- | -------- | ----------------------------------------------- | -------- |
| Lesser   | Duration | +20% damage, -10% defense                       | 30 sec   |
| Greater  | Duration | +35% damage, -5% defense, pain resistance       | 45 sec   |
| Enhanced | Duration | +50% damage, no defense penalty, immune to fear | 60 sec   |

**Scaling:** Duration +2 sec per class level

## XP Progression

**XP Sources**:

- Take damage in combat: 2-5 XP per hit received
- Kill enemies while raging: 10-20 XP per kill
- Fight while wounded: 5-15 XP per wounded combat
- Guided training: 10-100 XP per session
- Unguided training: 5-50 XP per session

**Level-Up Availability Thresholds**:

- Strong synergy classes: ~500 XP (~50 actions)
- Moderate synergy classes: ~1000 XP (~75 actions)
- No synergy classes: ~1500 XP (~100-150 actions)

## Synergy Bonuses

**Strong Synergy Example** (Warrior Level 15):

- Faster learning: 2x XP from rage-triggering actions
- Better effectiveness: +25% damage bonus while raging
- Reduced cost: -25% stamina activation cost

**Scaling with Class Level**:

| Class Level | XP Multiplier | Effectiveness | Cost Reduction |
| ----------- | ------------- | ------------- | -------------- |
| Level 5     | 1.5x (+50%)   | +15%          | -15%           |
| Level 15    | 2.0x (+100%)  | +25%          | -25%           |
| Level 30+   | 2.5x (+150%)  | +35%          | -35%           |

## Mechanics

**Trigger Actions:** Take damage in combat, kill enemies, fight while wounded

**Activation:** Manual activation or automatic trigger when taking significant damage (player choice).

**Pain Resistance:** Greater tier provides resistance to status effects caused by wounds.

**Fear Immunity:** Enhanced tier grants complete immunity to fear and morale-breaking effects.

**Duration Extension:** Duration extends by +2 seconds per class level of the primary synergy class.

**Cooldown:** 5 minutes after rage ends before it can be activated again.
