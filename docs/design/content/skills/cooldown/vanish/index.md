# Vanish

## Overview

**Type**: Mechanic Unlock
**Pool**: Stealth
**Description**: Emergency stealth entry ability. Once per encounter, instantly enter stealth even when observed.

## Synergy Classes

Classes with strong synergies: Thief, Assassin, Scout

**Synergy Benefits**: 2x XP acquisition, enhanced effectiveness, reduced costs (scales with class level)

## Mechanics

| Effect      | Description                                                 |
| ----------- | ----------------------------------------------------------- |
| Unlock      | Once per encounter, instantly enter stealth (even observed) |
| Requirement | Must have escape route (can't vanish if surrounded)         |
| Cooldown    | 10 minutes                                                  |

## XP Sources

- Escape detection: 15-25 XP
- Successfully flee: 10-20 XP
- Break line of sight: 5-15 XP
- Guided training: 10-100 XP/session
- Unguided training: 5-50 XP/session

## Level-Up Availability

- Strong synergy: ~500 XP (~50 actions)
- Moderate synergy: ~1000 XP
- No synergy: ~1500 XP

## Synergy Examples

### Strong (2x XP)

**Assassin Level 15** - Escape specialist

- Reduced cooldown (8 min)
- 2x XP from escapes

### Moderate (1.5x XP)

**Scout Level 10** - Evasion expert

- Slightly reduced cooldown

### None (1x XP)

**Merchant** - No connection

- No bonuses
