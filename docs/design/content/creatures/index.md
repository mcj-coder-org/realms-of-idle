# Creatures Library

Creatures populate the world. All have harvestable components used in crafting.

## Creature Categories

| Category              | Description                  | Index             |
| --------------------- | ---------------------------- | ----------------- |
| [Wildlife](wildlife/) | Normal animals, tameable     | wildlife/index.md |
| [Magical](magical/)   | Magically-enhanced creatures | magical/index.md  |
| [Bosses](bosses/)     | Unique powerful creatures    | bosses/index.md   |

## Classification Axes

| Axis         | Values                                                   | Effect                |
| ------------ | -------------------------------------------------------- | --------------------- |
| Intelligence | Beast / Cunning / Sapient                                | Tactics, skill access |
| Magic        | None / Innate / Mana                                     | Spell access          |
| Social       | Independent / Pack / Herd / Swarm                        | Encounter size        |
| Aggression   | Passive / Defensive / Territorial / Aggressive / Hostile | Attack triggers       |
| Rarity       | Common / Uncommon / Rare / Legendary                     | Spawn rate, loot      |

## Aggression Behaviors

| Rating      | Behavior                  |
| ----------- | ------------------------- |
| Passive     | Never attacks, flees      |
| Defensive   | Only if attacked/cornered |
| Territorial | Attacks if in territory   |
| Aggressive  | Attacks on sight          |
| Hostile     | Seeks out targets         |

## Harvestables

All creatures provide:

- **Meat** - Food source
- **Hide/Scales** - Leather/armor
- **Bones** - Crafting materials
- **Organs** - Alchemy reagents
- **Special parts** - Creature-specific
