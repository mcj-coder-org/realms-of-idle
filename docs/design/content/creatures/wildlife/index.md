# Wildlife

Normal animals found in the world. Can be hunted for resources or tamed as companions/mounts.

## Categories

| Category     | Examples            | Behavior               |
| ------------ | ------------------- | ---------------------- |
| Prey         | Deer, rabbits, fish | Passive, flees         |
| Predators    | Wolves, bears       | Territorial/Aggressive |
| Pack animals | Horses, oxen        | Passive, tameable      |
| Birds        | Eagles, ravens      | Various                |

## Harvestables

All wildlife provides:

- Meat (food)
- Hide/fur (leather)
- Bones (crafting)
- Organs (alchemy)
