# Docile Wildlife

Small, peaceful creatures that pose no threat and flee when approached. Primary sources of basic resources for beginning hunters and crafters.

## Creatures

| Creature                      | Level | Habitat          | Primary Resource  | Difficulty |
| ----------------------------- | ----- | ---------------- | ----------------- | ---------- |
| [Rabbit](rabbit/index.md)     | 1-2   | Fields, forests  | Meat, fur, organs | Trivial    |
| [Deer](deer/index.md)         | 3-5   | Forests, meadows | Venison, leather  | Easy       |
| [Songbird](songbird/index.md) | 0-1   | Everywhere       | Feathers, down    | Trivial    |

## Behavior Patterns

All docile wildlife share common traits:

| Trait           | Description                             |
| --------------- | --------------------------------------- |
| Aggression      | Passive (never initiates combat)        |
| Flee Response   | Immediate flight when detecting threats |
| Group Behavior  | May warn others when spooked            |
| Activity        | Most active at dawn/dusk (crepuscular)  |
| Nesting/Denning | Prefer hidden locations for safety      |

## Hunting Considerations

### For Beginners

- **Best Training:** These creatures teach basic tracking and approach skills
- **Low Risk:** No combat danger, perfect for learning
- **Common:** Easy to find near any settlement
- **Quick Respawn:** Populations recover rapidly
- **Basic Resources:** Provides essential crafting materials

### Hunter Tips

| Tip                 | Benefit                            |
| ------------------- | ---------------------------------- |
| Approach downwind   | Prevents early detection           |
| Move during feeding | Creatures less alert while eating  |
| Use cover           | Reduces detection range            |
| Dawn/dusk hunting   | Peak activity times                |
| Track to den/nest   | Multiple creatures in one location |

## Ecology

### Habitat Distribution

| Region Type     | Common Species         | Abundance |
| --------------- | ---------------------- | --------- |
| Farmland/Fields | Rabbit, Songbird       | Very High |
| Forest Edge     | Deer, Rabbit, Songbird | High      |
| Deep Forest     | Deer, Songbird         | Moderate  |
| Mountains       | Deer, Some songbirds   | Moderate  |
| Urban/Gardens   | Rabbit, Songbird       | High      |

### Population Dynamics

- **Fast Breeders:** Rabbits and songbirds reproduce rapidly
- **Seasonal Patterns:** Deer breeding in autumn, songbirds in spring
- **Predator Pressure:** Major prey species for predators
- **Human Impact:** Hunting sustainable due to high reproduction rates
- **Agricultural:** Rabbits and deer can become crop pests
