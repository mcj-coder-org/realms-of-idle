# Axe

## Overview

Forge an axe with curved bit for cutting wood. Primary tool for lumberjacks, useful as weapon.

---

## Mechanics

### Requirements

| Requirement | Value                                                             |
| ----------- | ----------------------------------------------------------------- |
| Class       | [Blacksmith](../../../../classes/crafter/blacksmith/) (Level 10+) |
| Workspace   | Forge, Anvil                                                      |
| Tools       | Hammer, Tongs                                                     |
| Time        | 6 hours                                                           |

### Ingredients

| Ingredient                                             | Quantity | Notes       |
| ------------------------------------------------------ | -------- | ----------- |

| [Wood Handle](../../../../item/component/wood-handle/) | 1 unit   | Curved haft |

| Property     | Value      |
| ------------ | ---------- |

| Stamina Cost | 25         |
| Base XP      | 35         |
| Failure Rate | 15%        |

## Process

1. **Heat Metal** - Bring ingots to forging temperature
2. **Form Bit** - Shape axe head with curved cutting edge
3. **Create Eye** - Forge hole for handle attachment
4. **Heat Treat** - Quench bit for edge retention
5. **Attach Handle** - Fit curved handle through eye
6. **Secure** - Wedge handle in place
