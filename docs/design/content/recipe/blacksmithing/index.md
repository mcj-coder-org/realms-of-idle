# Blacksmithing

Blacksmithing recipes transform metal ingots and components into finished tools, weapons, and armor through heat, hammer, and skill.

## Blacksmithing Recipes

### Weapons

- **[Dagger](dagger/)** - Short blade for close combat
- **[Longsword](longsword/)** - Versatile sword for combat

### Tools

- **[Pickaxe](pickaxe/)** - Mining tool
- **[Axe](axe/)** - Logging tool
- **[Hammer](hammer/)** - Forging tool

## Requirements

| Requirement | Value                                           |
| ----------- | ----------------------------------------------- |
| Class       | [Blacksmith](../../classes/crafter/blacksmith/) |
| Workspace   | Forge, Anvil                                    |
| Fuel        | [Coal](../../material/fuel/coal/)               |
| Tools       | Hammer, Tongs                                   |

## Recipe Tiers

| Tier       | Level | Recipes          |
| ---------- | ----- | ---------------- |
| Basic      | 1-9   | Copper items     |
| Apprentice | 10-24 | Iron items       |
| Journeyman | 25-49 | Steel items      |
| Master     | 50+   | Masterwork items |

## Material Variants

Most blacksmithing recipes use `<Material/Metal/*>` Ingot as input:

| Ingot Type   | Tier | Effect                         |
| ------------ | ---- | ------------------------------ |
| Copper Ingot | 1    | Basic items, lower durability  |
| Iron Ingot   | 2    | Standard items                 |
| Steel Ingot  | 3    | Superior items, enhanced stats |
