# Leather Strip

## Overview

Cut leather hides into strips for weapon grips, armor reinforcement, and tool bindings.

---

## Mechanics

### Requirements

| Requirement | Value                                                                                                   |
| ----------- | ------------------------------------------------------------------------------------------------------- |
| Class       | [Hunter](../../../../classes/gatherer/hunter/) or [Blacksmith](../../../../classes/crafter/blacksmith/) |
| Workspace   | None                                                                                                    |
| Tools       | Knife                                                                                                   |
| Time        | 30 minutes per batch                                                                                    |

## Process

1. **Select Hide** - Choose strong, flexible leather
2. **Mark Cuts** - Layout strip pattern
3. **Cut Strips** - Use sharp knife to cut even strips
4. **Check Quality** - Ensure consistent width and thickness
5. **Roll and Store** - Keep strips for later use
