# Processing

Processing recipes transform raw materials into components used in crafting. These are intermediate steps between gathering and final crafting.

## Processing Recipes

- **[Leather Strip](leather-strip/)** - Cut leather for weapon components
- **[Wood Handle](wood-handle/)** - Shape wood for tool hafts

## Requirements

| Requirement | Value                                     |
| ----------- | ----------------------------------------- |
| Workspace   | None (can be done anywhere)               |
| Tools       | Knife (for leather), Knife/Saw (for wood) |
| Time        | 30 minutes per batch                      |

## Outputs

| Input                                      | Output                                               |
| ------------------------------------------ | ---------------------------------------------------- |
| [Leather](../../material/organic/leather/) | [Leather Strip](../../item/component/leather-strip/) |
| [Wood](../../material/organic/wood/)       | [Wood Handle](../../item/component/wood-handle/)     |

## Related Classes

- **[Hunter](../../classes/gatherer/hunter/)** - Processes leather
- **[Lumberjack](../../classes/gatherer/lumberjack/)** - Processes wood
- **[Blacksmith](../../classes/crafter/blacksmith/)** - Uses components
