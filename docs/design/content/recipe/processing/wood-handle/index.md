# Wood Handle

## Overview

Shape seasoned wood into handles for tools and weapons. The right handle makes the difference between a tool that works and one that fails.

---

## Mechanics

### Requirements

| Requirement | Value                                                                                                           |
| ----------- | --------------------------------------------------------------------------------------------------------------- |
| Class       | [Lumberjack](../../../../classes/gatherer/lumberjack/) or [Blacksmith](../../../../classes/crafter/blacksmith/) |
| Workspace   | None                                                                                                            |
| Tools       | Knife, Saw                                                                                                      |
| Time        | 1 hour per batch                                                                                                |

### Ingredients

| Ingredient                                 | Quantity | Notes           |
| ------------------------------------------ | -------- | --------------- |
| [Wood](../../../../material/organic/wood/) | 1 unit   | Seasoned timber |

### Output

| Material                                               | Quantity | Variants                           |
| ------------------------------------------------------ | -------- | ---------------------------------- |
| [Wood Handle](../../../../item/component/wood-handle/) | 3 units  | Axe haft, sword grip, hammer shaft |

### Stats

| Property     | Value              |
| ------------ | ------------------ |
| Difficulty   | Basic              |
| Stamina Cost | 8                  |
| Base XP      | 12                 |
| Failure Rate | 5% (can ruin wood) |

## Process

1. **Select Wood** - Choose appropriate wood (ash for axes, oak for hammers)
2. **Mark Pattern** - Layout handle shape
3. **Rough Cut** - Cut to approximate shape with saw
4. **Shape** - Use knife for final shaping
5. **Fit Check** - Test against tool head if available
6. **Finish** - Sand smooth, oil if desired
