# Metals

Metals are extracted from the earth and refined for use in crafting weapons, armor, and tools.

## Metal Types

- **[Copper](copper/)** - Tier 1 metal, soft and conductive
- **[Iron](iron/)** - Tier 2 metal, strong and versatile
- **[Steel](steel/)** - Tier 3 metal, refined for superior quality

## Gathering

| Class                                     | Materials            | Notes                    |
| ----------------------------------------- | -------------------- | ------------------------ |
| [Miner](../../../classes/gatherer/miner/) | Copper Ore, Iron Ore | Surface and shaft mining |

## Processing

| Recipe                                           | Input      | Output |
| ------------------------------------------------ | ---------- | ------ |
| [Smelt Ore](../../../recipe/smelting/smelt-ore/) | Ore + Coal | Ingot  |

## Used In

- **[Weapons](../../item/weapon/)** - Daggers, swords, axes
- **[Armor](../../item/armor/)** - Helmets, chainmail, greaves
- **[Tools](../../item/tool/)** - Hammers, tongs, pickaxes
