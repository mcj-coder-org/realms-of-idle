# Fuel

Fuel provides the heat needed for smelting, forging, and alchemy. Without fuel, metal remains ore and recipes fail.

## Fuel Types

- **[Coal](coal/)** - Primary fuel for forges and furnaces

## Gathering

| Class                                     | Materials | Notes                        |
| ----------------------------------------- | --------- | ---------------------------- |
| [Miner](../../../classes/gatherer/miner/) | Coal      | Mined alongside ore deposits |

## Used In

- **[Smelting](../../../recipe/smelting/)** - Required for ore processing
- **[Forging](../../../recipe/blacksmithing/)** - Required for metalworking
- **[Alchemy](../../../recipe/alchemy/)** - Heat source for recipes
