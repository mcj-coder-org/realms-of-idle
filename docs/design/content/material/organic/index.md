# Organic Materials

Organic materials come from living sources - plants and animals. These materials are essential for crafting components, tools, and weapons.

## Material Types

- **[Leather](leather/)** - Tanned animal hides for components and armor
- **[Wood](wood/)** - Hardened plant material for handles and components

## Gathering

| Class                                               | Materials      | Notes                    |
| --------------------------------------------------- | -------------- | ------------------------ |
| [Hunter](../../../classes/gatherer/hunter/)         | Leather, pelts | From creature processing |
| [Lumberjack](../../../classes/gatherer/lumberjack/) | Wood           | From forest harvesting   |
